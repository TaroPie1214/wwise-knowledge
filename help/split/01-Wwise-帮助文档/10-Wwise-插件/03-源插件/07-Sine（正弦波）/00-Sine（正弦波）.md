## Sine（正弦波）

Sine 是一款样本源插件，其可利用正弦波形为游戏生成各种各样的声音。

您可能注意到了 Source Plug-in Editor 和 Contents Editor 中的一些属性值旁边带有标志。该标志表明属性值是否通过 RTPC 与 Game Parameter 关联。

以下表格介绍了两种 RTPC 标志：

| 标志 | 名称 | 描述 |
| --- | --- | --- |
| |  | | --- | |  | | RTPC - 开启 | 属性值已使用 RTPC 绑定到游戏中的参数值。 |
| |  | | --- | |  | | RTPC - 关闭 | 属性值未与游戏中的参数值绑定。 |
