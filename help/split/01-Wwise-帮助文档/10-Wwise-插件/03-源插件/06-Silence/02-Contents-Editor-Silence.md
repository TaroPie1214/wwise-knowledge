### Contents Editor: Silence

您可以通过 Contents Editor 快速访问与 Silence 插件相关联的属性。必须点击 Silence 源才能显示相关的列标题。

|  |  |
| --- | --- |
| [备注] | 备注 |
| 由于您无法对空白信号源插件应用 Randomizer，因此 Rand. Min 和 Rand. Max 属性可以用来指定随机化的最小和最大偏置值。 |

| 界面元素 | 描述 |
| --- | --- |
| Name | 名称。空白信号源插件的名称。 |
| Use | 采用。决定声音对象内的哪个源将会被：  - 播放。 - 包含在 SoundBank（音频包）中。  此选项仅在声音对象内具有多个源时才可见。 |
| Duration | 空白信号的时长。  默认值：1 默认  滑杆范围：0.001 至 10 输入范围：0.001 至 3,600  单位：秒 |
| Rand. Min | 随机化最小值。对 Length 值进行偏置，定义空白信号源可能持续的最短时间。  默认值：0 默认  滑杆范围：-10 至 0  输入范围：-3,600 至 0  单位：秒 |
| Rand. Max | 随机化最大值。对 Length 值进行偏置，定义空白信号源可能持续的最长时间。  默认值：0 默认  滑杆范围：0 至 10  输入范围：0 至 3,600  单位：秒 |
| Notes | 备注。有关空白信号源的其它信息。 |
|  | 向声音对象中添加新的源。  点击 Add Source（添加源）按钮时，将显示可添加源的列表。 |
