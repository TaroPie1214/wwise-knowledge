## Silence

Silence 插件是一个源插件示例，您可以用它创建不播放任何音频或振动的空白信号源。

您可能注意到了 Source Editor 和 Contents Editor 中的一些属性值旁边会带有标志。该标志表明属性值是否通过 RTPC 与 Game Parameter 关联。

| 标志 | 名称 | 描述 |
| --- | --- | --- |
| |  | | --- | |  | | RTPC - 开启 | 属性值已使用 RTPC 绑定到游戏中的参数值。 |
| |  | | --- | |  | | RTPC - 关闭 | 属性值未与游戏中的参数值绑定。 |
