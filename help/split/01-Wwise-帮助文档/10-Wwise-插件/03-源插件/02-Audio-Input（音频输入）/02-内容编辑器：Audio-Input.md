### 内容编辑器：Audio Input

通过 Contents Editor（内容编辑器）可以快速访问与 Audio Input（音频输入）相关的属性。 须点击 Audio Input 源，以显示相关列标题。

| 界面元素 | 描述 |
| --- | --- |
|  | 点击列标题区 **Configure Columns**（配置列）快捷方式（右键点击）选项。p  此时将会打开 [“Object Property Settings”一节](../../../09-参考主题/04-Project-Explorer/12-搜索和工程全局编辑/06-Object-Property-Settings.md "Object Property Settings")。指定要显示的列及其顺序。  |  |  | | --- | --- | | [备注] | 备注 | | You cannot configure the columns for Source Contents Editors. | |
| Name | 名称。Audio Input 源插件的名称。 |
| Use | 采用。决定声音对象内的哪个源将会被：  - 播放。 - 包含在 SoundBank（音频包）中。 - 此选项仅在声音对象内具有多个源时才可见。 |
| Gain | 增益。音频输出的音量或振幅。  Default value: -12  Range: -96.3 to 0  Units: dB |
| Notes | 备注。有关 Audio Input 源的详细信息。 |
|  | 向声音对象中添加新的源。  点击 Add Source（添加源）按钮时，将显示可添加源的列表。 |
