### Source Editor：Audio Input

Source Editor（源编辑器）显示所有与所选的 Audio Input（音频输入）插件相关联的基本属性。您可以设置 [Gain](../../../14-词汇表.md#glossary_gain "Gain（增益）")（增益）或应用 RTPC（实时参数控制）。

有关集成 Audio Input 的信息，请参阅 [Wwise SDK 文档](https://www.audiokinetic.com/library/?source=SDK&id=effectplugin.html)。

| 界面元素 | 描述 |
| --- | --- |
| Name | 名称。该 Audio Input 的名称。 |
| Source Plug-in（源插件） | 源插件。源插件的类型。 |
| Notes | 备注。有关 Audio Input 的其他信息。 |
| **Effect Settings** | |
| **Audio Input（音频输入）** | |
| Gain | 增益。音频输出的音量或振幅。  Default value: -12  Range: -96.3 to 0  Units: dB |
| **RTPC（实时参数控制）** | |
| RTPC 选项卡中，允许对 Audio Input 的 **Gain** 属性创建 RTPC。相关通用信息，请参阅 [“RTPC tab”一节](../../../09-参考主题/04-Project-Explorer/01-Audio-tab/05-Common-tabs-and-categories-audio-structures/02-RTPC-tab/00-RTPC-tab.md "RTPC tab") 页。 | |
