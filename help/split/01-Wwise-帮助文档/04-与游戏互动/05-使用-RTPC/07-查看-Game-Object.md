## 查看 Game Object

[游戏对象](../../14-词汇表.md#glossary_game_object "Game Object")（Game Object）有时候以复杂的方式与 RTPC 交互。为了跟踪这些交互，您可以将 RTPC 坐标图设置为显示您感兴趣的游戏对象。

**查看 RTPC 坐标图中的游戏对象的方法是：**

1. 在 RTPC 选项卡视图的顶部面板中，选择列表中带有关联游戏参数的若干 RTPC。

   曲线图中将显示相应的 Game Parameter 光标标识。
2. 在顶部面板中单击 **Show Game Objects**（显示游戏对象），然后在 Wwise 工具栏中单击 **Start to Capture**（开始捕获）。

   All your game objects will appear in the RTPC graph.
