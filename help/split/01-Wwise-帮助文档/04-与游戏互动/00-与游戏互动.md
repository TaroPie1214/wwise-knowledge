# 与游戏互动

本章节包含有关如何使用 Event、Dynamic Sequence 和 Wwise Game Sync 驱动游戏声音的概念和步骤。

- **[*管理 Event*](01-管理-Event/00-管理-Event.md "管理 Event")**– 创建 Event；添加 Action 并为其指派 Target；设置 Event Action 的属性；定义 Event Action 的作用域。
- **[*管理动态对话*](02-管理动态对话/00-管理动态对话.md "管理动态对话")**– 了解 Dynamic Dialogue 系统；使用 Dialogue Event 和 State；创建路径和后备路径；使用 Probability 和 Weight 功能。
- **[*使用 State*](03-使用-State/00-使用-State.md "使用 State")**– 创建 State Group；定义 State 之间的 Transition；将 State 指派给对象和总线。
- **[*使用 Switch*](04-使用-Switch/00-使用-Switch.md "使用 Switch")**– 创建 Switch 和 Switch Group；将 Game Parameter 值映射到 Switch；将对象指派给多个 Switch Group。
- **[*使用 RTPC*](05-使用-RTPC/00-使用-RTPC.md "使用 RTPC")**– 管理 RTPC 中所用的 Game Parameter；使用 Game Parameter 控制属性值；使用 RTPC 坐标图；构建智能音高曲线；使用 LFO、Envelope、Time Modulator；设置旁链压缩。
- **[*使用 Trigger*](06-使用-Trigger.md "使用 Trigger")**– 创建 Trigger；了解 Trigger 机制。
- **[*将 State 和 State Group 用于动态对话*](07-将-State-和-State-Group-用于动态对话.md "将 State 和 State Group 用于动态对话")**– 创建 State 和 State Group 并用于 Dynamic Dialogue（含实用示例）。
