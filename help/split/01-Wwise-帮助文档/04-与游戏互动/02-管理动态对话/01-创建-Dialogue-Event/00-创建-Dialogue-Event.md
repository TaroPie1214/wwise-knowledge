## 创建 Dialogue Event

游戏中动态对话的规则要在 Dialogue Event（对白事件）内定义。这些事件的创建涉及以下步骤：

- [“创建新的 Dialogue Event”一节](01-创建新的-Dialogue-Event.md "创建新的 Dialogue Event")
- [“定义 Dialogue Event 设置”一节](02-定义-Dialogue-Event-设置.md "定义 Dialogue Event 设置")
- [“将 State Group 添加到 Dialogue Event”一节](03-将-State-Group-添加到-Dialogue-Event.md "将 State Group 添加到 Dialogue Event")
- [“使用 Path 重现游戏中的条件”一节](04-使用-Path-重现游戏中的条件.md "使用 Path 重现游戏中的条件")
- [“将对象指派给路径”一节](05-将对象指派给路径.md "将对象指派给路径")

您可以在 Dialogue Event Editor（对白事件编辑器）中管理对白事件。

  

|  |  |
| --- | --- |
| [技巧] | 技巧 |
| 如果您需要创建多条路径，则可以使用快捷方式来创建。要加快创建过程，将 Project Explorer（工程浏览器）中的对象拖放到路径中最后一个未选中状态。Wwise 将自动创建新路径并将对象指派给该路径。有关此快捷方式的详细信息，请参阅[“Dialogue Event 技巧和经验总结”一节](../03-Dialogue-Event-技巧和经验总结.md "Dialogue Event 技巧和经验总结")。 |
