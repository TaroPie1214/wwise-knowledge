## 平台支持

本节列出了 speechEngine 插件支持的平台、架构以及 SDK 版本。

**Windows：**

- 架构：

  - x64
- SDK 版本：

  - vc160
  - vc170

**Android：**

- 架构：

  - arm64-v8a
  - x86\_x64
- SDK 版本：

  - Android NDK r25b
  - 最低 API 21

**iOS：**

- 架构：

  - iphoneos（不支持模拟器）
- SDK 版本：

  - Xcode 15
  - Xcode 16

**Linux：**

- 架构：

  - x64
