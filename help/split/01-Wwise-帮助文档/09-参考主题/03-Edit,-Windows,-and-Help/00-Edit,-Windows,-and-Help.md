## Edit, Windows, and Help

除 [Project](../01-工程/00-工程.md "工程")（工程）、[Views](../02-视图/00-视图.md "视图")（视图）和 [Layouts](../../02-入门/04-个性化您的工作空间/02-处理布局/00-处理布局.md "处理布局")（布局）外，Wwise 还设有其他一些专用菜单（详见下文相应页面）。

- [“Edit”一节](01-Edit.md "Edit")（编辑）：提供一些常用快捷选项，方便在 Wwise 工程内执行相应操作。
- [“Windows”一节](02-Windows.md "Windows")（窗口）：管理各个浮动视图。
- [“Help”一节](03-Help.md "Help")（帮助）：快速访问多个信息源。
