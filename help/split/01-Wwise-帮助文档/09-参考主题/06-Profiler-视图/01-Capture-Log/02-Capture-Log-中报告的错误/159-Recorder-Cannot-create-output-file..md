#### Recorder: Cannot create output file.

“Recorder: 无法创建输出文件。”与 Recorder 插件相关。插件无法创建输出文件。

**推荐的解决步骤：**

- 确认 Wwise 工程中提供的输出文件路径。
- 确保输出路径具备写入权限。
- 有关插件配置的更多详细信息，请参阅 [“Recorder 插件”一节](../../../../10-Wwise-插件/04-效果器插件/17-Recorder-插件.md "Recorder 插件")。
