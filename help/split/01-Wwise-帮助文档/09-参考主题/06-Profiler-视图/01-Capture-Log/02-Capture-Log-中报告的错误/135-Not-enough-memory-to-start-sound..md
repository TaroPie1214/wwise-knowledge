#### Not enough memory to start sound.

“内存不足，无法启动声音。”可用内存不足，无法分配播放声音所需的全部缓冲区和结构。在这种情况下，声音不会播放。若超出总体内存预算，则将出现此错误。在此之前，Capture Log（捕获日志）中应会显示 [“Memory allocation failed.”一节](115-Memory-allocation-failed..md "Memory allocation failed.")。有关推荐的解决步骤，请参阅 [“Memory allocation failed.”一节](115-Memory-allocation-failed..md "Memory allocation failed.")。
