#### File access error during source creation. Look for previous file errors in the log.

The sound engine encountered a file-related error when accessing the media source file. In general, this error is preceded by a "file not found" or "file permission" error above this error line in the log. That error contains the specific file and path that was used to start playing this sound.

有关详细信息，请参阅“[“File ... not found in path(s): ...”一节](72-File-...-not-found-in-path(s)-....md "File ... not found in path(s"): ...")”和“[“File ... could not be opened due to permissions, access rights or conflicting open modes. (Found in path(s): ...).”一节](70-File-...-could-not-be-opened-due-to-permissions,-a.md "File ... could not be opened due to permissions, access rights or conflicting open modes.
