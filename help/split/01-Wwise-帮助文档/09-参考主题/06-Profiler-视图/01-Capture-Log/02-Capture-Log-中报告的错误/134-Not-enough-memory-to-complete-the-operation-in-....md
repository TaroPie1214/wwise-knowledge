#### Not enough memory to complete the operation in ...

“内存不足，无法在…内完成操作。”大部分 API 函数调用都会分配一些内存。若超出总体内存预算，则将出现此错误。在此之前，Capture Log（捕获日志）中应会显示 [“Memory allocation failed.”一节](115-Memory-allocation-failed..md "Memory allocation failed.")。有关推荐的解决步骤，请参阅 [“Memory allocation failed.”一节](115-Memory-allocation-failed..md "Memory allocation failed.")。
