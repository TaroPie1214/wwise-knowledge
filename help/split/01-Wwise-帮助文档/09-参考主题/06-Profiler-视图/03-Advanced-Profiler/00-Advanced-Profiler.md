## Advanced Profiler

Advanced Profiler（高级性能分析器）显示有关声音引擎内部工作的广泛深入的信息。比如，您可以获取与声音引擎的 Memory Manager（内存管理器）中注册的内存类别相关的信息。您可以通过单击 Show Live Data 按钮 ![](../../../../images/btn_live_off.png) 来在捕获的同时查看相应信息，或者通过在坐标图中拖动 Performance Monitor 的时间光标来查看特定时间点的信息。除此之外，还可按下 **Alt+逗点** (,) 来向左移动一个音频帧，或按下 **Alt+句点** (.) 来向右移动一个音频帧。
