## Profiler 视图

三个布局涵盖了 Wwise 内不同范围的性能分析，而且每个布局都有自己特定的视图。

| Layout（布局） | 描述 | 所含视图 |
| --- | --- | --- |
| Profiler | Profiler 布局允许您在捕获时监控每个游戏元素的性能。您还可以在将 Soundcaster 触发的模拟内容集成到游戏之前监视这些模拟的性能。 | - [“Advanced Profiler”一节](03-Advanced-Profiler/00-Advanced-Profiler.md "Advanced Profiler") - [“Capture Log”一节](01-Capture-Log/00-Capture-Log.md "Capture Log") - [“Performance Monitor”一节](02-Performance-Monitor/00-Performance-Monitor.md "Performance Monitor") |
| Voice Profiler | Voice Profiler（性能分析器）布局允许监控游戏中捕获的声部，并检查驱动每个声部的结构。 | - [“Capture Log”一节](01-Capture-Log/00-Capture-Log.md "Capture Log") - [Project Explorer](../04-Project-Explorer/00-Project-Explorer.md "Project Explorer") - Property Editor - [“Transport Control”一节](../02-视图/10-Transport-Control.md "Transport Control") - [“Voice Inspector”一节](04-Voice-Inspector.md "Voice Inspector") - [“Voice Monitor”一节](09-Voice-Monitor.md "Voice Monitor") |
| Game Object Profiler | Game Object Profiler 布局允许您监控游戏中注册的游戏对象，包括发声体、听者或既是发声体又是听者的对象。 | - [“Game Object 3D Viewer”一节](11-Game-Object-3D-Viewer/00-Game-Object-3D-Viewer.md "Game Object 3D Viewer") - [“Game Sync Monitor”一节](12-Game-Sync-Monitor.md "Game Sync Monitor") |

**性能分析器附加视图**

- [“Profiler Statistics 视图”一节](13-Profiler-Statistics-视图/00-Profiler-Statistics-视图.md "Profiler Statistics 视图"): 允许您在捕获会话结束后，收集游戏中某些音频元素的相关统计信息。
