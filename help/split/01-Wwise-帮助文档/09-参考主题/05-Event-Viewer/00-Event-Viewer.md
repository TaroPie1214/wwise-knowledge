## Event Viewer

Event Viewer 中将显示已经为项目创建的 Action Event。它包含以下三个选项卡：

- [“Filtered 选项卡”一节](01-Filtered-选项卡.md "Filtered 选项卡")允许您按 Action Type 筛选 Event 列表。
- [“Current Selection 选项卡”一节](02-Current-Selection-选项卡.md "Current Selection 选项卡")允许您查看当前所选对象的 Event。
- [“Orphan 选项卡”一节](03-Orphan-选项卡.md "Orphan 选项卡")允许您查看已经创建，但尚未指派对象的 Event。

您还可以通过 Event Viewer 浏览 Event、从工程中删除 Event，并且打开 [“Event Editor”一节](../04-Project-Explorer/02-Events-选项卡/01-Event-Editor/00-Event-Editor.md "Event Editor") 编辑 Event。
