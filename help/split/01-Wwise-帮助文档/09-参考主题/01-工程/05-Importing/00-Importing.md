## Importing

Wwise 有两种不同的导入功能，用于添加声音对象及其关联的媒体文件：

- [“Audio File Importer 对话框”一节](01-Audio-File-Importer-对话框.md "Audio File Importer 对话框")导入用于Wwise 对象的媒体文件。轻松导入单个或少量音频文件也可以[使用制表符分割的文件](../../../03-设置工程/05-管理工程中的媒体文件/02-导入媒体文件/05-从用制表符分割的文本文件导入媒体文件.md "从用制表符分割的文本文件导入媒体文件")来导入大量音频文件及其工程层级结构。
- [“Voice Asset Importer 对话框”一节](02-Voice-Asset-Importer-对话框.md "Voice Asset Importer 对话框")仅导入 Sound Voice 对象，也可通过制表符分隔文件导入关联的 Random Container。适用于批量导入对话。

在以上导入界面中点击 **Import** 后，可能会弹出的对话框包括以下三种：

- [“Import Conflict Manager 对话框”一节](03-Import-Conflict-Manager-对话框.md "Import Conflict Manager 对话框")：只有当正在导入的文件与工程冲突时才会弹出此对话框。
- [“Import Completed 对话框”一节](04-Import-Completed-对话框.md "Import Completed 对话框")：成功完成导入时弹出，并显示与本次导入有关的信息。
- [“Import Stopped 对话框”一节](05-Import-Stopped-对话框.md "Import Stopped 对话框")：中途停止导入时弹出，显示已完成部分的相关说明。
