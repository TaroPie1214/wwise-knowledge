### Logs Settings

The Logs Settings dialog allows you to filter out messages, warnings, and errors displayed in three of the five tabs of the Logs view, namely the [“General 选项卡”一节](00-Logs-视图.md#general_tab_log "General 选项卡"), [“打开和关闭工程”一节](../../../03-设置工程/01-处理工程/01-管理工程.md#project_load_log "打开和关闭工程"), and [“WAAPI 选项卡”一节](00-Logs-视图.md#waapi_tab_log "WAAPI 选项卡"). 对于另外两个选项卡（[“Conversion 选项卡”一节](00-Logs-视图.md#conversion_tab_log "Conversion 选项卡") 和 [“SoundBank Generation 选项卡”一节](00-Logs-视图.md#soundbank_generation_log "SoundBank Generation 选项卡")），请转到 Project Settings（工程设置）对话框的 [“Logs 选项卡”一节](../../01-工程/08-Project-Settings/04-Logs-选项卡.md "Logs 选项卡")。

| 界面元素 | 描述 |
| --- | --- |
| Ignore | 忽略。决定在下次 Wwise 输出指定日志时是否显示对应消息类型。默认情况下将显示所有消息类型。在选中此复选框后，Wwise 将忽略该消息类型，日志中也不再予以显示。 |
| Message | 消息。描述遇到的问题。 |

|  |  |
| --- | --- |
| [技巧] | 技巧 |
| 您也可以通过直接在日志中右键单击一条或多条消息并选择 **Add Message(s) to Ignore List** 来将消息添加到 Log Ignore List。 |
