## 搜索和工程全局编辑

除了在 [“Queries 选项卡”一节](../07-Queries-选项卡/00-Queries-选项卡.md "Queries 选项卡") 内部创建针对工程全局的查询，还可通过五种不同视图对您的工程进行搜索和编辑，所有视图均可用通过多数 Project Explorer 元素的快捷菜单访问。

  

|  |
| --- |
|  |

**快捷菜单搜索选项**

- **Batch Rename**: Opens the selected elements (there must be at least two) in the [“Batch Rename 视图”一节](01-Batch-Rename-视图.md "Batch Rename 视图").
- **Show in List View**： 显示在列表视图中。在 [“List View（列表视图）”一节](02-List-View（列表视图）.md "List View（列表视图）") 中打开所选元素。
- **Show in Multi Editor**： 显示在多项编辑器中。在 [“Multi Editor”一节](03-Multi-Editor.md "Multi Editor") 中打开所选元素。
- **Show in Schematic View**： 显示在对象网络图中。在 [“Schematic View”一节](04-Schematic-View/00-Schematic-View.md "Schematic View") 中打开所选元素。
- **Find all reference**：查找所有引用。在 [“Reference View 视图”一节](05-Reference-View-视图.md "Reference View 视图") 中打开所选元素。

|  |  |
| --- | --- |
| [技巧] | 技巧 |
| 大多数搜索和编辑视图的各列内容，如 [“Query Editor”一节](../07-Queries-选项卡/01-Query-Editor.md "Query Editor")，都可以在 [“Object Property Settings”一节](06-Object-Property-Settings.md "Object Property Settings") 对话框中调整。 |
