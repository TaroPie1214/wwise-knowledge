### State Properties 对话框

The State Properties dialog allows you to add and remove State properties from the object. Unlike the Configure Columns dialog, this does not just remove from view the property column in the table. 此外，各列顺序在 State 选项卡中是固定的；您不能重新排列它们。

| 界面元素 | 描述 |
| --- | --- |
| Properties | 适用于所选对象的所有 State 属性的列表。  勾选相应复选框将属性应用到对象。 |
|  | 选择列表中的所有属性。 |
|  | 取消选择列表中的所有属性。 |
|  | 关闭 State Property 对话框并为所选对象的 State 选项卡表格应用修改。 |
|  | 关闭 State Property 对话框，不为所选对象的 State 选项卡表格应用修改。 |
