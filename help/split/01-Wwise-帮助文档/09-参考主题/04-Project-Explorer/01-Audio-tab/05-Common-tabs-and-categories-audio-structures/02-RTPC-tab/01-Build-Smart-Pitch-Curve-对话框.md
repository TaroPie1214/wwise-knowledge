##### Build Smart Pitch Curve 对话框

In the Build Smart Pitch Curve dialog, you can define the properties of a smart pitch curve. 智能音高曲线是 Wwise 使用您提供的值生成的 RTPC 曲线。这些曲线代表游戏属性值变化所导致的自然音高变化。

| 界面元素 | 描述 |
| --- | --- |
| Native (X Axis parameter) value | 原声值。在设为该 Game Parameter（游戏参数）值时，可听到对应声音的原始音高。 |
| Subdivision level | 细分等级。曲线的精度，分为 1 至 10 个等级。 |
|  | 根据原始值和细分等级创建智能音高曲线。 |
|  | Closes the dialog without building a smart pitch curve. |
