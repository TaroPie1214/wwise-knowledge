#### Opus 参数

标准版 Opus 编码器适用于所有平台，而 OpusNX 版仅适用于部分平台。两者内部算法相同，所以参数也基本一样。现介绍如下：

| 界面元素 | 描述 |
| --- | --- |
| Bitrate/quality | 码率/品质。此滑杆用于指定单个 Opus 声道的码率，即千位/秒 (kbit/s)。码率越高，声音品质越好。 |

For more general information about the Opus codec, refer to: <https://wiki.xiph.org/OpusFAQ>.
