#### PCM Parameters

PCM encoding parameters are described below:

| 界面元素 | 描述 |
| --- | --- |
| Max Bit Depth | This parameter specifies the maximum bit depth used when encoding into PCM. The default of 16 bits provides excellent quality. Setting this to 24 bits preserves the full resolution of matching source material. |
