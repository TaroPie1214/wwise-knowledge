#### Property Editor：Switch（切换开关）

您可以通过 Switch Property Editor（切换开关属性编辑器）为所选 Switch 命名或者添加说明。Switch 不得重名，并且名称只能包含字母、数字和下划线。第一个字符必须是字母或下划线。

| 界面元素 | 描述 |
| --- | --- |
| Name | Switch 的名称。 |
| Notes | 备注。有关 Switch 的其它信息。 |
