#### Contents Editor 视图：State Group

当前所选 State Group（状态组）内的全部 State 均显示在对应 Contents Editor（内容编辑器）中。您可以快速编辑与各个 state 关联的多个属性，而无需在属性编辑器中逐个打开编辑。

| 界面元素 | 描述 |

| Notes | 备注。State 的其它信息。 |
