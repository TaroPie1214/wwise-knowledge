## 从段落中移除音轨和子音轨

当您不再需要音轨或子音轨时，可以从段落中移除它们。删除音轨会同时删除其所有子音轨。

**从音乐段落移除音轨或子音轨的方法如下：**

1. 在 Music Segment Editor 中，右键点击要移除的音轨或子音轨。

   |  |  |
   | --- | --- |
   | [备注] | 备注 |
   | 您也可以在 Project Explorer 的 Audio 选项卡中移除音轨。 |
2. 在快捷菜单中，选择 **Delete**。

   所选音轨及其子音轨将从段落中删除。

|  |  |
| --- | --- |
| [备注] | 备注 |
| 删除音轨或子音轨并不会删除相关的已转码音频文件。当您清除 cache （音频缓存）时仅会从工程中删除落单文件。有关删除音频文件的详细信息，请参阅 [“清除缓存”一节](../../03-设置工程/05-管理工程中的媒体文件/06-清除缓存.md "清除缓存")。 |
