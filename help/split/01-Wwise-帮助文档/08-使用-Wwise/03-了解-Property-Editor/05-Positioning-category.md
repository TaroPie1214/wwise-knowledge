## Positioning category

The Positioning category contains positioning properties of the selected object. 您可以设置对象是使用 Speaker Panning（扬声器声像摆位）和 3D Position（3D 定位）中的一种还是两种。You can then define how each of these types of positioning is implemented. 对于对象的音频信号分配，您还可以指定通过中置扬声器播放的百分比。
