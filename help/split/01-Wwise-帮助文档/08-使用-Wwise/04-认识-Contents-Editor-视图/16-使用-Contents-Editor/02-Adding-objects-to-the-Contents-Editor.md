### Adding objects to the Contents Editor

对象加载到 Property Editor 中时，它的内容将显示在 Contents Editor 中。也可以将对象直接添加到 Contents Editor 中，将它们从 Project Explorer 拖放到 Contents Editor 中即可。记住，将对象拖放到 Contents Editor 中时，您是在将其从层级结构中的当前位置移到新位置，即 Contents Editor 中其它对象的父级对象下面。如果不想移动对象，可以按住 Ctrl 同时将对象拖拽到 Contents Editor 中，这样会复制对象，并将其添加到 Contents Editor 中。

源也可以添加到对象，可以直接将其导入到 Contents Editor 中或添加源插件。
