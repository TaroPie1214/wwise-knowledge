## Project Explorer keyboard shortcuts

| 操作 | 使用此快捷方式 |
| --- | --- |
| 前往搜索字段 | Ctrl+F |
| 折叠子对象 | 数字小键盘中的斜线键 |
| 展开节点 | 数字小键盘中的向右键或加号键 |
| 折叠节点 | 数字小键盘中的向左键或减号键 |
| 向上导航 | 向上箭头 |
| 向下导航 | 向下箭头 |
| 展开子对象 | 数字小键盘中的星号键 |
