# Profiler 键盘快捷方式

## Profiler 键盘快捷方式

| 操作 | 使用此快捷方式 |
| --- | --- |
| 打开 Profiler Settings 对话框 | Alt+G |
| 前往下一个蓝色圆圈 | Ctrl + 向上箭头键 |
| 向下前往下一个蓝色圆圈 | Ctrl + 向下箭头键 |
| 强制使用时间光标捕获日志条目 | Shift + 点击 |
