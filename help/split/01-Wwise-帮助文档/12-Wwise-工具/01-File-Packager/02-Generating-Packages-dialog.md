## Generating Packages dialog

The Generating Packages dialog displays information about each file that was added to the package during the generation process. 在生成过程中发现的任何问题都会突出显示为黄色。

| 界面元素 | 描述 |
| --- | --- |
| |  | | --- | |  |  (类型 —— 颜色) | 显示不同类型消息的对应颜色。SoundBank日志分为以下类型：  **White** -- 白色。针对一般消息。  **Yellow**（黄色）—－代表包生成警告。  **Red**（红色）－－代表包生成错误。 |
| （消息） | 已添加到包的原始文件的位置。如果在生成过程中出现问题，则有关问题的信息将显示在黄色圆圈的旁边。 |
| |  | | --- | |  | | 将日志中的信息复制到 Windows 剪贴板，以便将信息粘贴到新的文件中。 |
| |  | | --- | |  | | 停止包生成过程。 |
| |  | | --- | |  | | Closes the Building Packages dialog. |
