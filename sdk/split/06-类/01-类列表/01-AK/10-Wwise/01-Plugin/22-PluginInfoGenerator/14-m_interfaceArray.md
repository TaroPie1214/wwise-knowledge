|  Array of all used interfaces for the plug-in.  It is used as a template before instantiation to create a local temporary plug-in info. A copy of its content is made in which the host fills instances it provides. At instantiation, this local copy is merged with the instantiated plug-in's interface instances into a finalized array that is returned by the Instantiate method.
