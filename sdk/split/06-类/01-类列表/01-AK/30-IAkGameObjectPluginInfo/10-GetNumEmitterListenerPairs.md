Retrieve the number of emitter-listener pairs (rays) of the game object. A game object may have more than one position, and be listened to more than one listener. The returned value is the product of these two numbers. Use the returned value as a higher bound for the index of [GetEmitterListenerPair()](class_a_k_1_1_i_ak_game_object_plugin_info_ac64ca233a93939f8d7d41f83a2ff8c27.html#ac64ca233a93939f8d7d41f83a2ff8c27). Note that rays whose listener is irrelevant to the current context are ignored.
