## 详细描述

This class takes care of the registration of plug-ins in the [Wwise](namespace_a_k_1_1_wwise.html) engine. Plug-in developers must provide one instance of this class for each plug-in.

参见
:   - [创建声音引擎插件](soundengine_plugins.html)
