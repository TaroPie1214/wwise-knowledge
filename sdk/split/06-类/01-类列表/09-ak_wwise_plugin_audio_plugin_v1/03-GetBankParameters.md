This is the usual function that gets overridden to provide data from the Authoring plug-in to the Sound Engine plug-in. Its goal is to retrieve data (usually from the PropertySet) and write it in the proper format and order, so the Sound Engine part can retrieve it and use it.  By design, this function is called to store data, as provided in the plug-in definition XML.  Because these can be changed at run-time, the parameter block should stay relatively small. For example, it should only provide basic members. Array sets, full sections, binary data or Object Stores might be better sent using the GetPluginData function, as the system can then ask to update only a portion of the parameters.  The design pattern, in that case, is to call GetPluginData at the end of GetBankParameters with some unique in\_idParam ([AK::IAkPluginParam::ALL\_PLUGIN\_DATA\_ID](class_a_k_1_1_i_ak_plugin_param_a7d4422a73ca832f64edc0465e59be771.html#a7d4422a73ca832f64edc0465e59be771)) indicating that you want to send all the parameters to the data writer.   |  |  | | --- | --- | |  | **备注:** In order to use the in\_pDataWriter instance, your plugin must request the data writer interface: [ak\_wwise\_plugin\_host\_data\_writer\_v1](structak__wwise__plugin__host__data__writer__v1.html "Interface used to write data during sound bank generation."). |   参数  |  |  |  | | --- | --- | --- | | [in] | in\_this | Current instance of this interface. | | [in] | in\_guidPlatform | The unique ID of the queried platform | | [in] | in\_pDataWriter | A pointer to the data writer instance being used. |  返回  true if successfully putting some parameters in the bank  参见  - [生成声音包](wwiseplugin_backend.html#wwiseplugin_bank) - [Property Set](plugin_backend_model.html#wwiseplugin_propertyset)  在文件 [AudioPlugin.h](_audio_plugin_8h_source.html) 第 [96](_audio_plugin_8h_source.html#l00096) 行定义.  被这些函数引用 [AK.Wwise::Plugin::V1::AudioPlugin::Interface::Interface()](_audio_plugin_8h_source.html#l00167). |
