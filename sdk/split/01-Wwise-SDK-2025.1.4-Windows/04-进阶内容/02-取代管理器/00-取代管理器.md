# 取代管理器

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

取代管理器

声音引擎随附默认的 Memory Manager 和 Streaming Manager， 提供将声音引擎集成到游戏中所需要的基本功能。

然而，如果游戏需要更大的灵活性或更强的功能，或者您已经有 属于您自己的内存管理器、流管理器或 I/O 调度器，您可自行实现适当的接口来代替默认接口，以取代默认管理器。

有关更多信息，请参阅：

- [流播放/流管理器](streamingdevicemanager.html)
- [Memory Manager](memorymanager.html)
