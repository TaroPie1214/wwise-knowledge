注解

有些 Wwise 操作可通过命令行接口来完成，包括 SoundBank 生成。这在自动操作流程中集成 Wwise 时非常有用，比如在例行游戏构建当中一并构建音频素材。

# Wwise 命令行

以下是调用 Wwise 命令行可执行文件 WwiseConsole.exe 的两种不同方式，分别对应 Windows 和 Mac 运行环境。

在 Windows 上
:   对于 Windows，直接在 "%WWISEROOT%\Authoring\x64\Release\bin" 下调用 WwiseConsole.exe。命令行参数应遵循以下基础结构，其中 `operation` 和 `arguments` 为必填参数。

:   WwiseConsole.exe operation [arguments] [--option1 [parameters]] [--option2 [parameters]] ...

在 Mac 上
:   对于 Mac，通过 WwiseConsole.sh Shell 脚本间接调用 WwiseConsole.exe。

使用 Unix 或 Windows 式路径在 Terminal 中调用 WwiseConsole.sh Shell 脚本命令行参数。这些参数与其对应的 Windows 命令行相同：

MAC:~ username$ /Path/to/Authoring/Wwise.app/Contents/Tools/WwiseConsole.sh operation [arguments] [--option1 [parameters]] [--option2 [parameters]]

任何包含空格的字符串参数前后都要加上引号。 所有路径参数只能使用反斜杠指定，路径结尾不带反斜杠。

## 示例

下面是使用命令行生成 SoundBank 的一个示例。它会针对 Windows 和 Linux 平台生成 English (US) 语言的 SoundBank。

"%WWISEROOT%\Authoring\x64\Release\bin\WwiseConsole.exe" generate-soundbank "C:\MyProject\MyProject.wproj" --platform "Windows" "Linux" --language "English(US)"
