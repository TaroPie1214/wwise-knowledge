# 示例

[Converting external source](ak_wwise_cli_convertexternalsource_example_converting_external_source.html)   
针对所有平台使用当前工程设置对 MyProject.wproj 的外部源进行转码。

[Creating a new blank project](ak_wwise_cli_createnewproject_example_creating_a_new_blank_project.html)   
针对 Windows 和 Linux 平台创建 MyProject.wproj。

[Generating all the SoundBanks](ak_wwise_cli_generatesoundbank_example_generating_all_the_soundbanks.html)   
针对 Windows 和 Linux 平台使用 English (US) 语言生成 MyProject.wproj 的所有 SoundBank。

[Importing a tab delimited file](ak_wwise_cli_tabdelimitedimport_example_importing_a_tab_delimited_file.html)   
使用默认的 useExisting 操作在安静 (quiet) 记录模式下导入制表符分隔文件。

[Migrating a project to the most recent version](ak_wwise_cli_migrate_example_migrating_a_project_to_the_most_recent_version.html)   
迁移工程并启用详细日志记录。

[Moves a project's media IDs into its work units](ak_wwise_cli_movemediaidstoworkunits_example_moves_a_project_s_media_ids_into_its_work_units.html)   
Moves a project's file IDs from a single file <project-name>.mediaid to its work units.

[Moving a project's media IDs out of work units](ak_wwise_cli_movemediaidstosinglefile_example_moving_a_project_s_media_ids_out_of_work_units.html)   
将工程的媒体 ID 从 Work Unit 移到单个 XML 文件 (<project-name>.mediaid)。

[Opening a project](ak_wwise_cli_verify_example_opening_a_project.html)   
打开工程并启用详细日志记录。

[Running a WAAPI server](ak_wwise_cli_waapiserver_example_running_a_waapi_server.html)   
加载 MyProject.wproj 并运行 WAAPI 服务器。在必要时，先迁移工程。使用 WAMP 端口 8085 和默认的 HTTP 端口 8090。

[Updating the contents of <project-name>.mediaid](ak_wwise_cli_updatemediaidsinsinglefile_example_updating_the_contents_of_project_name_mediaid.html)   
加载工程并更新 <project-name>.mediaid（如有）的内容。
