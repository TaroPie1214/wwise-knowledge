# 编写音频插件的设计工具部分

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

编写音频插件的设计工具部分

设计工具插件包含两个文件：

- [XML 定义](plugin_xml.html)：定义模型（即插件的属性）。
- [动态库](plugin_dll.html) (.dll)：提供 SoundBank 生成和图形界面的实现。

动态库必须链接到插件的声音引擎部分。

有关插件这一部分的更多详细信息，请参阅 [编写音频插件的声音引擎部分](effectpluginsoundengine.html) 章节。

# 设计工具插件 C/C++ API

设计工具插件 C/C++ API 存放在以下位置：

- `<Wwise>/SDK/include/AK/Wwise/Plugin`

C++ API 可在 `AK::Wwise::Plugin` 命名空间下找到。

## 要求

C++ API 需要兼容 C++17 的编译器。

有关更多详细信息，请参阅 [平台要求](reference_platform.html) 章节。
