# 术语表

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

术语表

|  |  |
| --- | --- |
| 术语 | 定义 |
| Query | 查询。符合 Wwise Authoring Query Language 规范的文本字符串。    在执行查询时，始终返回 Wwise 对象序列。    有关查询语法的详细信息，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。 |
| Query Source | 查询源。定义查询从哪里查找初始对象序列。 |
| Query Transform | 查询转换。定义查询转换函数。该函数将前一部分的结果作为输入来生成另一序列。 |
| Sequence | 序列。一系列可枚举的 Wwise 对象。 |
| Input Sequence | 输入序列。左边查询源或查询转换生成的序列。 |
| Wwise Object | Wwise 对象。Wwise 工程中的对象实例。    大部分 Wwise 对象都可在 Project Explorer 中找到。比如，Sound、Bus、Blend Container、Virtual Folder、Work Unit 等。有些对象不在 Project Explorer 中显示，但可在专门的视图中找到。比如，Audio Source、Music Clip、Music Cue 等。    每个对象都具有：   - name - type - id - children - 属性 - 引用   有关各种 Wwise 对象类型的完整列表，请参阅 [Wwise 对象参考](wobjects_index.html) 章节。 |
| Wwise Object Type | Wwise 对象类型。Wwise 对象的类型。    有关各种 Wwise 对象类型的完整列表，请参阅 [Wwise 对象参考](wobjects_index.html) 章节。 |
| Accessor | 访问器。针对 Wwise 对象设置的命名字段，用于从对象检索数值。 |
| Built-in Accessor | 内置访问器。大多数 Wwise 对象自带的专用访问器。    例如：   - name - id - type - path   **注意：**内置访问器不区分大小写。 |
| Property | 属性。针对 Wwise 对象设置的命名字段，用于存储数值。    每个 Wwise 对象都定义有一系列自己的属性。如需查看各项属性的完整列表，请转到 [Wwise 对象参考](wobjects_index.html) 章节，并选择特定的对象。    比如：Volume、Pitch 和 Lowpass。    有关如何定义插件属性或使用 Custom Properties 的详细信息，请参阅 [属性 XML 描述](plugin_xml_properties.html) 章节。 |
| Property Accessor | 属性访问器。用于返回属性值的访问器。    您可以通过以下三种方式使用属性访问器：   - PROPERTY\_NAME：返回用户界面中定义的值，考虑 Override 设置（如有）。 - @PROPERTY\_NAME：返回直接在 WWU 文件中定义的属性的值，不考虑 Override 设置。 - @@PROPERTY\_NAME：与 PROPERTY\_NAME 相同，不过可在 PROPERTY\_NAME 与内置访问器名称冲突时使用。   **注意：**属性访问器不区分大小写。 |
| Reference | 引用。针对 Wwise 对象设置的命名字段，用于存储对工程中另一对象的引用。有些引用还可定义本地对象（称为自定义对象）。    每个 Wwise 对象都定义有一系列自己的引用。如需查看各项引用的完整列表，请转到 [Wwise 对象参考](wobjects_index.html) 章节，并选择特定的对象。    比如：Volume、Pitch 和 Lowpass。 |
| Reference Accessor | 引用访问器。用于返回 Wwise 对象的访问器。    您可以通过以下三种方式使用引用访问器：   - REFERENCE\_NAME：返回用户界面中定义的值，考虑 Override 设置（如有）。 - @REFERENCE\_NAME：返回直接在 WWU 文件中定义的属性的值，不考虑 Override 设置。 - @@REFERENCE\_NAME：与 REFERENCE\_NAME 相同，不过可在 REFERENCE\_NAME 与内置访问器名称冲突时使用。   **注意：**可结合使用多个引用访问器。    **注意：**引用访问器不区分大小写。 |
