重要的关键字和概念

查询始终从源开始执行，后面会实施一系列转换。每项转换都会以对象序列的形式为下一转换提供输入。查询源由 **from** 语句定义。转换由特定关键字定义，关键字各有不同的参数。比如，where volume < 0。

|  |  |
| --- | --- |
| 关键字 | 说明 |
| from | **from** 关键字用于指定将从哪里开始初始化序列。所有查询都有对应的源。倘若省略 **from** 关键字，则将工程中的所有对象隐式设为查询源。    如需详细了解 **from** 关键字都可使用哪些源，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。    为方便起见，还可省略 **from object**，来将现有对象用作查询源。    示例：  - $ from project - $ from type sound - $ from object "\\Containers\\Default Work Unit\\Hello" - $ "Event:Play\_Hello", "Bus:Main Audio Bus"  以下为使用 **from** 语句生成的序列：      有关更多详细信息和使用示例，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。 |
| where | **where** 关键字允许对输入序列的对象进行筛选，并滤掉那些与指定表达式不匹配的对象。    **where** 关键字后面须添加布尔表达式。在执行该表达式时，将检查输入序列中的每一对象。    示例：  - where volume < 0 - where name = "Hello" - where parent.name : "container" - where outputbus = "Bus:Main Audio Bus" and name =/\_[0-9]+$/  以下为使用 **where** 语句转换的序列：      有关更多详细信息和使用示例，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。 |
| select | **select** 关键字允许为输入序列的各个对象生成对象序列。    示例：  - select children - select outputbus - select ancestors - select effects  以下为使用 **select** 语句转换的序列：      有关更多详细信息和使用示例，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。 |
| ... | 除此之外，还有很多其他转换。如需了解更多详细信息，请参阅 [Wwise Authoring Query Language (WAQL) 参考](waql_reference.html) 章节。 |
