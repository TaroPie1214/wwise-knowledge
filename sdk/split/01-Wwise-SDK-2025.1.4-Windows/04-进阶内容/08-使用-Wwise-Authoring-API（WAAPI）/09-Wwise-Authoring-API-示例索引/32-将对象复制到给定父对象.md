# 将对象复制到给定父对象

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

将对象复制到给定父对象

复制由 "object" ID 指定的对象，并将副本作为 "parent" 的子对象。若 "parent" 已经包含同名子对象，则对相应对象进行重命名。

# 函数 URI

[ak.wwise.core.object.copy](ak_wwise_core_object_copy.html)

# 参数

{

"object": "{A076AA65-B71A-45BB-8841-5A20C52CE727}",

"parent": "{66666666-7777-8888-9999-AAAAAAAAAAAA}",

"onNameConflict": "rename"

}

# 结果

{

"id": "{AAAAAAAA-BBBB-CCCC-DDDD-EEEEEEEEEEEE}",

"name": "new\_object\_name",

"type": "Sound",

"path": "\\Containers\\Default Work Unit\\Sound1",

"shortId": 1234

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
