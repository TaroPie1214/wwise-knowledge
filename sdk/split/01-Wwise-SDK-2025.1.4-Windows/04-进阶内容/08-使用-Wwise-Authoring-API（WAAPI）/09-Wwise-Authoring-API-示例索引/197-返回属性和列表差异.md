# 返回属性和列表差异

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

返回属性和列表差异

返回 2 个对象之间的属性和列表差异。

# 函数 URI

[ak.wwise.core.object.diff](ak_wwise_core_object_diff.html)

# 参数

{

"source": "\\Containers\\Default Work Unit\\Source Sound",

"target": "\\Containers\\Default Work Unit\\Target Sound"

}

# 结果

{

"lists": [

"Metadata",

"RTPC"

],

"properties": [

"OverridePositioning",

"OutputBus",

"Volume",

"OverrideOutput"

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
