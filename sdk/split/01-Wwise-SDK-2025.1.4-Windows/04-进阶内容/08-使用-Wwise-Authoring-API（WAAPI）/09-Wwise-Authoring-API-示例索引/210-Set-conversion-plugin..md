|  |

| Wwise SDK 2025.1.4 - Windows |

Set conversion plugin.

Changes an object's or project's conversion plug-in to the one specified (Vorbis).

# 函数 URI

[ak.wwise.core.audio.setConversionPlugin](ak_wwise_core_audio_setconversionplugin.html)

# 参数

{

"conversion": "{6D1B890C-9826-4384-BF07-C15223E9FB56}",

"platform": "Windows",

"plugin": "Vorbis"

}

# 结果

{}
