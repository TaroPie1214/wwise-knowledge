# Commiting a file modification in the source control.

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Commiting a file modification in the source control.

The modification and the message on it is sent to the source file.

# 函数 URI

[ak.wwise.core.sourceControl.commit](ak_wwise_core_sourcecontrol_commit.html)

# 参数

{

"files": [

"//projet-file//Originals/SFX/existingFile.wav"

],

"message": "some comments about this commit"

}

# 结果

{

"log": []

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
