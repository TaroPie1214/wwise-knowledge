# 按照名称发送 Event

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

按照名称发送 Event

使用 Event 的名称将 Event 发送到声音引擎。

# 函数 URI

[ak.soundengine.postEvent](ak_soundengine_postevent.html)

# 参数

{

"event": "Play\_Music",

"gameObject": 1122334

}

# 结果

{

"return": 8

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
