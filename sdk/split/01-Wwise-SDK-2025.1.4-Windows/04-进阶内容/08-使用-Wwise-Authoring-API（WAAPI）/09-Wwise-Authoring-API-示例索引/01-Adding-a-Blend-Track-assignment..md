# Adding a Blend Track assignment.

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Adding a Blend Track assignment.

Equivalent to performing a drag-and-drop operation in the Blend Tracks Editor.

# 函数 URI

[ak.wwise.core.blendContainer.addAssignment](ak_wwise_core_blendcontainer_addassignment.html)

# 参数

{

"object": "{7A12D08F-B0D9-4403-9EFA-2E6338C197C1}",

"child": "{A076AA65-B71A-45BB-8841-5A20C52CE727}"

}
