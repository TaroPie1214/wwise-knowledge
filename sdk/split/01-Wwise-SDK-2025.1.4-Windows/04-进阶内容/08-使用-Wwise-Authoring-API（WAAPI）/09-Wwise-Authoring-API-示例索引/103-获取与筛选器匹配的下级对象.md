# 获取与筛选器匹配的下级对象

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

获取与筛选器匹配的下级对象

From all the descendants in the Containers, return the objects whose names start with 'My'. 此项使用 '^My' 正则表达式。

# 函数 URI

[ak.wwise.core.object.get](ak_wwise_core_object_get.html)

# 参数

{

"waql": "\"\\Containers\" select descendants where name = /^My/"

}

# 选项

{

"return": [

"name"

]

}

# 结果

{

"return": [

{

"name": "MySound"

},

{

"name": "MyRandomContainer"

}

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
