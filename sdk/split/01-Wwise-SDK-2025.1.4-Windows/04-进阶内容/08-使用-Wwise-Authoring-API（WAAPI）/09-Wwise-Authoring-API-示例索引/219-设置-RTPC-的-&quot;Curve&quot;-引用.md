# 设置 RTPC 的 &quot;Curve&quot; 引用

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

设置 RTPC 的 "Curve" 引用

设置给定 RTPC 的曲线控制点。

# 函数 URI

[ak.wwise.core.object.set](ak_wwise_core_object_set.html)

# 参数

{

"objects": [

{

"object": "{5A1EC9A4-5C47-4555-82EF-50722C168C0C}",

"@Curve": {

"type": "Curve",

"points": [

{

"x": 100,

"y": 20.82785,

"shape": "Linear"

},

{

"x": 1000,

"y": -21.8509,

"shape": "Linear"

}

]

}

}

]

}

# 结果

{}
