# 统一获取各个声道的峰值

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

统一获取各个声道的峰值

在修剪区域内获取 Audio Source 对象的峰值。

# 函数 URI

[ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInTrimmedRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksintrimmedregion.html)

# 参数

{

"object": "{AAAAAAAA-BBBB-CCCC-DDDD-EEEEEEEEEEEE}",

"numPeaks": 750,

"getCrossChannelPeaks": true

}

# 结果

{

"peaksBinaryStrings": [

"longbase64string"

],

"numChannels": 1,

"maxAbsValue": 32767,

"peaksArrayLength": 750,

"peaksDataSize": 1000

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
