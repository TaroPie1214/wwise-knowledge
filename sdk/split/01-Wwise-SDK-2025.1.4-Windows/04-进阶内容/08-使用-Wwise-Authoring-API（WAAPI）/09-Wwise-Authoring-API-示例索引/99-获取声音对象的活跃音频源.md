# 获取声音对象的活跃音频源

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

获取声音对象的活跃音频源

使用 "activeSource" 访问器获取声音的活跃音频源。

# 函数 URI

[ak.wwise.core.object.get](ak_wwise_core_object_get.html)

# 参数

{

"waql": "\"{24979032-B170-43E3-A2E4-469E0193E2C4}\""

}

# 选项

{

"return": [

"id",

"name",

"activeSource"

]

}

# 结果

{

"return": [

{

"id": "{24979032-B170-43E3-A2E4-469E0193E2C4}",

"name": "My Tone",

"activeSource": {

"id": "{B7026191-28B7-4CFF-AE5A-4F0FD63607EC}",

"name": "sine\_880hz\_48k\_21ms"

}

}

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
