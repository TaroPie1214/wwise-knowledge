# Retrieving the current allocation for a layout element

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Retrieving the current allocation for a layout element

Returns the current allocated rectangle of a layout element, identified by its GUID.

# 函数 URI

[ak.wwise.ui.layout.getElementRectangle](ak_wwise_ui_layout_getelementrectangle.html)

# 参数

{

"id": "{39949670-A96A-47C5-8EEF-F8AD26BD6B34}"

}

# 结果

{

"left": 50,

"top": 50,

"right": 250,

"bottom": 75

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
