# Set Game Parameter range

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Set Game Parameter range

Equivalent to clicking the Edit button for Min or Max and setting the range of a Game Parameter.

# 函数 URI

[ak.wwise.core.gameParameter.setRange](ak_wwise_core_gameparameter_setrange.html)

# 参数

{

"object": "\\Game Parameters\\Default Work Unit\\MyGameParam",

"min": 10,

"max": 100,

"onCurveUpdate": "stretch"

}

# 结果

{}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
