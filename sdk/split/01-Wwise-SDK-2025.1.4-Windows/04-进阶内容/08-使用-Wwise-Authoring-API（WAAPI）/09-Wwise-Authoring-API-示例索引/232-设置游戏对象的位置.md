# 设置游戏对象的位置

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

设置游戏对象的位置

按照给定朝向将 "gameObject" 的位置设为 "position/position"。

# 函数 URI

[ak.soundengine.setPosition](ak_soundengine_setposition.html)

# 参数

{

"gameObject": 1122334,

"position": {

"orientationFront": {

"x": 100,

"y": 200,

"z": 0

},

"orientationTop": {

"x": 0,

"y": 0,

"z": 200

},

"position": {

"x": 5,

"y": 10,

"z": 100

}

}

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
