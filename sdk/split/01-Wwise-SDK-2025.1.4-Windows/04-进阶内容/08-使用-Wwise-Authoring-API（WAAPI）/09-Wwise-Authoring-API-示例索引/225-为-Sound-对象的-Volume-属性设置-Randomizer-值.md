# 为 Sound 对象的 Volume 属性设置 Randomizer 值

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

为 Sound 对象的 Volume 属性设置 Randomizer 值

为由 "object" 定义的对象的 "Volume" 属性设置 Randomizer 值。

# 函数 URI

[ak.wwise.core.object.setRandomizer](ak_wwise_core_object_setrandomizer.html)

# 参数

{

"object": "{A076AA65-B71A-45BB-8841-5A20C52CE727}",

"property": "Volume",

"platform": "{66666666-7777-8888-9999-AAAAAAAAAAAA}",

"enabled": true,

"min": -8,

"max": 6

}

# 结果

{}
