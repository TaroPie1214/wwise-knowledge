# Adding a Blend Track to a Blend Container

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Adding a Blend Track to a Blend Container

Equivalent to clicking New Blend Track in the Blend Track Editor.

# 函数 URI

[ak.wwise.core.blendContainer.addTrack](ak_wwise_core_blendcontainer_addtrack.html)

# 参数

{

"object": "{7A12D08F-B0D9-4403-9EFA-2E6338C197C1}",

"name": "newBlendTrack"

}

# 结果

{

"id": "{D7126155-914E-4248-A8C1-4DABCCACA016}"

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
