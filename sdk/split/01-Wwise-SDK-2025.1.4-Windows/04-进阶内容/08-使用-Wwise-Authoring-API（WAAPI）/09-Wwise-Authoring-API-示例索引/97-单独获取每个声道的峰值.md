# 单独获取每个声道的峰值

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

单独获取每个声道的峰值

获取 Audio Source 对象的峰值。

# 函数 URI

[ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksinregion.html)

# 参数

{

"object": "{AAAAAAAA-BBBB-CCCC-DDDD-EEEEEEEEEEEE}",

"timeFrom": 0,

"timeTo": 2,

"numPeaks": 750,

"getCrossChannelPeaks": false

}

# 结果

{

"peaksBinaryStrings": [

"longbase64string",

"longbase64string"

],

"numChannels": 2,

"maxAbsValue": 32767,

"peaksArrayLength": 750,

"peaksDataSize": 1000

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
