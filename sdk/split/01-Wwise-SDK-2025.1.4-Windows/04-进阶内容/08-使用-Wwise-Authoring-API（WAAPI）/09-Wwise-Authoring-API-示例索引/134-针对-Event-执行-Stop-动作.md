# 针对 Event 执行 Stop 动作

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

针对 Event 执行 Stop 动作

针对给定 Event 执行 Stop 动作，在 5 秒内按照 log1 曲线淡出。

# 函数 URI

[ak.soundengine.executeActionOnEvent](ak_soundengine_executeactiononevent.html)

# 参数

{

"event": "{FF59687C-48CF-4385-B1C5-CE84B0A63880}",

"actionType": 0,

"gameObject": 1122334,

"transitionDuration": 5000,

"fadeCurve": 2

}

# 结果

{}
