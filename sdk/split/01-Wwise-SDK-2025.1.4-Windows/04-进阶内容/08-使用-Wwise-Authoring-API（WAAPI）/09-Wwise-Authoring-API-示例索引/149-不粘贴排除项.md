# 不粘贴排除项

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

不粘贴排除项

将存在差异的属性、引用和列表从源对象粘贴到目标对象。为列表选择非默认 Paste Mode 并粘贴 "Volume" 属性和 "RTPC" 列表以外的所有条目。

# 函数 URI

[ak.wwise.core.object.pasteProperties](ak_wwise_core_object_pasteproperties.html)

# 参数

{

"source": "\\Containers\\Default Work Unit\\Source Sound",

"targets": [

"\\Containers\\Default Work Unit\\Target Sound",

"\\Containers\\Default Work Unit\\Target Sound2"

],

"pasteMode": "addReplace",

"exclusion": [

"Volume",

"RTPC"

]

}

# 结果

{}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
