| Wwise SDK 2025.1.4 - Windows |

搜索特定类型的对象

检索“文本”属性中带有 "Tone" 的对象的 ID、名称和备注，并对结果进行筛选来仅获取 "Sound" 类型的对象。See [Wwise 对象参考](wobjects_index.html) for the types available.

# 函数 URI

[ak.wwise.core.object.get](ak_wwise_core_object_get.html)

# 参数

{

"waql": "from search \"Tone\" where type = \"Sound\""

}

# 选项

{

"return": [

"id",

"name",

"notes"

]

}

# 结果

{

"return": [

{

"id": "{24979032-B170-43E3-A2E4-469E0193E2C3}",

"name": "Tone SFX",

"notes": "MyNotes"

},

{

"id": "{A076AA65-B71A-45BB-8841-5A20C52CE727}",

"name": "MyTone",

"notes": ""

},

{

"id": "{72CB44BE-C699-4032-B615-EEFD0E707FFC}",

"name": "MyToneToMove",

"notes": ""

}

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
