Undock a view from a layout

Creates a floating view, at the given position, from a docked view in a layout.

# 函数 URI

[ak.wwise.ui.layout.undockView](ak_wwise_ui_layout_undockview.html)

# 参数

{

"name": "Designer",

"viewID": "{6609D1BA-0E47-48F4-B3B6-3DA1C345B66E}",

"posX": 200,

"posY": 100

}

{}
