# 获取连接状态

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

获取连接状态

检索当前连接状态。在本例中，Wwise 被连接到了 Integration Demo。

# 函数 URI

[ak.wwise.core.remote.getConnectionStatus](ak_wwise_core_remote_getconnectionstatus.html)

# 参数

{}

# 结果

{

"isConnected": true,

"console": {

"name": "PC-USERNAME",

"platform": "Windows",

"host": "127.0.0.1",

"customPlatform": "Windows",

"appName": "Integration Demo"

},

"status": "Connected to PC-USERNAME (Windows - Local host) Integration Demo"

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
