# Convert a WwiseObject

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Convert a WwiseObject

Converts the audio file of the specified object onto specified platforms and languages.

# 函数 URI

[ak.wwise.core.audio.convert](ak_wwise_core_audio_convert.html)

# 参数

{

"objects": [

"\\Containers\\Default Work Unit\\Hello"

],

"platforms": [

"Windows"

],

"languages": [

"SFX"

]

}

# 结果

{

"errors": []

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
