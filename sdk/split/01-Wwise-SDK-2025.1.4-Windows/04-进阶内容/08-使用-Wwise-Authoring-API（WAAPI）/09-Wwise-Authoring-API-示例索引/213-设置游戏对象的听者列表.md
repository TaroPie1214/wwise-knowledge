# 设置游戏对象的听者列表

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

设置游戏对象的听者列表

将 "listeners" 设为 "gameObject" 的活跃听者列表。

# 函数 URI

[ak.soundengine.setListeners](ak_soundengine_setlisteners.html)

# 参数

{

"emitter": 1122334,

"listeners": [

2233445,

5566778,

123456,

11222648

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
