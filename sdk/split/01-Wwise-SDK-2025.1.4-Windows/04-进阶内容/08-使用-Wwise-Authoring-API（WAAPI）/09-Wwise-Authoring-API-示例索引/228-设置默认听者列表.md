# 设置默认听者列表

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

设置默认听者列表

将 "listeners" 设为默认听者列表。

# 函数 URI

[ak.soundengine.setDefaultListeners](ak_soundengine_setdefaultlisteners.html)

# 参数

{

"listeners": [

2233445,

5566778,

123456,

11222648

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
