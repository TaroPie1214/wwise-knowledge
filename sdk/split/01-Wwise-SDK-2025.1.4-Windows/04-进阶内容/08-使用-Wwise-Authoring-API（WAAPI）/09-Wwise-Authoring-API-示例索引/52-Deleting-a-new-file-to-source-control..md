# Deleting a new file to source control.

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Deleting a new file to source control.

Remove an existing file to the SFX directory and the source control.

# 函数 URI

[ak.wwise.core.sourceControl.delete](ak_wwise_core_sourcecontrol_delete.html)

# 参数

{

"files": [

"//projet-file//Originals/SFX/existingFile.wav"

]

}

# 结果

{

"log": []

}
