# Adding a new files to source control.

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Adding a new files to source control.

Add two new audio files to the SFX directory and the source control.

# 函数 URI

[ak.wwise.core.sourceControl.add](ak_wwise_core_sourcecontrol_add.html)

# 参数

{

"files": [

"//projet-file//Originals/SFX/newFile1.wav",

"//projet-file//Originals/SFX/newFile2.wav"

]

}

# 结果

{

"log": []

}
