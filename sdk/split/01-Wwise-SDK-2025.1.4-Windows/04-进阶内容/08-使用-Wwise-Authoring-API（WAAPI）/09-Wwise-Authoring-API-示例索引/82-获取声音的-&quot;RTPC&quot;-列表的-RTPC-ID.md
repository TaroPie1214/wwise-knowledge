# 获取声音的 &quot;RTPC&quot; 列表的 RTPC ID

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

获取声音的 "RTPC" 列表的 RTPC ID

使用列表访问器 @ 引用 "RTPC" 列表。注意，RTPC 未命名。

# 函数 URI

[ak.wwise.core.object.get](ak_wwise_core_object_get.html)

# 参数

{

"waql": "\"{24979032-B170-43E3-A2E4-469E0193E2C3}\""

}

# 选项

{

"return": [

"@RTPC"

]

}

# 结果

{

"return": [

{

"@RTPC": [

{

"id": "{67A2C1B5-4FBF-4938-A86C-4AEE7E84822A}"

},

{

"id": "{30D99E75-A579-4163-8FEC-012985D0B451}"

}

]

}

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
