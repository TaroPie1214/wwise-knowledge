# 设置对象引用

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

设置对象引用

为给定对象设置 Output Bus。

# 函数 URI

[ak.wwise.core.object.setReference](ak_wwise_core_object_setreference.html)

# 参数

{

"object": "\\Containers\\Default Work Unit\\MySound",

"reference": "OutputBus",

"value": "\\Busses\\Default Work Unit\\Main Audio Bus\\Environment Bus"

}

# 结果

{}
