# 获取工程名称、路径和未同步状态

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

获取工程名称、路径和未同步状态

检索工程名称、文件路径和未同步状态。注意，工程名称在 WPROJ MXL 中定义，其有可能跟文件名不一致。For using WAAPI on Mac, please see [在 Mac 上使用 WAAPI](waapi_prepare.html#waapi_path_on_mac) .

# 函数 URI

[ak.wwise.core.object.get](ak_wwise_core_object_get.html)

# 参数

{

"waql": "\"\\\""

}

# 选项

{

"return": [

"name",

"filePath",

"workunitIsDirty"

]

}

# 结果

{

"return": [

{

"filePath": "C:\\WwiseProjects\\MyProject\\MyProject.wproj",

"name": "MyProject",

"workunitIsDirty": true

}

]

}

有关 Wwise Authoring API 的详细信息，请参阅 [使用 Wwise Authoring API（WAAPI）](waapi.html) 章节。
