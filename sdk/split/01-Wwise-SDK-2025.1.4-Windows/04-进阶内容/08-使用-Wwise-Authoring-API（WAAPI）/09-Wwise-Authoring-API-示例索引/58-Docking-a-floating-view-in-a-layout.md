Docking a floating view in a layout

Docks a floating view, using its GUID, in a specific panel of a layout.

# 函数 URI

[ak.wwise.ui.layout.dockView](ak_wwise_ui_layout_dockview.html)

# 参数

{

"name": "Designer",

"viewID": "{6609D1BA-0E47-48F4-B3B6-3DA1C345B66E}",

"targetID": "{39949670-A96A-47C5-8EEF-F8AD26BD6B34}",

"side": "left"

}

# 结果

{}
