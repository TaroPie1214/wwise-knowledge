|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

Set state properties to a Sound

Sets BypassEffects and Volume state properties to a Sound.

# 函数 URI

[ak.wwise.core.object.setStateProperties](ak_wwise_core_object_setstateproperties.html)

# 参数

{

"object": "\\Containers\\Default Work Unit\\MySound",

"stateProperties": [

"BypassEffects",

"Volume"

]

}

# 结果

{}
