|  |

按照名称发送 Trigger

按照名称发送给定 "trigger" 以作用于 "gameObject"。

# 函数 URI

[ak.soundengine.postTrigger](ak_soundengine_posttrigger.html)

# 参数

{

"trigger": "MyTrigger",

"gameObject": 1122334

}

# 结果

{}
