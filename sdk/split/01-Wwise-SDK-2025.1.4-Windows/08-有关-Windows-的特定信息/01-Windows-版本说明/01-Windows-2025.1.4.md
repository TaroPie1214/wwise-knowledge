# 构建要求

- **在使用 Visual Studio 2019 构建时**：Windows 10 SDK 2104 (10.0.20348.0)
- **在使用 Visual Studio 2022 构建时**：Windows SDK for Windows 11 (10.0.22621.0)

# 漏洞修复

- **WG-80968** 已修复：在 Windows 上的 Integration Demo 中将来自输入设备的输入注册了两次。
- **WG-81241** 已修复：在初始化之后马上终止声音引擎可能会引发崩溃。

# 社区报告的漏洞修复

- **WG-81205** 已修复：(Acoustics) 在 Portal 重叠时出现不符合预期的声笼行为。
