# Wwise SDK

Wwise® 是一款针对游戏开发的综合音频中间件解决方案，可创建精密、丰富的互动音频。它由紧密集成的设计工具和声音引擎构成，可让声音设计师和程序员在更短的时间内更经济地完成令人震憾的音频。

Wwise 声音引擎由两部分构成：一是提供全套功能的跨平台行为层，二是针对各平台高度优化的底层部分。声音引擎的开发采用了易于集成的理念，因此充分利用游戏中的高级音频功能易如反掌。
