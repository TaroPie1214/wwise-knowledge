声音引擎集成纵览

在使用 Wwise SDK 将 Wwise 声音引擎整合到游戏中时，需要执行七项基本任务。

**1.** [SDK 中所含库和头文件的概述](libraries.html)   
开始的时候最好先确定工程需要什么库和头文件。

**2.** [定义 C++ 工程设置](projectsettings.html)   
然后您需要在您的 C++ 工程中定义具体的设置。

**3.** [初始化声音引擎模块](workingwithsdks_initialization.html)   
真正的任务从初始化内存管理器、流管理器、声音引擎和通信开始。

**4.** [创建重复调用来执行音频处理](workingwithsdks_framerendering.html)   
在不同的模块初始化后，您需要设置重复的处理调用。

**5.** [终止声音引擎的不同模块](workingwithsdks_termination.html)   
然后终止您先前初始化的所有模块。

**6.** [处理系统专用事件](workingwithsdks_system_calls.html)   
请勿忘记处理各种 OS 特定事件（后台应用程序、Home 按钮、处理用户音乐）

**7.** [将 Wwise 元素集成到游戏中](workingwithsdks_integratingelements.html)   
最后一步，集成您想在游戏引擎集成中使用的元素，例如游戏对象、事件和 SoundBank。
