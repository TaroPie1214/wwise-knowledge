# 切换开关示例

设置切换开关（Switch）与设置状态非常相似，不同的是前者通过使用 [AK::SoundEngine::SetSwitch()](namespace_a_k_1_1_sound_engine_a9c9d57ba7b3af2b896e26297a7657264.html#a9c9d57ba7b3af2b896e26297a7657264) 函数来完成：

[AK::SoundEngine::SetSwitch](namespace_a_k_1_1_sound_engine_a9c9d57ba7b3af2b896e26297a7657264.html#a9c9d57ba7b3af2b896e26297a7657264)( AK::SWITCHES::GROUND::GROUP, AK::SWITCHES::GROUND::SWITCH::GRAVEL, GAME\_OBJECT\_ID\_HUMAN );

除 ID [外，AK::SoundEngine::SetSwitch()](namespace_a_k_1_1_sound_engine_a9c9d57ba7b3af2b896e26297a7657264.html#a9c9d57ba7b3af2b896e26297a7657264) 方法也接受字符串（Unicode 或 Ansi）。在这种情况下，您将指定由音频设计师定义的切换开关组（Switch Group）和切换开关名称：

[AK::SoundEngine::SetSwitch](namespace_a_k_1_1_sound_engine_a9c9d57ba7b3af2b896e26297a7657264.html#a9c9d57ba7b3af2b896e26297a7657264)( L"Ground", L"Gravel", GAME\_OBJECT\_ID\_HUMAN );

在示例程序中，您可以试验切换开关，方法是单击 Start Walking 按钮（以开始每半秒钟触发一次`Play_FootStep` 事件），然后使用“Ground switch”下拉列表更换地面材料。

请参阅 [集成详情——切换开关](soundengine_switch.html) 了解更多详情。
