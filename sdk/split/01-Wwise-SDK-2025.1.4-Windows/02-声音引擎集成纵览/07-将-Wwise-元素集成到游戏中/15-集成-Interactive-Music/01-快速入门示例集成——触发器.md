# 触发器示例

您可以使用 [AK::SoundEngine::PostTrigger()](namespace_a_k_1_1_sound_engine_a1cd4b83f1a684a70e009cc9d81bbe57a.html#a1cd4b83f1a684a70e009cc9d81bbe57a) 函数启动触发器（trigger）：

[AK::SoundEngine::PostTrigger](namespace_a_k_1_1_sound_engine_a1cd4b83f1a684a70e009cc9d81bbe57a.html#a1cd4b83f1a684a70e009cc9d81bbe57a)( AK::TRIGGERS::PUZZLE\_SOLVED, GAME\_OBJECT\_ID\_HUMAN );

除 ID [外，AK::SoundEngine::PostTrigger()](namespace_a_k_1_1_sound_engine_a1cd4b83f1a684a70e009cc9d81bbe57a.html#a1cd4b83f1a684a70e009cc9d81bbe57a) 方法也接受字符串（Unicode 或 Ansi）。在这种情况下，您将指定由音频设计师定义的触发器名称：

[AK::SoundEngine::PostTrigger](namespace_a_k_1_1_sound_engine_a1cd4b83f1a684a70e009cc9d81bbe57a.html#a1cd4b83f1a684a70e009cc9d81bbe57a)( L"Puzzle\_Solved", GAME\_OBJECT\_ID\_HUMAN );

请参阅 [集成详情——触发器](soundengine_triggers.html) 了解更多详情。
