# 新增功能

- **WG-66725** 在 Wwise Project Settings 中，现在可指定保存工程和 SoundBank 文件时应用的行结束符 (LF 或 CRLF)。

# 漏洞修复

- **WG-60363** 已修复：(Mac Authoring) 在路径包含 Unicode 字符时，无法导入音频文件。
- **WG-66813** 已修复：有些后台任务存在延迟。

# 社区报告的漏洞修复

- **WG-45904** 已修复：在播放期间更改 Music Segment 位置时可能会发生崩溃或触发断言。
- **WG-53851** 已修复：在使用 Voice Inspector 的情况下重新启动 Profiler 会话会引发崩溃。
- **WG-65198** 已修复：在发生文件转码错误或版本控制错误后，Vorbis 转码发生崩溃。
- **WG-65670** 已修复：在重新加载 Work Unit 时，AK Convolution Reverb 曲线 ID 发生冲突。
- **WG-65746** 已修复：在重新加载 Work Unit 后，Media ID 发生了变化。
- **WG-65759** 已修复：(Spatial Audio) 在将听者放在与 Portal 的墙壁平行的位置时，衍射路径有时会丢失。
- **WG-66279** 已修复：在有些使用 Emitter with Automation 的声音仍处于播放状态的情况下终止声音引擎可能会引发崩溃。
- **WG-66419** 已修复：Subversion Commit 对话框中不显示经过修改的文件。
- **WG-66428** 已修复：在频繁使用磁盘时，由于使用 AkFilePackageLowLevelIO 而导致出现声部匮乏（CPU 等待）。
- **WG-67005** 已修复：在通过 WAAPI Import 导入本地化 WAV 文件时没有顾及给定对象路径。
