# 新增功能

# 漏洞修复

- **WG-30459** 已修复：(Mac Authoring) Option 键没有映射到 Alt 键。
- **WG-49993** 已修复：在过渡期间销毁游戏对象时，Voice Inspector 无法正常报告过渡。
- **WG-50165** 已修复：在将某些 DirectInput 设备连接到电脑时，设计工具出现短暂的 UI 卡顿。
- **WG-50638** 已修复：在播放包含空白关联列表的 Music Switch Container 时出现访问冲突。
- **WG-51166** 已修复：在通过延迟型 I/O 实现方式准备媒体时触发断言。
- **WG-51425** 已修复：在基准值为 1 时，Random/Sequence Container 中的 Loop Count 随机化不起作用。
- **WG-51513** 已修复：(Mac Authoring) 文本在文本编辑控件中被粘贴了两次。
- **WG-51532** 已修复：(Mac Authoring) Cmd 没有映射到 Ctrl 键。
- **WG-52243** 已修复：在使用 Opus 编解码器处理特定内容时可能会发生崩溃。
- **WG-52513** 已修复：在请求中止已经中止过一次的延迟操作时发生崩溃。
- **WG-52754** 已修复：在执行 AkTransitionManager::ProcessTransitionList 时，声音引擎发生罕见崩溃。

# 漏洞修复

- **WG-51498** 已修复：AK::SoundEngine 命名空间引用页中缺少函数。
