# 平台 SDK 的变化

- **Linux**：已将工具链更新至 Clang 3.8。
- **Lumin**：已更新至 SDK 0.19。
- **Mac, iOS, tvOS**：已更新至 Xcode 10.1。
- **Switch SDK**：已更新到 SDK 7.4。

# 性能变化

- **WG-42578** 修复了启用 Voice Inspector 数据时性能方面存在的小问题。

# API 变化

- **WG-42595** (WAAPI) 现在需要 [ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksinregion.html) 和 [ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInTrimmedRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksintrimmedregion.html) 参数（`getCrossChannelPeaks` 除外）。`numPeaks` 参数现在为无符号整数。

# 其它更改

- **WG-41269** 为了方便理解，在 WAAPI 参考文档中添加了示例库。
- **WG-41484** 提供了新的 C# WAAPI 客户端，程序员操作起来更加便捷。
- **WG-41562** 在 Voice Inspector 的 Game Objects 和 Voices List 面板中添加了游戏对象快捷菜单。
- **WG-42050** 在 Voice Inspector 的 Contribution List 面板中添加了 Reset Column Widths to Default 快捷菜单选项。
- **WG-42545** 更新了 Voice Inspector，衰减行现在显示射线图标而非游戏对象图标，同时将游戏对象分隔符由 | 改为了 -->。
- **WG-42649** 移除了名称中包含不支持字符且未被引用的皮肤文件。

- **WG-40521** 已修复：在播放 ASIO 源时断开 ASIO 设备会引发崩溃。
- **WG-41128** 已修复：(Linux) 无法加载 DSP 插件。
- **WG-41602** 已修复：Voice Inspector 无法针对音频源对象正确报告 Modulator。
- **WG-42103** 已修复：Project Load Log 反复显示曲线范围导致的重复报错。
- **WG-42220** 已修复：无法在 Voice Inspector 的 Contribution List 中正常进行搜索。
- **WG-42229** 已修复：Voice Inspector 无法报告音乐过渡的淡入或淡出。
- **WG-42234** 已修复：Voice Inspector 报告的 Music Segment LPF 和 HPF 包络值有误。
- **WG-42235** 将 Voice Inspector Contribution List 中的 Automation 重命名为了 Music Segment Envelope。
- **WG-42238** 已修复：在使用 Voice Inspector 时，淡入和淡出引发断言。
- **WG-42278** 已修复：在浏览器中选择带有活跃滤波器的 User-Defined Auxiliary Sends 时，辅助总线不生效。
- **WG-42284** 已修复：在将条目从语音/音效对应的 Contents Editor 拖到 Dialogue Event Editor 时发生崩溃。
- **WG-42286** 已修复：在筛选结果时，Voice Inspector 的 Game Object 和 Voices List 面板中的列表收起。
- **WG-42368** 已修复：在浏览音乐过渡的源/目标对象时，Multi Editor 引发断言。
- **WG-42393** 已修复：Voice Inspector 没有针对 Switch Container 所应用的全局 RTPC 报告数值。
- **WG-42406** 已修复：Voice Inspector 没有针对辅助发送报告 Live Edit 条目。

- **WG-42426** 已修复：在将鼠标悬停于 Schematic View 中的元素之上时，工具提示闪烁不定。
- **WG-42440** 已修复：在打开 REV2 Source Editor 视图的情况下生成 SoundBank 时，有时会出现数据丢失。
- **WG-42473** 已修复：在通过 State 变化进行触发时，Voice Inspector 没有针对 MIDI 音符报告 Fade In/Out 条目。
- **WG-42504** 已修复：在 RTPC 选项卡中查看 Game Object 轨迹时发生罕见崩溃。
- **WG-42506** 已修复：在与 Schematic View 进行交互时发生罕见崩溃。
- **WG-42526** 已修复：在滚动时间线时，Voice Inspector 报告的 Attenuation 值出现刷新问题。
- **WG-42613** 已修复：在 Release 版本中，未初始化的滤波器无法有效关联 LPF 和 HPF 值。
- **WG-42615** 已修复：在发送 Stop Event 后在同一帧内触发同一音乐时引发断言。
- **WG-42627** 已修复：在删除搜索结果中的顶部条目后重新创建该条目时发生崩溃。
- **WG-42635** 已修复：在从反馈循环进行过渡时，Voice Inspector 发生崩溃。
- **WG-42653** 已修复：在调整 Schematic View 大小时，视图闪烁不定。
- **WG-42654** 已修复：在游戏中，没有将 Make-up Gain 属性应用到总线。
- **WG-42683** 已修复：Generate SoundBanks 视图的进度条没有反应。
- **WG-42721** 已修复：XAudio2 在终止时挂起。
- **WG-42724** 已修复：有时不好区分 Voices Graph 选项卡中的 Auxiliary Bus 虚线路径。

- **WG-42824** 已修复：XAudio2 sink 有时会出现并发问题。
- **WG-42826** 已修复：`SpatialAudio::SetGeometry()` 内存不足，导致发生崩溃。
- **WG-42858** 已修复：在有些 Spatial Audio 错误消息中，游戏对象 ID 被截断成 32 位。
