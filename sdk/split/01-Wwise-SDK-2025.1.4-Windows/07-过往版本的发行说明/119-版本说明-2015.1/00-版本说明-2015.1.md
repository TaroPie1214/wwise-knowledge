# 平台 SDK 的变化

- 将 Vita 更新至 SDK 3.5。
- 将 Mac 更新至 SDK 10.9。
- 将 iOS 更新至 SDK 8.3。
- 将 Perforce 插件更新至 Perforce SDK 2015.1。
- 将 Wii U 更新至 SDK 2.12.13。
- 添加了 Visual Studio 2015 RC。

# 新功能

- **WG-3006** 在 Wwise 工程设置中添加了使用快捷菜单创建事件的功能。
- **WG-13227** 添加了事件动作“Post Event”。
- **WG-14398** 添加了事件动作“Reset Playlist”。
- **WG-15494** 添加了通过新的 Batch Rename 视图重新命名多个对象的功能。
- **WG-18296** 在 SoundFrame 中暴露了 SoundBank。
- **WG-18954** 在 Conversion Settings Editor 中添加了 Min Sample Rate 和 Max Sample Rate 列，以及“Allow channel upmix”选项。
- **WG-23258** 替换已弃用的 iOS API 引用，以避免用户音乐造成音频中断。
- **WG-25961** 在 SoundFrame 中添加了对 SeekOnEvent 的支持。
- **WG-26106** 为 Sony Morpheus 添加了 3D 音频支持。
- **WG-26436** 在 Advanced Profiler 视图中添加了事件和播放目标的信息。
- **WG-26466** 在 RTPC 选项卡和 Attenuation 页面中为游戏对象监视添加了光标。
- **WG-26626** 修正了 iOS Integration Demo 中的“Mute for Background Music”实现。
- **WG-26763** 添加了通过 Wwise 连接到 Nuendo 进行互动的功能。
- **WG-26880** 添加了从 WwiseCLI.exe 覆盖 .cache 文件夹路径的选项。
- **WG-27004** 向混音器插件暴露了距离调制优先级。

请参阅 [新功能概述2015.1](whatsnew_2015_1_new_features.html) ，了解更多详情 。

# API 变化

- **WG-23258** 将 iOS 音频会话服务支持迁移到 AVFoundation，以更好地处理应用程序音频会话。这会影响音频会话中断回调和声音引擎初始化，还可提供 AVFoundation 支持的新功能。移除了 ListenToAudioSessionInterruption() API。
- **WG-26235** 添加了 [AkInitSettings::bUseLEngineThread](struct_ak_init_settings_a31218fafe5dea9bca79b979cfb1e6fb2.html#a31218fafe5dea9bca79b979cfb1e6fb2 "Use a Wwise-owned thread for processing audio. If set to false, audio processing will occur only insi...") 参数，此参数可禁用音频渲染线程，以允许进行自定义调度。
- **WG-26577** 移除了插件 API 的 PlayAudioInput()。
- **WG-26909** 向混音器插件暴露了每对发射器和听者的衰减。
- **WG-26935** 通过 IAkPluginContext 向插件暴露了 PostMonitorMessage()。
- **WG-26973** 通过 IAkPluginContextBase 向插件暴露输出设备 ID。
- **WG-27009** 在计量前添加新的混音器插件挂钩，让混音器插件将不同的数据提供给电平表和输出总线。
- **WG-27280** 暴露 PS4 的主要输出处理。

# 行为变化

- **WG-26215** 重构音乐过渡算法以在选择“跳转到项目”后重新评估规则，并在播放列表中执行前进操作。

# 性能变化

- **WG-22200** 优化的 Vorbis。
- **WG-27772**（iOS）改进了音频会话中断和应用程序状态变换（活跃、不活跃、后台）期间的核心音频收接端操作

# 其它更改

- **WG-23228** 添加了 SoundFrame 命令“Inspect Object”和“Find in Project Explorer”。
- **WG-23258** 将被弃用的 iOS 音频会话 C API 替换成新的 ObjC API。
- **WG-26052** 移除了过时的 AkCopyStreamedFiles.exe 工具。
- **WG-26274** 在 Project Explorer 中添加了用于折叠所有后代的快捷键。
- **WG-26526** 移除了集成演示中的无用预置。
