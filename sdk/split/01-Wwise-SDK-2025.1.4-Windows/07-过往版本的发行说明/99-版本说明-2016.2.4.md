版本说明 2016.2.4

以下各节列举并描述Wwise版本2016.2.3与版本2016.2.4之间的变化。

# 平台 SDK 的变化

- **Switch SDK** SDK 版本为 1.4.0.
- **PS4 SDK** SDK 版本为 4.508.41.

# 漏洞修复

- **WG-32985** 已解决：当触发速率（Trigger Rate）的过渡时长（transition duration）到达 0 时崩溃。
- **WG-33050** 已解决：在 Android 4.1 和更早版本上初始化崩溃。
- **WG-33149** 已解决：将包络作为 RTPC 应用于持续时间。
- **WG-33215** 已解决：有些对象引用在 Multi Editor 中不显示。
