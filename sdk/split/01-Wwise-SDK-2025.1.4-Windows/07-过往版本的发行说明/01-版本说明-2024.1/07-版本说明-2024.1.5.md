# 行为改进

- **WG-77644** 现在关闭 Object Tab Group 中的最后一个 Object Tab 不会再关闭整个 Object Tab Group。

# 漏洞修复

- **WG-76254** 已修复：(Spatial Audio) 浮点舍入错误导致在有些边界边缘和非边界边缘组合中找不到衍射路径。
- **WG-76989** 已修复：在 Music Segment Editor 中移动片段后“撤销”操作不起作用。
- **WG-77205** 已修复：(Spatial Audio) 浮点舍入有时会导致与透过 Portal 播放的 Room Tone 的距离不太稳定。
- **WG-77237** 已修复：在 Wwise Wave Viewer 中，HiDPI 监视器上区域选择的位置出现偏移。
- **WG-77241** 已修复：(Spatial Audio) 源自 Portal 的路径即使是条直线有时也会有少量的衍射。
- **WG-77307** 已修复：在 Primary Editor 的 RTPC 选项卡中，无法将 **MIDI Parameter** 设为 **Control Input**。
- **WG-77355** 已修复：在 User Preferences 对话框中使用“浏览”按钮时没有更新外部编辑器名称。
- **WG-77393** 已修复：仅包含媒体的 SoundBank 被视为没有用户。
- **WG-77536** 已修复：有可能通过拖放将 WAV 文件导入到已卸载的 Work Unit 中。
- **WG-77550** 已修复：崩溃报告 zip 文件中没有包含布局文件。

# 社区报告的漏洞修复

- **WG-70474** 已修复：在内存不足时，Reflect 触发断言。这样在动态链接的时候会发生崩溃。
- **WG-76827** 已修复：(Spatial Audio) 在对 Room 使用非均匀缩放时距离计算出现错误。
- **WG-76919** 已修复：在复制粘贴对象时可能会发生崩溃。
- **WG-77296** 已修复：在有两个文档库时，Perforce 在 Stream 模式下无法提交。
- **WG-77358** 已修复：在通过 Ctrl+单击来将属性重置为默认值时可能会发生崩溃。
- **WG-77705** 已修复：在尝试下载用于 Windows 平台的源码数据包时，Audiokinetic Launcher 出现错误。
- **WG-77738** 已修复：在将 Wwise 连接到在 **Profile and Edit** 模式下运行的游戏时，即便连接之前未做修改，且已对音乐进行了更新，音乐有时也会重新播放。
- **WG-77754** 已修复：在 Migration 对话框中，在文件不是只读时， **Allow Wwise to check out files** 复选框不起作用。

# 文档改进

- **WG-76251** 在 [快速入门示例集成——语言和语音](quickstart_sample_integration_languages.html) 中添加了有关在切换当前语言之前要做什么的详细说明。
