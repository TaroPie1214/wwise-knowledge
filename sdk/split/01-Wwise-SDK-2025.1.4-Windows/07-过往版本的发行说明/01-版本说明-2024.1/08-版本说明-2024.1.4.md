# 行为改进

- **WG-77224** (Spatial Audio) 现在使用几何构造测量 Room Tone 透射路径距离。

# 其他改进

- **WG-74672** 移除了设置 Event 目标时显示的空白上下文菜单。
- **WG-76921** 在只读状态下，多行编辑控件现在使用主题滚动条和对应主题颜色。

# 漏洞修复

- **WG-69687** 已修复：(Spatial Audio) 为 Reverb Zone 计算得出的方向和散布系数不太稳定，有时会跳变，尤其是在靠近透明表面相交的角落时。
- **WG-69998** 已修复：(Spatial Audio) 有时会将百分之一以上的少量衍射错误地应用于 Reverb Zone。
- **WG-74281** 已修复：现已无法再从 Switch Container Property Editor 创建新的 Switch Group 和 State Group。
- **WG-75900** 已修复：无法正确计算薄弱墙壁上的反射。
- **WG-76435** 已修复：在粘贴后，Project Explorer 错误地在对象名称末尾显示省略号 (...)。
- **WG-76437** 已修复：(Spatial Audio) 在使用 Distance Probe 的情况下穿过 Portal 时明显感觉透射损失不连续。
- **WG-76649** 已修复：在反复对列表进行排序时可能会发生崩溃。
- **WG-76719** 已修复：在删除对象后在 Object Property Settings 对话框中单击 **Reset to Default** 可能会导致 List View 发生崩溃。
- **WG-76862** 已修复：Attenuation Editor 中的第一个选项卡没有标签。
- **WG-76878** 已修复：在 Property Editor 很窄时，类别按钮的大小不对。
- **WG-76991** 已修复：在加载已卸载的 Work Unit 下嵌套的 Work Unit 时触发断言。
- **WG-77056** 已修复：在连接到 `.prof` 文件时会持续更新 Wwise 用户界面，而不是在操作结束的时候才更新一次。
- **WG-77129** 已修复：(WAAPI) 在使用 WAAPI 创建 Contents Editor 视图时发生崩溃。

# 社区报告的漏洞修复

- **WG-76723** 已修复：在 Crankcase Audio REV 2 插件内容被归入虚声部的情况下计算虚声源时出现错误。
- **WG-76728** 已修复：在将效果器从某一对象复制粘贴另一对象而两个对象均应用 Bypass State 和 Bypass Effect 时发生崩溃。
- **WG-76747** 已修复：作用于 Volume 属性和另一属性的调制器被应用了两次。
- **WG-76930** 已修复：(Spatial Audio) 在修改 Geometry Instance 的变换后马上将其移除可能会引发崩溃。
- **WG-76979** 已修复：在 Wwise 与平台连接或准备播放时同步对象可能会引发崩溃。
- **WG-76980** 已修复：在连接到游戏而其与包含 MIDI 控制器绑定的工程关联时可能会发生崩溃。
- **WG-76997** 已修复：在尝试播放媒体文件头异常的流播放音频源时没有生成错误并表明无法在开始播放时创建音频源。
- **WG-77001** 已修复：在加载 Work Unit、应用滤波器或运行 WAAPI 脚本后刷新 Project Explorer 或 List View 可能会引发崩溃。
- **WG-77017** 已修复：在 Project Explorer 中创建 Event 时可能会发生崩溃。
- **WG-77029** 已修复：在 I/O 内存不足时，流播放文件的音频丢失。
- **WG-77064** 已修复：在对 API 调用 `AK::SoundEngine::SetMultipleObstructionAndOcclusion` 实施性能分析时发生崩溃。
- **WG-77066** 已修复：在删除正在播放的 Music Sequence Container 时发生崩溃。
- **WG-77142** 已修复：对于应用有 Bypass State 属性的总线，在 States 选项卡中更改 **Change occurs at** 下的选项时发生崩溃。
- **WG-77210** 已修复：在重新加载多个经过修改的 Work Unit 时可能会发生崩溃。
- **WG-77214** 已修复：在对执行大量 `SetSwitch` 调用的游戏实施性能分析时出现性能问题。

# 文档改进

- **WG-75601** (Wwise Console) 明确了 [generate-soundbank](ak_wwise_cli_generatesoundbank.html) 引用页对 `–save` 选项的描述。
- **WG-75994** (WAAPI) 更新了 [订阅 Wwise Authoring API 中的话题](waapi_subscribe.html) 页面，添加了订阅代码示例的链接。
