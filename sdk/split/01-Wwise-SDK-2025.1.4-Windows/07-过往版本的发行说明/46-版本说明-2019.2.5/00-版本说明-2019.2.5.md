# 行为改进

- **WG-50203** [AkOpusDecoderFactory.h](_ak_opus_decoder_factory_8h.html) 现在并入到了 AllPluginFactories.h 中。若游戏中使用了 Opus，则需从代码中移除对应的 `#include` 行。否则，需将 AkOpusDecoder.lib 添加到链接选项。

# 性能改进

- **WG-47171** 针对提供且启用 AVX2 的平台提升了混音操作的效率，尤其是对于 Ambisonics 声道配置，效果更为明显。

# 漏洞修复

- **WG-49233** 已修复：未调用 `AK::SoundEngine::SetDefaultListeners()` 即调用 `AK::SoundEngine::AddListener()`，导致返回的指针为 NULL，进而引发崩溃。
- **WG-49745** 已修复：Project Explorer 内 Events 选项卡中的 Event ID 字段显示有误。
- **WG-49903** 已修复：在同一 PC 上运行两个 Wwise 设计工具时无法正常地从中选择所要连接的设计工具。
- **WG-49919** 已修复：在缺少原始文件时，Wwise 没有创建新的 .mediaid 文件。
- **WG-49901** 已修复：在 Actor-Mixer Hierarchy 中带有若干效果器的条目下执行复制/粘贴操作后出现性能下降。
- **WG-49292** 已修复：在启用 **Use game-defined aux sends** 时，无法将 Spatial Audio 早期反射总线发送到所指派的 Room 游戏对象和 Room 辅助总线。
- **WG-49636** 已修复：有时无法正确刷新 Mixer 插件名称。
- **WG-49572** 已修复：在出现文件权限错误时引发异常。
- **WG-49145** 已修复：在 Blend Container 中设置音轨时发生罕见崩溃。
- **WG-49755** 已修复：在转换源文件采样率时设置了错误的标记点位置。
- **WG-49996** 已修复：在复制/粘贴多个对象后没有评估非 ShareSet 插件的 Media ID。
- **WG-49965** 已修复：Convolution Reverb 插件触发良性断言。
- **WG-49820** 已修复：在执行 SoundBank 生成命令后没有自动关闭对话框。
- **WG-49904** 已修复：针对 RTPC 插值计算的断言检查有点太严格了。
- **WG-49970** 已修复：在播放包含奇数个声道的 PCM 或 ADPCM 音频时可能会越界读取数组。
- **WG-48578** 已修复：`ReplaceOutput()`、`RemoveOutput()` 和 `GetSpeakerConfiguration()` 无法使用 `AddOutput()` 所提供的 AkDeviceOutputID。
- **WG-50041** 已修复：在 Project Explorer 中所选的对象包含对 State Group 的无效引用时发生崩溃。
- **WG-50057** 已修复：Mute 和 Solo 按钮对通过 `PostMIDIOnEvent` 播放的声音不起作用。

# 社区报告的漏洞修复

- **WG-50098** 已改进：在将父对象添加到 Actor-Mixer Hierarchy 中的元素时发生崩溃。
