# 新增功能

- **WG-48759** 新增了两个全局回调： `AkGlobalCallbackLocation_Suspend` 和 `AkGlobalCallbackLocation_WakeupFromSuspend`。
- **WG-50146** 向 iOS PlatformInitSettings 添加了新的音频会话类别 AkAudioSessionCategoryPlayback。
- **WG-50604** 在 Property Editor 和多个其他编辑器的 RTPC 选项卡中添加了性能分析筛选器栏。

# 行为改进

- **WG-50461** SetGameObjectAuxSendValues、SetGameObjectOutputBusVolume 和 SetMultipleObstructionAndOcclusion 现在会将输入值限制在 0 和 1 之间。
- 不过对于 2019.2.6，无法将 Wwise 设计工具连接到之前的小版本声音引擎。

# 性能改进

- **WG-47937** 大幅提升了处理无声输入音频（包括混响尾音）时 Wwise Convolution 的性能。同时添加了新的 Wwise Convolution 共享集设置 Input Thresh，以便忽略低于指定音量的高频输入音频，从而降低卷积处理的资源消耗。
- **WG-49532** 在使用 Game Profiler 捕获数据时，提升了性能并降低了内存用量。
- **WG-50517** 在 Game Sync Monitor 中启用 Voice Inspector Data 的情况下执行性能分析时，提升了性能并降低了内存用量。

# 漏洞修复

- **WG-46457** 已修复：Game Object 3D Viewer 中的 Camera 列表包含重复条目。
- **WG-47032** 已修复：同时运行多个 WwiseConsole 实例可能会导致文件转码失败。
- **WG-47776** 已修复：对于 BusVolume 上设有 State 的总线，可能会将音量变化错误地应用于子总线。在这种情况下，将把音量变化应用于子总线的 BusVolume 而不是 OutputBusVolume。
- **WG-48729** 已修复：在移动性能分析器光标时，Voice Inspector 有时会发生崩溃。
- **WG-49826** 已修复：在复制 Music Switch Container 内的过渡时没有复制其 Source 和 Desitination。
- **WG-49993** 已修复：在过渡期间销毁游戏对象时，Voice Inspector 无法正常报告过渡。
- **WG-50097** 已修复：无法正确显示 Wwise Pitch Shifter 插件输入。
- **WG-50164** 已修复：在听者移动后针对 Portal 后面的“发声体”游戏对象播放单个短促声音时，Spatial Audio Room 混响尾音和早期反射尾音被截断。
- **WG-50229** 已修复：在将 Virtual voice behavior 设为 Kill if finite else virtual 时，Virtual Voice 选项中的 On return to physical voice 复选框错误地显示为灰色。
- **WG-50260** 已修复：在 Spatial Audio 衍射角度接近 180 度时结果为 0% 衍射。
- **WG-50378** 已修复：(Spatial Audio) 在分配内部数据结构时内存不足，进而引发崩溃。
- **WG-50399** 已修复：在使用 Spatial Audio 功能而未将设计工具连接到游戏的情况下运行 Profile 或 Debug 版本的声音引擎会发生缓慢泄漏。
- **WG-50415** 已修复：Game Sync Monitor 中的筛选功能无法按预期工作。
- **WG-50506** 已修复：在 MS Spatial 驱动程序故障时出现卡顿。
- **WG-50508** 已修复：在 Effects 列表中使用导航器按钮时，Work Unit 的 XML 文件中的 PluginName 字段丢失。
- **WG-50510** 已修复：Game Aux Send Value 和 User Aux Send Value 所应用的 RTPC 不在有效范围内，导致音频采样超出值域。
- **WG-50511** 已修复：在尝试将数据写入到用户没有相应权限的文件夹时发生崩溃。
- **WG-50514** 已修复：在从 Advanced Profiler 的 Voices 选项卡拖动对象时发生崩溃。
- **WG-50515** 已修复：在启用 Max Radius 数据选项的情况下生成 SoundBank 时缺少 Event 可能会引发崩溃。
- **WG-50526** 已修复：在使用 WwiseConsole.exe 时发生 Win32 ATOM 泄漏。
- **WG-50536** 已修复：无法正确显示 Wwise Pitch Shifter 插件输入和和声效果器。
- **WG-50569** 已修复：在文件的声道数大于 10 时，Vorbis 解码器发生崩溃。
- **WG-50638** 已修复：在播放包含空白关联列表的 Music Switch Container 时出现访问冲突。
- **WG-50672** 已修复：SoundBanksInfo.xsd Schema 中的 ActionSetState 和 ActionSetSwitch 颠倒。
- **WG-50691** 已修复：Motion Source 插件中的 Actuator Configuration 没有正确反映平台链接设置。
- **WG-50732** 已修复：在将 SoundBank 生成到用户没有写入权限的目录时出现卡顿。
- **WG-50758** 已修复：在捕获期间，Actor-Mixer Hierarchy 对象的 RTPC 选项卡显示不相关的游戏对象值。
- **WG-50839** 已修复：Izotope 插件不支持全部声道配置（Reverb 和 Box Modeler 没有改进）。
