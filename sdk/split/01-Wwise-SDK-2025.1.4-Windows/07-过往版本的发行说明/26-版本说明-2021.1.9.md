# 新增功能

- **WG-59959** (Command Add-on) 现在支持检索命令参数中的属性值。请参阅 [参数变量](defining_custom_commands.html#defining_custom_commands_variables) 了解更多信息。

# 行为改进

- **WG-60301** 针对 `AK::SoundEngine::SetSpeakerAngles()` 函数添加了关于扬声器之间的角度不小于 5 度的限制。

# 其他改进

- **WG-55188** (wp.py) 脚本默认为 Python 3。
- **WG-59208** 将 CivetWeb 更新到了 1.16 版本（由 WAAPI 用来进行通信）。
- **WG-60203** (wp.py) 为 Switch 和 PS5 添加了对 Visual Studio 2019 解决方案的支持。

# 漏洞修复

- **WG-57858** 已修复：在 WAAPI 仍处于锁定状态时过早地发布 WAAPI 主题 [ak.wwise.core.project.loaded](ak_wwise_core_project_loaded.html) 。
- **WG-59346** 已修复：无法在 Game Object 3D Viewer 中选择 "Listener-relative" 摄像机类型。
- **WG-59355** 已修复：在工程包含渲染的 iZotope Trash Box Modeler 插件时生成 SoundBank 可能会发生崩溃。
- **WG-59844** 已修复：在内存不足的情况下设置带有过渡的 RTPC 时，声音引擎发生崩溃。
- **WG-59946** 已修复：在 Query Editor 中搜索效果器和元数据属性时，UI 控件发生重叠。
- **WG-60072** 已修复：(Mac Authoring) Game Object 3D Viewer 中没有文本。
- **WG-60083** 已修复：在取消停靠前端的 REV 2 插件时出现卡顿。

# 社区报告的漏洞修复

- **WG-56848** 已修复：在注册刚刚注销的 Game Object 时可能会发生崩溃。
- **WG-57754** 已修复：在 Music Switch Container Association Editor 中，在单击其他地方或移动横向分隔器时，底部列表中的选定行有时会自行改变。
- **WG-58417** 已修复：在使用 Custom Property 的 Event 上添加引用后无法打开工程。
- **WG-59557** 已修复：Address Sanitizer 报告了与函数 `AK::DeviceEnumeratorHelper()` 相关的错误。
- **WG-59585** 已修复：在实施声像摆位的过程中，`CAkSpeakerPan::CreateSpreadCache()` 可能会引发崩溃。
- **WG-60039** 已修复：Query Editor 中的 WAQL 字段不接受长度大于编辑控件宽度的文本。
- **WG-60121** 已修复：在执行 SoundBank 生成后操作时关闭 Wwise 会发生崩溃。
- **WG-60262** 已修复：在对流播放 Opus 文件寻址时可能会发生崩溃。
