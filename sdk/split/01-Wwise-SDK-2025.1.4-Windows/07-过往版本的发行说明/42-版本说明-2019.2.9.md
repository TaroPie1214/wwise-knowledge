# 新增功能

- **WG-49719** 在 Query Editor 中添加了新的搜索条件。现在可搜索发送到特定 Early Reflections Auxiliary Bus 的对象。
- **WG-52493** 为刷新 Workgroup 图标，创建了新的键盘快捷方式 Shift+F5。

# API 改进

- **WG-47828** 向 AkMemID 添加了新的 AkMemType\_Media 内存类型位，以指示对 Memory Manager 的媒体分配。这些分配通过 rpmalloc 分配器中单独的内存堆执行。
- **WG-52776** 向 [AkMemSettings](struct_ak_mem_settings.html) 页面分配挂钩添加了 uVMPageSize 和 uDevicePageSize，以在改写页面分配器时指定自定义对齐。

# 其他改进

- **WG-48505** Spatial Audio：添加了新的错误消息来指示几何早期反射所用 Auxiliary Bus 和 SetImageSource API 所用 Auxiliary Bus 之间存在冲突。

# 漏洞修复

- **WG-30459** 已修复：(Mac Authoring) Option 键没有映射到 Alt 键。
- **WG-46388** 已修复：在没有使用 Commcentral.lib 的情况下实施性能分析时，非必要依赖项强行与之建立链接。
- **WG-47359** 已修复：无法使用 Query Editor 条件搜索总线的衰减设置。
- **WG-49139** 已修复：在将 AK\_INVALID\_GAME\_OBJECT\_ID（指示全局作用域）传给 [AkAuxSendValue::listenerID](struct_ak_aux_send_value_a185a87e879e5a602446b07252bdd8519.html#a185a87e879e5a602446b07252bdd8519 "Game object ID of the listener associated with this send. Use AK_INVALID_GAME_OBJECT as a wildcard to...") 字段的情况下，Voice Inspector 中没有显示源自 SetGameObjectAuxSendValues 的音量影响。
- **WG-50072** 已修复：(Spatial Audio) 对于大型 Room 和 Portal 网络，在听者变换 Room 时 CPU 用量突然增加。
- **WG-52013** 已修复：在将某些 DirectInput 设备连接到电脑时，设计工具出现短暂的 UI 卡顿。
- **WG-52033** 已修复：在 Capture Log 视图中，性能随着时间不断下降。
- **WG-52070** 已修复：在 GUI 之外实施更改后重新打开插件时，无法正常刷新 Mastering Suite 坐标图。
- **WG-52243** 已修复：在使用 Opus 编解码器处理特定内容时可能会发生崩溃。
- **WG-52286** 已修复：无法通过总线稳定地实时编辑声像器的 X-Y 坐标。
- **WG-52342** 已修复：在消息队列中仍有 SetSpeakerAngles 或 SetListenerSpatialization 消息时终止声音引擎会引发内存泄漏。
- **WG-52513** 已修复：在请求中止已经中止过一次的延迟操作时发生崩溃。
- **WG-52534** 已修复：在加载特定 .prof 文件时发生崩溃。
- **WG-52563** 已修复：在使用 Opus 时效果器带有尾音，导致发生声音泄漏。
- **WG-52588** 已修复：本地化 SoundBank 元数据文件中的哈希值有误。
- **WG-52592** 已修复：Advanced Profiler 的 Loaded Media 选项卡无法解析文件名称。
- **WG-52606** 已修复：在应用了 iZotope Hybrid Reverb 效果器的情况下生成 SoundBank 时存在无效的 WAV 文件，因而引发崩溃。
- **WG-52643** 已修复：`AK::MemoryMgr::Init` 不允许使用自定义分配器将 `AkMemSettings:: pfSizeOfMemory` 改写为 nullptr。
- **WG-52654** 已修复：在系统请求执行大小为 0 的分配时，Memory Manager 报告内存不足错误。
- **WG-52657** 已修复：自定义关联混音器插件 GUI 时没有任何反应。
- **WG-52664** 已修复：在 Effects 列表中更改 Preset(Name) 或 Mode 时，Work Unit 的 XML 文件中的 PluginName 字段丢失。
- **WG-52689** 已修复：Authoring Audio Preferences 对话框中没有列出 Motion Bus 所对应的相连控制器。
- **WG-52761** 已修复：预留内存总值不准确。
- **WG-52804** 已修复：在 Clutch RPM Spike 为 0 时换到一档会导致 Crankcase REV 源插件挂起。
- **WG-52805** 已修复：在使用 Voice Inspector 对游戏实施性能分析时，播放的声音具有 255 个以上听者-发声体对，导致内存损坏，进而引发崩溃。
- **WG-52817** 已修复：在调用 Sound Engine [Term()](namespace_a_k_1_1_comm_a94e307d2974b89cc4fbc8ab0fef20d2f.html#a94e307d2974b89cc4fbc8ab0fef20d2f) 时仍处于录音状态，导致 Recorder 插件写入了无效的 WAV 文件头。
- **WG-52853** 已修复：(Mastering Suite) 推子似乎只支持整数值。
