# 平台 SDK 改进

- **Stadia SDK**：已更新至 SDK 1.43。
- **PS4 SDK**：已更新至 SDK 7.008.031。
- **Xbox One XDK**：已更新至 XDK 10.0.17134.5071（2018 年 7 月 qfe12）。
- **Switch SDK**：已更新至 SDK 9.4.1。

# 新功能

- **WG-34982** (Switch) 现在可通过 HTCS 远程连接到 Switch。

# API 改进

- **WG-47804** 2019.1.8 插件构建工具 (wp.py) 现在与 2019.2 版本同步。

# 行为变化

- **WG-42768** 将 IncrementalSoundBankData.xml 和所有SoundBank .stat 文件合并到了缓存文件夹内的单个二进制文件 (SoundBankInfoCache.dat) 中。这样最多可将文件大小缩减 83，并大幅减少生成 SoundBank 时的 I/O 调用数量。

# 性能改进

- **WG-42993** 缩短了生成 SoundBank 时的迭代时间，所有生成任务都可自动完成。
- **WG-45144** 对 Convolution Reverb 和 Filter 效果器进行了大幅优化，在很多情况下吞吐量性能几乎翻倍，内存用量降低一半。注意，必须重新生成包含 Convolution Reverb 媒体的 SoundBank，才能使改进生效。
- **WG-47711** 提升了在 Project Explorer 中检视对象时的性能。

# 其他改进

- **WG-48322** 提升了在较大工程中切换平台时的性能。
