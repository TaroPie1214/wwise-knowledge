版本说明 2017.2.4

以下各节列举并描述 Wwise 版本 2017.2.3 和版本 2017.2.4 之间的变化。有关早期版本的信息，请参阅 [过往版本的发行说明](previousreleases.html) 章节。如果使用联机帮助，您还可以从目录列的顶部选择别的 Wwise 版本。此版本为漏洞修复版本，因此对所有 2017.2.x 版本来说，工程、音频包和 API 都是兼容的，不会有任何变动。

# 漏洞修复

- **WG-36559** 已修复：(iOS) 在终止声音引擎时可能会出现崩溃。
- **WG-36830** 已修复：在监控大量游戏对象时，Profiler 的速度会减慢。
- **WG-37156** 已修复：iOS 11 上发生崩溃。
- **WG-37165** 已修复：Spatial Audio：门户错误消息不显示门户名称，而只显示门户 ID。
- **WG-37219** 已修复：AkAudioInput.xml 中未列出 Switch 平台。
- **WG-37385** 已修复：在重新加载 init.bnk 时发生崩溃。
- **WG-37388** 已修复：在执行主题查询时，WAAPI ak.wwise.waapi.getSchema 挂起。
- **WG-37392** 已修复：SetPluginCustomGameData 出现崩溃。
- **WG-37560** 已修复：在 File Manager 内使用非罗马字符重命名文件时，Rename File/Folder 对话框中显示问号而非实际文件名。
