# 平台 SDK 改进

- **Xbox One XDK**：已更新至 XDK 10.0.17134.5076（2018 年 7 月 qfe13）。
- **Stadia SDK**：已更新至 SDK 1.45。

# 新功能

- **WG-48291** Voice Monitor：添加了用来更改声部颜色模式的选项。

# API 改进

- **WG-47790** `AK::SoundEngine::ReplaceOutput` 和 `AK::SoundEngine::GetOutputID` 中的主输出设备现在可由 0 指派。

# 行为变化

- **WG-48499** PulseAudio (Linux/Stadia)：现在支持使用 `AkPlatformInitSettings::sampleType` 在 `PA_SAMPLE_FLOAT32LE` 和 `PA_SAMPLE_S16LE` 之间进行选择。在 Stadia 上，默认使用 `PA_SAMPLE_S16LE`。不再使用 `PA_STREAM_INTERPOLATE_TIMING` 和 `PA_STREAM_ADJUST_LATENCY`。

# 漏洞修复

- **WG-46449** 已修复：(Android) 在运行老版固件的 Vivo Y51A 手机上，音频输出含混不清。
- **WG-46596** 已修复：若请求的时间点在性能分析会话时段之前或之后，则 [ak.wwise.core.profiler.getRTPCs](ak_wwise_core_profiler_getrtpcs.html) 会返回错误。
- **WG-47718** 已修复：在内存不足时，rpmalloc 发生崩溃。
- **WG-48188** 已修复：在 Windows 上使用 /W4 进行编译时，AkPlatformFuncs.h 和 [AkStreamMgrModule.h](_ak_stream_mgr_module_8h.html) 报告编译警告。
- **WG-48235** 已修复：`FilePackager.Console` 无法打包设为流播放的音频文件。
- **WG-48265** 已修复：在 Linux 操作系统上，在尝试针对没有 IP 地址的设备枚举网络设备时，Wwise 发生崩溃。
- **WG-48357** 已修复：在禁用了虚声源定位且没有通向听者的路径（比如所有 Portal 均处于关闭状态）的情况下，声音传播功能不起作用。
- **WG-48379** 已修复：在调用 `AddOutputCaptureMarker` 后马上终止声音引擎会引发轻微的内存泄漏。
- **WG-48392** 已修复：Advanced Profiler 视图内 Listeners 选项卡的 Volumes 列中显示了错误的值。
- **WG-48402** 已修复：rpmalloc 错误地追踪当前所用虚拟内存。
- **WG-48417** 已修复：在启用 [AkInitSettings::bDebugOutOfRangeCheckEnabled](struct_ak_init_settings_a0ffc62cdbb07d0c8ebb2440fd177625b.html#a0ffc62cdbb07d0c8ebb2440fd177625b "Debug setting: Enable checks for out-of-range (and NAN) floats in the processing code....") 系统时，没有正确启用有些代码路径，并可能会错误地将管线中的下一条目指定为错误样本插入点。
- **WG-48440** 已修复：在特定版本的 Windows 上，MS Spatial Audio 插件发生崩溃。
- **WG-48442** 已修复：在通过命令行生成 External Source 时显示了错误的警告。
- **WG-48502** 已修复：在针对多声道 Opus 音频源执行 DecodeBank 操作时可能会发生内存损坏。
- **WG-48503** 已修复：`AkArray::BinarySearch` 中越界读取内存。
- **WG-48599** 已修复：在启用 AVX 支持的情况下使用 Visual Studio 2015 库会导致 Convolution 插件发生崩溃。
- **WG-48683** 已修复：`AK_ALIGN` 无法对齐内存。
- **WG-48743** 已修复：有些包含经过编码的循环点的流播放 PCM 或 ATRAC9 声音存在源匮乏问题。

# 社区报告的漏洞修复

- **WG-48348** 已修复：在执行性能分析的过程中使用文本筛选器时，Game Sync Monitor 挂起。
- **WG-48359** 已修复：Game Sync Monitor 中有时不显示工具提示，并且使用这些提示会导致性能下降。
- **WG-48493** 已修复：General Log 中的警告消息显示缺少 Skin Manager 'ErrorText'。
