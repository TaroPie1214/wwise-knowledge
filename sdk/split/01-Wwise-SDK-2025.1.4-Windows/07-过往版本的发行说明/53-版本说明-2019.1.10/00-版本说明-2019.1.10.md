# 新增功能

- **WG-48759** 新增了两个全局回调：`AkGlobalCallbackLocation_Suspend` 和 `AkGlobalCallbackLocation_WakeupFromSuspend` 。

# 漏洞修复

- **WG-47032** 已修复：同时运行多个 WwiseConsole 实例可能会导致文件转码失败。
- **WG-49904** 已修复：针对 RTPC 插值计算的断言检查有点太严格了。
- **WG-50522** 已修复：在尝试报告 ErrorCode\_MediaDuplicationLength 类型的错误消息时，声音引擎发生崩溃。
- **WG-50057** 已修复：Mute 和 Solo 按钮对通过 `PostMIDIOnEvent` 播放的声音不起作用。
- **WG-49320** 已修复：在同时共享同一 User-Defined Path 的声音数超过 255 时可能会发生崩溃。

# 社区报告的漏洞修复

- **WG-49570** 已修复：设计工具和声音引擎之间发生罕见的失去同步问题，进而引发崩溃。
- **WG-49572** 已修复：在出现文件权限错误时引发异常。
- **WG-49145** 已修复：在 Blend Container 中设置音轨时发生罕见崩溃。
