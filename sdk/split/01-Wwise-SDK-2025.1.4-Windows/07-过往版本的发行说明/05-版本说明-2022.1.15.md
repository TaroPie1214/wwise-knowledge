# 新增功能

- **WG-68160** File Manager 现在支持 MIDI 文件。

# 行为改进

- **WG-73325** CPU Timeline 和 CPU Usage 坐标图现在包含声音引擎执行期间等待获取全局锁花费的时间。这样有助于确认音频渲染当中的闲置情况。

# 漏洞修复

- **WG-71929** 在使用 High DPI 监视器时，单击 Project Explorer 中的对象有时不会在 Property Editor 中相应地予以显示。
- **WG-73161** 已修复：在尝试使用 Music Segment 快捷菜单时可能会发生崩溃。
- **WG-73196** 已修复：Stinger 的 Play At 属性值 Next Beat 和 Next Bar 颠倒了。
- **WG-73329** 已修复：Music Playlist Item 不允许将 Avoid Repeat 设为零。

# 社区报告的漏洞修复

- **WG-64858** 已修复：在将 SetOfflineRendering 设为 false 来恢复正常播放时，音频没有恢复。
- **WG-72711** 已修复：在将 SetOfflineRendering 设为 false 来恢复正常播放时，音频没有恢复。
- **WG-53536** 已修复：在使用 Convolution Reverb 的情况下加载或卸载音频素材时发生崩溃。
- **WG-73272** 已修复：(Spatial Audio) ErrorCode\_SpatialAudio\_PortalNotFound 没有正确报告 Portal ID。
- **WG-73328** 已修复：在取消预备互动音乐内容时，捕获日志中误报 "Failed to unprepare event" 错误。
