# 新增功能

- **WG-49699** Mastering Suite：现在会显示被禁用的控制点，并可通过双击恢复为启用状态。
- **WG-51642** 在 Attenuation Editor 的 Attenuation 选项卡中添加了性能分析筛选器栏。
- **WG-51843** 现在可通过 WAAPI 和 Tab Delimited Import 访问 Color 和 OverrideColor。

# 性能改进

- **WG-51703** 提升了处理 Voice Inspector 数据时的分析性能。

# 其他改进

- **WG-50715** 为提高对比度和清晰度，针对曲线和点对三种主题下的 Mastering Suite 颜色实施了改进。

# 漏洞修复

- **WG-50209** 已修复：在 Sink 插件初始化失败时发生崩溃。
- **WG-50819** 已修复：在磁盘已满的情况下生成 SoundBank 可能会发生崩溃。
- **WG-51290** 已修复：Capture Log 不显示来自插件的错误消息。
- **WG-51335** 已修复：在 Wwise 工程文件夹中删除和插入文件可能会导致设计工具锁定文件资源或发生崩溃。
- **WG-51409** 已修复：`AK::GetWindowsDevice` 针对默认设备返回 NULL。
- **WG-51425** 已修复：在基准值为 1 时，Random/Sequence Container 中的 Loop Count 随机化不起作用。
- **WG-51457** 已修复：在树视图控件中单击下拉列表之后无法在树视图中滚动鼠标滚轮。
- **WG-51474** Mastering Suite：现在可正确标记涉及坐标图中所作更改的撤消操作。
- **WG-51511** Mastering Suite：为插件 GUI 之外所作的更改设置了合理的数值限制。
- **WG-51513** 已修复：(Mac Authoring) 文本在文本编辑控件中被粘贴了两次。
- **WG-51532** 已修复：(Mac Authoring) Cmd 没有映射到 Ctrl 键。
- **WG-51579** 已修复：对于采用 5.1 IR 或 7.1 IR 的 Convolution Reverb，有时会将部分混响内容输出到错误的环绕声声道。
- **WG-51596** 已修复：在有些情况下，来自单个听者的音频被发送到给定输出设备。
- **WG-51605** 已修复：无法在 Game Object 3D Viewer 中禁用 Game Object Filter。
- **WG-51623** 已修复：在所加载 SoundBank 包含 Wwise 2019.2.6 之前版本生成的 Convolution Reverb 数据时触发断言或发生其他处理错误。
- **WG-51761** 已修复：在更改 BypassFX Event Action 的 Target 时发生崩溃。
- **WG-51810** 已修复：在主机转换外部源的过程中初始化了太多模块。
- **WG-51889** 已修复：在两个 Opus 文件短于音频帧长度时执行精确到采样点的过渡会导致死机。
- **WG-51913** 已修复：对于立体声输出配置，在将 Mastering Suite 的 Limiter 设为 Hard 模式时会同时在左右声道中输出左声道内容。
- **WG-51982** 已修复：(WAAPI) C# 客户端最多只能连续连接五次，继续连接的话会导致客户端卡顿。
- **WG-52005** 已修复：在单击工具栏后按下 F10 无法激活 Interactive Music 布局。
- **WG-52035** 已修复：在将标记点设为接近精确到采样点的过渡时发生崩溃。
- **WG-52068** 已修复：在针对总线和多个听者的内存分配失败时可能会发生崩溃。
