重要的迁移说明2018.1.6

对于此补丁版本，在迁移至 Wwise 2018.1.6 时需要特别注意。

# 在迁移到 2018.1.6 时，SoundSeed Grain 的 Source 选项卡中的 Marker 可能会移动位置。

假如在保存 SoundSeed Grain 标记时将 Marker Input Mode 设为 Detect From Transients，在 2018.1.6 中打开时标记可能会有所不同。如果希望保留目前检测到的标记，请务必在迁移前将 Marker Input Mode 设为 Manual Markers。
