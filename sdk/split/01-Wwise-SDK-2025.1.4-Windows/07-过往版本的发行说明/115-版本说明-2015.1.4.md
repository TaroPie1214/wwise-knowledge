版本说明 2015.1.4

以下各节列举并描述Wwise版本2015.1.3与版本2015.1.4之间的变化。

# 漏洞修复

- **WG-28491** [已解决：当定位定义于父代上时，AK::SoundEngine::Query::GetPositioningInfo()](namespace_a_k_1_1_sound_engine_1_1_query_a58bb2a69318ddee9d168c0e57fd2dac4.html#a58bb2a69318ddee9d168c0e57fd2dac4) 返回错误的声像摆位类型。
- **WG-28503** 已解决：当 bUseSoundBankMgrThread 为 false 时，SoundBank API 调用没有返回错误代码。
- **WG-28563** 已解决：当两根曲线都通过引用同一游戏参数的同一节点上时 RTPC 发生崩溃。
- **WG-28574** 已解决：切换轨过渡期间的切换延迟。
- **WG-28592** 已解决：（iOS）在SoloAmbient中播放用户音乐的同时启动AAC声音，发生崩溃。
- **WG-28626** 已解决：收到State或RTPC通知时发生罕见的崩溃。
- **WG-28633** 已解决：（Windows）当在 Windows 音频面板中切换音频设备时，罕见的竞争危害可导致崩溃。
- **WG-28658** 已解决：（Xbox One）设置 uMaxXMAVoices 为 0 来禁用 XMA 会造成在启动时崩溃。
- **WG-28706** 已解决：当声部进行流播放，并且采用“从已流逝时间”设置的虚声部时，Vorbis拼接中会发生崩溃。
