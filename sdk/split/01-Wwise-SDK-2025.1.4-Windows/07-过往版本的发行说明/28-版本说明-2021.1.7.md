# 行为改进

- **WG-56439** Spatial Audio：现在可在更新路径时平滑地更新发声体位置和虚声源位置。这样可在使用较大距离阈值时减少可闻的颤动杂音。

# 性能改进

- **WG-58649** 为避免在不必要的情况下销毁底层内存对 [AkArray::Copy](class_ak_array_a8a436cce42c404496e6ebb19072581e6.html#a8a436cce42c404496e6ebb19072581e6) 进行了调整，现在只会稍微减少声音引擎中的内存分配数和释放数。

# 漏洞修复

- **WG-55462** 已修复：在 Mac 上将 Keyboard Shortcuts and Commands 中创建的命令文件显示成了灰色（不可用）。
- **WG-56281** 已修复：在有 Room 和 Portal 时加载/卸载几何构造可能会发生崩溃。
- **WG-58028** 已修复：在删除 Sound Voice 时，Contents Editor 发生崩溃。
- **WG-58713** 已修复：(Mac Authoring) 无法在自定义命令中正常运行当前工作目录。

# 社区报告的漏洞修复

- **WG-55235** 已修复：在 Dialogue Event Editor 中选择大量路径时发生崩溃。
- **WG-57782** 已修复：(WAAPI) 在 writeToDisk 参数为 false 的情况下调用 [ak.wwise.core.soundbank.generate](ak_wwise_core_soundbank_generate.html) 时没有执行 SoundBank 生成前和生成后操作。
- **WG-58048** 已修复：在 WwiseUnityAddressables 数据包中使用了过时的 API。
- **WG-58172** 已修复：在使用 3D User-Defined Positioning 路径时发生崩溃。
- **WG-58185** 已修复：在使用重定向输出时，自定义命令会在程序前面添加 Change Working Directory (CWD)。
- **WG-58199** 已修复：有些随 Wwise 一起发布的动态库被报告为潜在恶意软件。现在会在适用情况下使用 Audiokinetic 的证书对其进行签名。
- **WG-58233** 已修复：在设置较低的声障值时，SetPortalObstructionAndOcclusion 有时不起作用。
- **WG-58435** 已修复：Wwise Channel Router 混音器插件没有在 SoundBank 中正确填充插件内容。
- **WG-58554** 已修复：在连接状态下不必要地禁用了 Game Aux、User Aux 和 Effects 的 Override parent 属性。
- **WG-58637** 已修复：过早地调用 `Ak::SoundEngine::GetSpeakerConfiguration` 会返回 0。
