重要的迁移说明 2014.1

**WG-25210** 总线和声部“音量回调”（AkSpeakerVolumeMatrixCallbackInfo）

总线和声部“音量矩阵”回调现在共用同一结构。The ID of both the destination bus and input (voice or bus) is accessible through the use of the [AK::IAkMixerInputContext](class_a_k_1_1_i_ak_mixer_input_context.html "Interface to retrieve information about an input of a mix connection (for processing during the Speak...") and [AK::IAkMixerPluginContext](class_a_k_1_1_i_ak_mixer_plugin_context.html "Interface to retrieve contextual information for a mixer.") interfaces, although the ID is internal in the case of voices. In the case of a voice, the callback is called as many times as there are connection to a mix bus, for example, once for the direct path and once for each send to auxiliary busses.

多声道音量向量和矩阵不再通过带声道名称的结构来表示，而表示为浮点指针，其类型定义为 [AK::SpeakerVolumes::VectorPtr](namespace_a_k_1_1_speaker_volumes_ad437e8acf5a9509d22a2d13629502afa.html#ad437e8acf5a9509d22a2d13629502afa) 或 AK::SpeakerVolumes::MatrixPtr。这些向量/矩阵的大小取决于声道配置。对于音量矩阵，输入声道应使用 [AK::SpeakerVolumes::Matrix::GetChannel()](namespace_a_k_1_1_speaker_volumes_1_1_matrix_a77aabf10ae35e9ccd346662e29fba169.html#a77aabf10ae35e9ccd346662e29fba169) 访问。向量的元素应使用标准大括号 [] 运算符访问。请参阅 [AkSpeakerVolumes.h](_ak_speaker_volumes_8h.html) 了解更多详情。声道顺序遵循 [AkSpeakerConfig.h](_ak_speaker_config_8h.html) 中定义的声道位顺序，LFE 除外，它始终位于末尾。此变化会影响 [AkSpeakerVolumeMatrixCallbackInfo](struct_ak_speaker_volume_matrix_callback_info.html) 回调和 [AK::SoundEngine::SetListenerSpatialization()](namespace_a_k_1_1_sound_engine_a3e3e9e5335df01fca37f9b3ed518e2bd.html#a3e3e9e5335df01fca37f9b3ed518e2bd)。

请参阅 [SDK 带来增强的声部/总线音量回调（AkSpeakerVolumeMatrixCallbackInfo）和节拍访问](whatsnew_2014_1_new_features.html#whatsnew_2014_1_new_feature_SpeakerVolumeCallback) 、 [使用 Speaker Matrix Callback 定制高级混音](goingfurther_speakermatrixcallback.html) 、AkSpeakerVolumeMatrixCallbackInfo 和 [AK::SpeakerVolumes](namespace_a_k_1_1_speaker_volumes.html "Multi-channel volume definitions and services.") 了解更多详情。

**WG-25442** GenAudio RTI 插件正迁移到新的混音器插件系统。

在 Wwise 2014.1 中，我们将同时支持过去遗留的 GenAudio RTI Effect 插件和新版 GenAudio RTI Mixer 集成插件。我们强烈建议切换到新的混音器系统，因为它可以提供更高灵活度的解决方案。

注：如果您决定继续使用 GenAudio RTI Effect 插件，遗留的效果器插件中存在行为变化：

- “方位角”控制角度现为 [-180, 180]，以前为 [0, 360]。如果您以前已使 RTPC 曲线图与此值相匹配，则将需要重新制作曲线图来匹配新的控制值。
- 另请注意，从 Wwise 2014.1 开始，有一个新功能可以让实际的方位角/仰角/距离来自动驱动 RTPC，而无需游戏来驱动 RTPC 值，请参阅有关内置游戏参数的信息。

**WG-25190** 改善了用户 SoundBank 设置持续系统。

自 2014.1 起，2014.1 前的用户 SoundBank 设置在迁移时将丢失。

- 在更改之前，无法迁移用户 SoundBank 设置。在更改之后，用户 SoundBank 设置现在包含在 Wwise 工程文件夹，并可与其它工程文件一起正确迁移。
- 工程 SoundBank 设置仍可正确迁移。

从 2014.1 开始，用户 SoundBank 设置将正确保留在 Wwise 工程用户的 \*.wsettings 文件中。

请参阅《安装与迁移指南》了解有关将工程迁移到 Wwise 新版本的一般建议。
