版本说明 2018.1.7

以下各节列举并描述 Wwise 版本 2018.1.6 和版本 2018.1.7 之间的变化。有关早期版本的信息，请参阅 [过往版本的发行说明](previousreleases.html) 章节。此版本为漏洞修复版本，因此对所有 2018.1.x 版本来说，工程、音频包和 API 都是兼容的，不会有任何变动。

|  |  |
| --- | --- |
|  | **备注:** **插件开发人员：** 此 Wwise 版本支持通过 Wwise 2018.1 构建的插件。对于通过旧版 Wwise 构建的插件，开发人员必须使用兼容版本重新编译并重新打包。 |

# 平台 SDK 的变化

- **Switch SDK**：已更新到 SDK 7.3.0。
- **PS4 SDK**：已更新到 SDK 6.008.031。
- **Xbox One XDK**：已更新至 XDK 10.0.17134.5044 (2018年7月 qfe3)。

# 性能变化

- **WG-42120** 提升了 Voices Graph 的性能。在鼠标没有悬停在连线上时，不会为其使用反锯齿功能。

# API 变化

- **WG-42595** (WAAPI) 现在需要 [ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksinregion.html) 和 [ak.wwise.core.audioSourcePeaks.getMinMaxPeaksInTrimmedRegion](ak_wwise_core_audiosourcepeaks_getminmaxpeaksintrimmedregion.html) 参数（`getCrossChannelPeaks` 除外）。`numPeaks` 参数现在为无符号整数。

# 漏洞修复

- **WG-34254** 已修复：假如 Random Container 和 Sequence Container 的过渡类型为 Trigger Rate，会将 Play Event 过渡应用于所有子对象而非容器本身。
- **WG-41021** 已修复：在将源文件加载到新的 SoundSeed Grain 实例后执行撤消操作会引发崩溃。
- **WG-41128** 已修复：(Linux) 无法加载 DSP 插件。
- **WG-41725** 已修复：在未选择任何路径点的情况下，假如使用鼠标滚轮调节 Positioning Automation 或其他属性的随机值，会发生崩溃。
- **WG-41814** 已修复：在 Event Editor 中单击 Add Exception...（添加例外…）按钮后，假如未在 Project Explorer Browser 中选择任何对象，可能会发生崩溃。
- **WG-41899** 已修复：在快速地重新注册已经注销的游戏对象时，Profiler 的 Emitter-Listener 选项卡中的列表可能会无限增大。
- **WG-41939** 已修复：在内存不足的情况下，部分 Effect 插件发生崩溃。
- **WG-42022** 已修复：现在菜单中的命令扩展按 ID 排序，重新启动 Wwise 后位置保持不变。
- **WG-42026** 已修复：在包含 `NULL` 参数或选项的插件中调用 `AK::Wwise::IPluginPropertySet::WaapiCall` 时发生崩溃。
- **WG-42085** 已修复：Opus 编码器错误地将 6.1 声道源文件上混为 7.1 声道。
- **WG-42103** 已修复：Project Load Log 反复显示曲线范围导致的重复报错。
- **WG-42132** 已修复：(iOS) 在 iPhone 5 和更老机型的设备上播放 Vorbis 音频时，应用程序发生崩溃。
- **WG-42148** 已修复：在将对象拖放到 SoundBank Manager 的列表时，不再重置滚动位置。
- **WG-42202** 已修复：在加载的性能分析会话中包含未安装的 Effect 插件时发生崩溃。
- **WG-42252** 已修复：因内存不足导致 Granular Synthesis 发生崩溃。
- **WG-42278** 已修复：在浏览器中选择带有活跃滤波器的 User-Defined Auxiliary Sends 时，辅助总线不生效。
- **WG-42284** 已修复：在将条目从语音/音效对应的 Contents Editor 拖到 Dialogue Event Editor 时发生崩溃。
- **WG-42440** 已修复：在打开 REV2 Source Editor 视图的情况下生成 SoundBank 时，有时会出现数据丢失。
- **WG-42504** 已修复：在 RTPC 选项卡中查看 Game Object 轨迹时发生罕见崩溃。
- **WG-42607** 已修复：在打开 Source Editor 的情况下，有时无法正确更新 Meter 视图。
- **WG-42615** 已修复：在发送 Stop Event 后在同一帧内触发同一音乐时引发断言。
- **WG-42627** 已修复：在删除搜索结果中的顶部条目后重新创建该条目时，Wwise 发生崩溃。
- **WG-42653** 已修复：在调整 Schematic View 大小时，视图闪烁不定。
- **WG-42654** 已修复：在游戏中，没有将 Make-up Gain 属性应用到总线。
- **WG-42683** 已修复：Generate SoundBanks 视图的进度条没有反应。
- **WG-42721** 已修复：XAudio2 在终止时挂起。
- **WG-42724** 已修复：有时不好区分 Voices Graph 选项卡中的 Auxiliary Bus 虚线路径。
- **WG-42763** 已修复：在打开 Audio Preferences 视图的情况下删除对象可能会引发崩溃。
