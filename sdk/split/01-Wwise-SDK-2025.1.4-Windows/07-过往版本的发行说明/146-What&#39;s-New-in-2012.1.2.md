What's New in 2012.1.2

- Wii U: updated to Cafe SDK 2.04
- 3DS: Updated to CTR-SDK 3.3.1

# 漏洞修复

- **WG-20878** Fixed: Crash in CAkSrcFileADPCM::AcquireVoice on 3DS, when triggering stereo sounds and hitting the hardware voice count limit.
- **WG-20865** Fixed: Can not paste objects in Project Explorer in some scenarios.
- **WG-20890** Fixed: Crash in Music Segment Editor when pasting music clips
