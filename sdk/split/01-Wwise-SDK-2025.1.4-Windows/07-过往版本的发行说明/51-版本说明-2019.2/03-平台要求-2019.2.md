本页记载用于构建 Wwise 声音引擎不同版本的各种工具的版本。

# 通用工具

[Microsoft](namespace_microsoft.html) Visual Studio 的以下版本用于构建 Wwise 程序库 ：

- **[Microsoft](namespace_microsoft.html)® Visual Studio 2015:** VS2015 Update 3 (14.0.25420.1)
- **[Microsoft](namespace_microsoft.html)® Visual Studio 2017:** VS2017 (15.9.12)
- **[Microsoft](namespace_microsoft.html)® Visual Studio 2019:** VS2019 (16.2.5)

您会发现，对于 Win32 和 x64，Wwise SDK 安装目录带有后缀 \_vc + *版本号*。这些版本号对应于：

- **vc140**: Visual Studio 2015 (toolset vc140).
- **vc150**: Visual Studio 2017 (toolset vc141).
- **vc160**: Visual Studio 2019 (toolset vc142).

从 Visual Studio 2015 开始，CRT DLL 版本兼容二进制文件并内嵌在了 Windows 中。

若不希望游戏依赖于 CRT DLL，可改为链接到 Debug (StaticCRT)、Profile (StaticCRT) 和 Release (StaticCRT) 文件夹中的库。
