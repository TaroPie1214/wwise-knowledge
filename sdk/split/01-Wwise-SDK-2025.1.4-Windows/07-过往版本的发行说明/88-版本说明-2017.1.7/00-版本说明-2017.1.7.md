以下各节列举并描述 Wwise 版本 2017.1.6 和版本 2017.1.7 之间的变化。有关早期版本的信息，请参阅 [过往版本的发行说明](previousreleases.html) 章节。如果使用联机帮助，您还可以从目录列的顶部选择别的 Wwise 版本。

# 漏洞修复

- **WG-36559** 已修复：(iOS) 在终止声音引擎时可能会出现崩溃。
- **WG-36830** 已修复：在监控大量游戏对象时，Profiler 的速度会减慢。
- **WG-36958** 已修复：在播放多个音乐实例时忽略了 Change occurs at 的·State。
- **WG-37065** 已修复：在停止时 Unity Editor 可能会出现崩溃。
- **WG-37117** 已修复：SoundBank 元数据文件中替代声部媒体 (Use reference language as stand-in) 的时长信息不正确。
- **WG-37123** 已修复：External Source 输出路径不规范。
- **WG-37156** 已修复：iOS 11 上发生崩溃。
- **WG-37219** 已修复：AkAudioInput.xml 中未列出 Switch 平台。
- **WG-37392** 已修复：SetPluginCustomGameData 出现崩溃。
