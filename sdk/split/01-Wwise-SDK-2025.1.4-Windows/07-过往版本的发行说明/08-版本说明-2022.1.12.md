# 新增功能

- **WG-70303** 为 Wwise Console 的 `generate-soundbank` 动作添加了 `use-user-overrides` 选项。此选项允许在通过 Wwise Console 生成 SoundBank 时使用基于用户的 Override 设置（如自定义 SoundBank 路径）。跟之前一样，在通过 Wwise Console 生成 SoundBank 时默认只使用常用的 Project Settings。

# 其他改进

- **WG-70494** 现在支持通过 Visual Studio 2022 来构建 Level 3 Source Code。

# 漏洞修复

- **WG-65549** 已修复：在接连打开并关闭工程时出现残留 Directory Watcher 线程。
- **WG-69203** 已修复：(Mac Authoring) 在编辑文本字段时没有更新文本。
- **WG-70505** 已修复：在所复制的列表行包含重新排序的列时可能会发生崩溃。
- **WG-70521** 已修复：在使用长的文本字符串筛选 Capture Log 时可能会发生崩溃或死机。
- **WG-70770** 已修复：在基于 ARM 的 CPU 上以 Debug 模式运行时，AkRingBuffer 断言失败。
- **WG-70874** 已修复：在退出 `WaveViewer.exe` 时发生崩溃。

# 社区报告的漏洞修复

- **WG-63424** 已修复：在使用滑杆控件时可能会发生崩溃。
- **WG-70692** 已修复：在启用  `AK_EnableGetSourceStreamBuffering` 的情况下渲染虚声部时发生崩溃。
- **WG-71095** 已修复：Pin/UnpinEventInStreamCache 调用失衡会导致在终止声音引擎时发生死锁。
