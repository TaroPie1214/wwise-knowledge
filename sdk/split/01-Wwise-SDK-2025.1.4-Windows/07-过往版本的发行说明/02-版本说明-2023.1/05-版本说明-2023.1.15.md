# 其他改进

- **WG-77995** 为了与新的 EULA 保持一致，对 License Manager 中的文本做了修改。
- **WG-79106** 对 Wwise 最终用户授权协议进行了更新。

# 漏洞修复

- **WG-78216** 已修复：(Spatial Audio) 在切换 Portal 的启用状态时触发了 AKASSERT(pPortal->IsEnabled()) 断言。
- **WG-78971** 已修复：(Spatial Audio) 在切换 Portal 的启用状态时，AkEdgeGraph.cpp 触发 ASSERT 断言。

# 社区报告的漏洞修复

- **WG-77210** 已修复：在重新加载多个经过修改的 Work Unit 时可能会发生崩溃。
- **WG-77558** 已修复：在使用无效的 Opus 转码声音文件生成 SoundBank 时可能会发生崩溃。
- **WG-78236** 已修复：(Spatial Audio) 在略微旋转 Portal 或将其连到略微旋转的 Room 时，即便存在直接视线，也会将衍射应用于穿过 Portal 的声音和房间底噪。
- **WG-78520** 已修复：在从物理上断开多个输出设备中的第一个时移除了错误的输出设备。
- **WG-78943** 已修复：(Spatial Audio) 在为大型 Portal 应用 Transmission Loss 时衰减距离偏短且没有几何构造。
- **WG-79220** 已修复：在通过 `WwiseConsole.exe` 生成 SoundBank 时没有显示 Source Control 日志消息。

# 文档改进

- **WG-78458** 明确了 `AK::StreamMgr::GetCurrentLanguage()` 的返回格式。
