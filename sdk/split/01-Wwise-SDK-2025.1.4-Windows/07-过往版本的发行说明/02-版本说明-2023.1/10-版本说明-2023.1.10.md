# 行为改进

- **WG-63760** 在 Wwise 设计工具中的插件调用 `AK::Wwise::Plugin::PropertySet` 的 `GetValue` 函数时，倘若参数名称无效，现在会返回零初始化值而非垃圾数据。这样可以避免在生成 SoundBank 的过程中出现媒体源不确定的问题。

# 性能改进

- **WG-74004** (Spatial Audio) 大幅提升了 Portal 和 Geometry 在复杂场景下的性能。

# 其他改进

- **WG-74685** 在 Windows 上，若要重新构建 `Samples` 目录下的插件 DLL，必须选择 "StaticCRT" 构建配置变量。
- **WG-75968** 将 Visual Studio 2022 版本升级到了 17.10.10。

# 漏洞修复

- **WG-73202** 已修复：在删除 Music Segment 时，Wwise 发生崩溃。
- **WG-74401** 已修复：在启用 CPU Profiler 后没有更新相应的列。
- **WG-75172** 已修复：列表中的有些 Severity 标识不一致。
- **WG-75523** 已修复：在 Debug 版本的 Wwise 设计工具中撤消复制粘贴 Master-Mixer Hierarchy 下的总线时触发断言。
- **WG-75575** 已修复：(Spatial Audio) Room 几何包含关系检测射线不像预期的那样是随机的。
- **WG-75611** 已修复：(Spatial Audio) 在 Spatial Audio Listener 上调用 `AK::SoundEngine::SetScalingFactor` 时，没有将缩放系数应用于房间底噪。
- **WG-75649** 已修复：有些性能分析错误消息有时会输出无效的字符。

# 社区报告的漏洞修复

- **WG-68248** 已修复：在使用 Multi Editor 后更改 Override 属性时可能会发生崩溃。
- **WG-70813** 已修复：在连接到文件后，Profiler Session 被系统锁定的时间过长。
- **WG-72481** 已修复：在重新加载 "Switch" 或 "State" Work Unit 时，Music Switch Container 和 Dialogue Event 关联丢失。
- **WG-72740** 已修复：在将非原地对象处理器效果器放在 Master Audio Bus 上时无法重新初始化 Audio Device，进而引发崩溃。
- **WG-72935** 已修复：(WAAPI) 在生成前或生成后操作同样使用 WAAPI 的情况下调用 `ak.wwise.core.soundbank.generate` 时发生死锁。
- **WG-74723** 已修复：在属性使用 Integer 类型时，自定义属性范围被限定为 16 位。
- **WG-75091** 已修复：(Spatial Audio) 在特定配置下对齐 Portal 或将无效的变换数据传给 Portal 参数时触发断言。
- **WG-75240** 已修复：在采用精确到采样点的过渡时，Random Container 和 Sequence Container 没有淡入。
- **WG-75246** 已修复：在通过命令行生成 SoundBank 并覆盖输出路径时，`PlatformInfo.xml` 中的 ExternalSourcesOutputRoot 标记有误。
- **WG-75326** 已修复：在处理 Audio Object 并在 RMS 模式下运行时，Meter 效果器插件偶尔会计算出零输出值。
- **WG-75526** 已修复：在更改 Music Transition 根对象的 Source Type 或 Destination Type 时可能会发生崩溃。
- **WG-75577** 已修复：在测量 Audio Objects 总线的电平时开始或停止播放音频对象，各个音频对象的电平表的顺序变化有时不一致。
- **WG-75655** 已修复：向 Motion 输出设备发送振幅大于 0 dB 的信号可能会导致最终的振动效果混乱无序。
- **WG-75798** 已修复：在加载工程或与游戏断开时可能会发生崩溃。
- **WG-75801** 已修复：在删除 Music Playlist Container 时可能会发生崩溃。
- **WG-75902** 已修复：在将 AK\_INVALID\_GAME\_OBJECT 作为游戏对象传递时，发出误导性的错误消息 "Unknown/Dead game object ID used in <tt>ExecuteActionOnEvent</tt>/<tt>SeekOnEvent</tt>"。
- **WG-76047** 已修复：在通过 Wwise 设计工具执行时，"Set LPF" 和 "Set HPF" Event Action 不起作用。
