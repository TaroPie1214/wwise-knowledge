# 新增功能

- **WG-68194** 现在在没有输入搜索文本的情况下单击 Project Explorer 中的 **Refresh** 按钮也会对内容进行排序。
- **WG-74409** 添加了禁止在双击时打开新选项卡的选项。

# 漏洞修复

- **WG-68568** 已修复：在启用耳机输入后退出耳机演示页面时，Integration Demo 发生崩溃。
- **WG-68572** 已修复：在解除 Project Explorer 焦点时，可能会停止播放 Transport 中正在播放的内容。
- **WG-70733** 已修复：在 Wwise 设计工具中关闭并重新打开工程时错误地报告工程已被另一进程打开。
- **WG-72860** 已修复：在对象被删除的情况下使用 WAAPI 主题 `ak.wwise.ui.commands.executed` 可能会引发崩溃。
- **WG-72976** 已修复：Project Migration 对话框中的文本被截断。
- **WG-73603** 已修复：(Spatial Audio) 在个别情况下，几何构造可能会导致误报边界边缘不严密，进而不必要地触发断言。
- **WG-73759** 已修复：在更改 Event 所含动作的类型或作用域时，会影响动作的内部顺序。
- **WG-73860** 已修复：在使用允许旁通所有效果器的 RTPC 时，Integrity Report 中出现误报。
- **WG-73901** 已修复：在通过设为 None 的效果器打开新的窗口时发生崩溃。
- **WG-73977** 已修复：(Spatial Audio) 在穿过 Portal 的路径上，Reverb Zone 缺少衍射。
- **WG-73997** 已修复：(Spatial Audio) 在使用 Distance Probe 时，若路径穿过有空隙的 Portal，会报告错误的距离。
- **WG-74076** 已修复：(Spatial Audio) 内存不足会触发断言。
- **WG-74152** 已修复：在连接游戏的情况下删除效果器时发生崩溃。
- **WG-74210** 已修复：在内存不足的情况下，Debug 配置的 Spatial Audio 模块断言失败。
- **WG-74212** 已修复：(Spatial Audio) 在将 `AK::SoundEngine:SetScalingFactor` 应用于 Spatial Audio Listener 时，路径计算没有按比例对用于最大路径长度的值进行缩放，导致声音突然中断。
- **WG-74278** 已修复：在内存不足的情况下终止 Wwise Recorder 插件会发生无限循环。
- **WG-74279** 已修复：(Spatial Audio) 若存在八条以上衍射路径，有时会舍弃较为突出的路径，而保留不那么突出的路径。
- **WG-74316** 已修复：要多执行一次撤消操作才能撤消为 RTPC 的 X 轴添加 Game Parameter（如 Envelope 或 LFO）。
- **WG-74426** 已修复：(Spatial Audio) 在将 Wwise 23.1.6 连接到使用 Wwise 23.1.5 或更早版本构建的性能分析会话或游戏时发生崩溃。
- **WG-74469** 已修复：CPU Timeline 视图中的 Voice 和 Bus 没有按照 Wwise 工程中指定的颜色显示。

# 社区报告的漏洞修复

- **WG-70074** 已修复：无法使用 WAAPI 函数 `ak.wwise.core.object.getAttenuationCurve` 和 `ak.wwise.core.object.setAttenuationCurve` 访问所有衰减曲线。
- **WG-71898** 已修复：在所加载的 SoundBank 包含已经加载并在播放的 Music Track 时，声音引擎可能会发生崩溃。
- **WG-72685** 已修复：在选择 **Load Preset** 和 **Save Preset** 对话框中的列时可能会发生崩溃。
- **WG-73493** 已修复：负责控制 RTPC 的全局调制器会影响循环容器中播放的声音并触发断言。
- **WG-73989** 已修复：在使用 Live Media Transfer 时发生崩溃。
- **WG-74202** 已修复：在连接游戏的情况下无法正确地对在 Wwise 中察看的源插件进行初始化。
- **WG-74225** 已修复：在对使用转码媒体素材的效果器（如 [AK](namespace_a_k.html "Definition of data structures for AkAudioObject") Convolution 或 Impact）上的属性进行修改而其被用于播放 Actor-Mixer Hierarchy 下的容器时音频渲染发生崩溃或出现其他问题。
- **WG-74437** 已修复：在选择 Profile and Edit (Sync All Modified Objects) 的情况下重新连接 Wwise Profiler 时发生崩溃。
- **WG-74519** 已修复：Spatial Audio `SetGeometry` API 发生崩溃。

# 文档改进

- **WG-73974** 对 [针对不同的 Wwise 平台构建工程](effectplugin_tools_building.html) 中的文本格式做了改进。
