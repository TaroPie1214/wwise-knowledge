# 新增功能

- **WG-68160** File Manager 现在支持 MIDI 文件。

# 行为改进

- **WG-73325** CPU Timeline 和 CPU Usage 坐标图现在包含声音引擎执行期间等待获取全局锁花费的时间。这样有助于确认音频渲染当中的闲置情况。

# Miscellaneous Changes

- **WG-73326** 对 IntegrationDemo 中的 `Demo3dAudio.cpp` 进行了更新，以包含相应示例来展示如何使用 `AK::SoundEngine::RegisterAudioDeviceStatusCallback` 对设备初始化当中的更改做出响应。

# 漏洞修复

- **WG-68970** 已修复：Unreal Brush 包含未对齐边缘的三角形（含 T 型交点），导致边界边缘被误报并发出 "Geometry is not watertight" 警告，使得无法将几何构造用作 Room。
- **WG-69595** 已修复：(Spatial Audio) 路径会从开放表面之间有空隙的 Portal 逸出。
- **WG-71103** 已修复：(Spatial Audio) Reverb Zone 的父对象的距离计算有误。
- **WG-71800** 已修复：(Spatial Audio) 并排放置的 Portal 使用了错误的衍射点。
- **WG-71929** 在使用 High DPI 监视器时，单击 Project Explorer 中的对象有时不会在 Property Editor 中相应地予以显示。
- **WG-72811** 已修复：Performance Monitor 中 Average 列的统计数据有时显示 "-nan %"。
- **WG-72990** 已修复：属性被覆盖时，Voice Inspector 仍然显示了父节点对声部的影响（如 Output Bus Volume）。
- **WG-73058** 已修复：在单击填充有大量内容的 Capture Log 时可能会选中错误的行。
- **WG-73129** 已修复：(Spatial Audio) 对 `AkInitSettings::fGameUnitsToMeters` 的解释有误，导致混响的空间化出错。
- **WG-73161** 已修复：在尝试使用 Music Segment 快捷菜单时可能会发生崩溃。
- **WG-73196** 已修复：Stinger 的 Play At 属性值 Next Beat 和 Next Bar 颠倒了。
- **WG-73231** 已修复：(Spatial Audio) 在听者穿越不同的 Room 时，有时会将与外部 Room 的距离返回为很大的数值，导致听不到该 Room 中的声音。
- **WG-73299** 已修复：在重新加载工程后将原本为 0 的 Game Parameter Simulation Value 重置为了默认值。
- **WG-73329** 已修复：Music Playlist Item 不允许将 Avoid Repeat 设为零。
- **WG-73372** 已修复：在启用 Play Originals 且 Trim Start 值很小时，可能会从开头而非给定值位置播放声音。

# 社区报告的漏洞修复

- **WG-72564** 已修复：加载空音乐切换容器时由于内存不足导致的音频包加载失败。
- **WG-53536** 已修复：在使用 Convolution Reverb 的情况下加载或卸载音频素材时发生崩溃。
- **WG-73169** 已修复：(Spatial Audio) 在移除 Outdoor Room 时，`AkAcousticPortal::CalcRatio` 发生崩溃。
- **WG-73272** 已修复：(Spatial Audio) ErrorCode\_SpatialAudio\_PortalNotFound 没有正确报告 Portal ID。
- **WG-73287** 已修复：在几何构造包含重复或冗余的三角形时，可能会产生边界边缘并发出 "Geometry is not watertight" 警告。
- **WG-73328** 已修复：在取消预备互动音乐内容时，捕获日志中误报 "Failed to unprepare event" 错误。
