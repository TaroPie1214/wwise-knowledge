版本说明 2017.1.3

以下各节列举并描述Wwise版本2017.1.2与版本2017.1.3之间的变化。有关早期版本的信息，请参阅 [过往版本的发行说明](previousreleases.html) 章节。如果使用联机帮助，您还可以从目录列的顶部选择别的 Wwise 版本。

# 平台 SDK 的变化

- **PS4 SDK:** PS4 VSI integration minimal support is now Visual Studio 2015.

# 行为变化

- **WG-34958** Faders and sliders now only react to mouse wheel when they have keyboard focus.

# 漏洞修复

- **WG-32997** 已修复：在通过右键单击导入时，无法导入各语种的语音素材。
- **WG-34477** 已修复：在 (WAAPI) object.childAdded 和 childRemoved 选项中包含 shortid 时会生成断言 (Assert)。
- **WG-34648** 已修复：在运行内存不足时，CAkLimiter::Remove() 可能会出现故障。
- **WG-34701** 已修复：在管线缓冲区大小不是 2 的乘幂时，卷积混响出现故障。
- **WG-34728** 已修复：在注销“听者”游戏对象后出现崩溃。
- **WG-34733** 已解决：Reflect 的 Mute／Solo 在断开连接时的变更无法在连接后推送到游戏中去。
- **WG-34734** 已解决：退出监视 Reflect 插件时发生崩溃。
- **WG-34753** 已修复：在注销游戏对象时，未清理切换开关值的内存。
- **WG-34818** 已修复：若在捕获开始时 Capture Log 视图已打开，则 Save Log 仅允许保存 .prof。
- **WG-34828** 已修复：在平台变换后关闭 Wwise 时会触发断言。
- **WG-34870** 已修复：在首次连接后，捕获的 .prof 会话中不显示 Game Object 名称。
- **WG-34884** 已修复：在结合使用用户定义的定位（未选中 **Follow Listener Orientation**）和游戏定义的发送（未指定听者）时出现崩溃。
- **WG-34885** 已修复：Control Surface Bindings 组中不显示索引列表。
- **WG-34915** 已修复：Wwise Reflect 中触发罕见和良性断言。
- **WG-34973** 已修复：PS4 上的振动出现内存泄漏。
- **WG-35004** 已修复：总线应用的 State 导致音频包随机变更。
- **WG-35020** 已修复：在结合 `SetMedia` 或 `PrepareMedia` 使用 Vorbis 数据时出现崩溃。
- **WG-35086** 已修复：iOS 11：AAC 解码器故障。
- **WG-35098** 已修复：在 Android 上远程连接时，Reflect 出现崩溃。
