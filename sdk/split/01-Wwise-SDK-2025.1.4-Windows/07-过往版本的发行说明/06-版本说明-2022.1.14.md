# 漏洞修复

- **WG-70816** 已修复：(Spatial Audio) 在 Game Object 3D Viewer 中显示 Portal 信息时，无法正确显示括号中所示的 Room ID。
- **WG-72028** 已修复：在部分平台上使用有些自定义内存分配挂钩时，因为使用了带有标记的指针而导致 AkBusCallbackMgr 发生崩溃。
- **WG-72104** 已修复：在内存不足的情况下无法正确清理 Prepared Game Sync。
- **WG-72280** 已修复：在 `StreamMgr` 出现争用问题时可能会发生崩溃。
- **WG-72319** 已修复：(WwiseConsole) 无法解析相对工程路径。

# 社区报告的漏洞修复

- **WG-71749** 已修复：在按列对 Work Unit 进行排序后没有更新 File Manager 版本控制信息。
- **WG-72060** 已修复：Windows 平台上出现争用问题，导致在终止 Sound Engine 的过程中 Windows 回调发生崩溃。
- **WG-72095** 已修复：在 Attack Time 或 Release Time 过短时，Mastering Suite 的 Multiband Compressor 会生成杂音。
- **WG-72100** 已修复：对 I/O 挂钩 `Close` 函数的调用发生延迟。这会导致文件保持打开状态的时间过长。无需更改 I/O 挂钩代码。
- **WG-72101** 已修复：在使用 Job Manager 同时执行多项 Spatial Audio 任务时发生崩溃。
- **WG-72381** 已修复：在使用 Opus 或 Vorbis 的情况下调节 Pitch 时，5.1 声音存在 Right 和 Center 声道互换的情况。

# 文档改进

- **WG-71448** 明确了诸多场景下 Convolution Reverb 输入信号的来源。
