# 新增功能

- **WG-79096** 在 Property Editor 中单击展开箭头后，可使用向左键和向右键展开或折叠分组。
- **WG-79513** Audio File Importer 和 Media Pool 现在允许将区域导入到 Music Clip 中。
- **WG-80946** (WAAPI) `ak.wwise.waapi.getSchema` 现在会返回对函数或主题的描述。
- **WG-80965** (WAAPI) `ak.wwise.core.executeLuaScript` now supports `luaString` as an argument to execute Lua code without a file.

# 行为改进

- **WG-80630** 在处于 waapi-server 模式时，WwiseConsole 现在会发布 `ak.wwise.core.log.itemAdded` 主题，并通过 `ak.wwise.core.log.get` 返回结果。
- **WG-80891** 对于可能有意插入静默时段的一次性声音，Media Pool 区域检测不会再尝试包含开头和结尾的无声内容。现在会将这些无声内容自动剪掉。

# 性能改进

- **WG-81182** 将虚声部的性能提升了 15。
- **WG-81236** 提升了设计工具内声音引擎更新的性能。

# 其他改进

- **WG-78734** 在 Property Editor 中为 Favorites 类别添加了 **Reset to Default** 选项。
- **WG-80821** 对 Blend Container 和 Switch Container 中所用 RTPC 的变化做了小幅优化。

- **WG-79362** 已修复：在首次为工程编制索引时进程没有响应。

- **WG-80095** 已修复：在流播放的声部由虚声部转为实声部时可能会触发断言。
- **WG-80161** 已修复： `AkRTree` 接收到不规范的几何构造，导致陷入无限维度计算并引发崩溃。
- **WG-80401** 已修复：在 Attenuation Settings 坐标图中移动光标时没有针对 Transport Control 对象更新可闻衰减预览。
- **WG-80544** 已修复：在 MacOS 上通过触控板滚动时没有反应。
- **WG-80615** 已修复：在选择 Next Cue 或 Custom Cue 时没有启用 Stinger Cue Name 字段。
- **WG-80724** 已修复：(Acoustics) 在使用 Distance Probe 的情况下处于带房间底噪的 Room 之内时距离衰减无效。 无法通过远离发声体的第一个 Portal 正确地对 Room Transmission Loss 做插值处理。
- **WG-80783** Fixed: `.meta` files are parsed by the project database during asset import.
- **WG-80807** 已修复：在切换布局时，浮动窗口会退出最大化状态。
- **WG-80940** 已修复：(Acoustics) `AkSoundGeometry::DistanceSqToPortalCutout()` 发生崩溃。

- **WG-80998** 已修复：在退出失焦/挂起的 Unity 游戏时卡死。

- **WG-80424** 对 [声障、声笼及 Game-defined Auxiliary Sends](soundengine_obsocc.html) 章节进行了更新。
- **WG-80725** 明确了通过 MIDI 触发 Spatial Audio Event 的相关限制。
- **WG-80818** 对[管理文件包](https://www.audiokinetic.com/library/edge/?source=Help&id=Managing_File_Packages)及子页面中的内容做了更新和调整。明确了打包 DLC 的最佳做法。
- **WG-80982** 在 [Balance-Fade Speaker Panning 图解](https://www.audiokinetic.com/library/edge/?source=Help&id=balance_fade_illustrated)中对上混场景做了详细的说明。
