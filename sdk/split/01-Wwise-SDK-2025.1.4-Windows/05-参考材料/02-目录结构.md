# SDK 输入库 与分布式二进制文件

The input libraries required to integrate the Wwise Sound Engine are available for each [build configuration](goingfurther_builds.html#wwise_sdk_build_config) in their respective platform folders contained in the root of the SDK installation directory. 程序库目录结构定义如下：

- {Wwise SDK Dir}/{平台名称}/{配置名称}/lib，其中
  - {Wwise SDK Dir} 是 SDK 安装目录
  - {平台名称}是可用平台名称（x64、Mac、iOS……）之一。后缀可进一步限定目标 CPU 架构或编译器。
  - {配置名称} 是三个构建配置（Debug、Profile、Release）之一。

例如，在 {Wwise SDK Dir}\x64\_vc160\Profile\lib 中可以找到使用 Visual Studio 2019 的 Windows 平台的性能分析库。

请参阅 [构建配置](goingfurther_builds.html) 了解此 SDK 中所含各种库的描述。
