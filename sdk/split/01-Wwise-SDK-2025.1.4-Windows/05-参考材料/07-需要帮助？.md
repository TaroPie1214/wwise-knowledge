# 使用帮助

Wwise Help 包含相应的参考主题，方便查看 Wwise 中各项界面元素的详细信息。

若要在 Wwise 内打开与当前视图或对话框相关的参考主题，请执行以下操作之一：

- 在任一视图或对话框的标题栏中单击 Help 图标。
- 在菜单栏中依次单击 **Help** > **Wwise Help**。
- 按下 **F1**。

# 获取支持

如果遇到困难或认为可能检测到了错误，请先确保当前所用为最新 Wwise 小版本。因为每次小版本更新都会进行修复，相关的问题有可能已经得到了解决。

如问题仍然存在，可考虑以下方案：

- 通过官网的[问答版块](https://www.audiokinetic.com/qa/)向社区求助。
- 通过在 Audiokinetic Launcher 中依次选择 **Help** > **Report a Bug...** 来报告问题。有关操作流程的详细信息，请参阅[报告 Wwise 错误](https://www.audiokinetic.com/library/wwise_launcher/?source=InstallGuide&id=reporting_bugs)。
- 通过 Audiokinetic [客户门户](https://www.audiokinetic.com/customers/)提交技术支持申请单。
