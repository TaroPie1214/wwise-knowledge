# 通用工具

使用以下 [Microsoft](namespace_microsoft.html) Visual Studio 版本构建 Wwise 声音引擎库：

- **Microsoft® Visual Studio 2017:** VS2017 (15.9.61)
- **Microsoft® Visual Studio 2019:** VS2019 (16.11.40)
- **Microsoft® Visual Studio 2022:** VS2022 (17.10.10)

The Wwise Authoring application is built using the same Microsoft® Visual Studio 2022 version used to build the Sound Engine libraries. 建议使用该版本构建 Wwise 设计工具插件。

您会发现，对于 Win32 和 x64，Wwise SDK 安装目录带有后缀 \_vc + *版本号*。这些版本号对应于：

- **vc160**: Visual Studio 2019 (toolset vc142).
- **vc170**: Visual Studio 2022 (toolset vc143).

CRT DLL 版本兼容二进制文件并内嵌在了 Windows 中。

若不希望游戏依赖于 CRT DLL，可改为链接到 Debug (StaticCRT)、Profile (StaticCRT) 和 Release (StaticCRT) 文件夹中的库。
