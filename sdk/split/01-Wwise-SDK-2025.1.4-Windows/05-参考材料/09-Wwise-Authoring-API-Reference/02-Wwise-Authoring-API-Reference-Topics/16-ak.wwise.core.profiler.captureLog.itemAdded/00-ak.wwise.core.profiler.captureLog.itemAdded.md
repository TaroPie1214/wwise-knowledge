# ak.wwise.core.profiler.captureLog.itemAdded

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.captureLog.itemAdded

在新的条目被添加到 Capture Log 中时发送。注意，将发送所有条目。不会像 Capture Log 视图一样应用筛选。

# 选项

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **types** | array | 定义要接收哪种类型的 Capture Log 条目。此筛选器有助于减少序列化数据的数量。若未指定任何类型，则发送所有类型的条目。 |
| types **[...]** | string | 可能的值：  - Notification - InteractiveMusic - Midi - ExternalSource - Marker - State - Switch - SetParameter - ParameterChanged - Bank - Prepare - Event - DialogueEventResolved - ActionTriggered - ActionDelayed - Message - APICall - GameObjectRegistration |

[选项架构](ak_wwise_core_profiler_capturelog_itemadded_options_schema.html)

# 发布

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **type** \* | string | 条目类型。与 Capture Log 中的 Type 列对应。 |
| **time** \* | integer | Event 发生的时间。此项为从声音引擎初始化开始所经过的时间（毫秒）。  32 位整数。范围：[-2147483648,2147483647] |
| **objectId** | string | 条目对应对象的 ID (GUID)。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| **objectName** | string | 条目对应对象的名称。 |
| **objectShortId** | integer | 条目对应对象的 Short ID（32 位）。  32 位无符号整数。范围：[0,4294967295] |
| **gameObjectId** | integer | 条目对应的游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **gameObjectName** | string | 条目对应的游戏对象名称。 |
| **playingId** | integer | 条目对应的 Playing ID。  32 位无符号整数。范围：[0,4294967295] |
| **description** \* | string | 条目的描述。与 Capture Log 中的 Description 列对应。 |
| **severity** \* | string | Normal：正常执行。Message：在 Capture Log 中显示为黄色。Error：在 Capture Log 中显示为红色。可能的值：  - Normal - Message - Error |
| **errorCodeName** | string | 条目对应的错误代码名称。 |

(必填 \*)
