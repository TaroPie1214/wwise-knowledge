# Publish

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **time** \* | integer | 游戏对象的注销时间。此项为从声音引擎初始化开始所经过的时间（毫秒）。  32 位整数。范围：[-2147483648,2147483647] |
| **gameObjectId** \* | integer | 条目对应的游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **gameObjectName** \* | string | 条目对应的游戏对象名称。 |

(必填 \*)
