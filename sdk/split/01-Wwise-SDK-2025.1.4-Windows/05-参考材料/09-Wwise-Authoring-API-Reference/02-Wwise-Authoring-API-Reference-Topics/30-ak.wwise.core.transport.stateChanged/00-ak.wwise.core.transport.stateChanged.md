|  |

| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.transport.stateChanged

在走带对象的 State 发生变化时发送。

# 选项

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **transport** \* | integer | 被监视走带对象的 ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)

[选项架构](ak_wwise_core_transport_statechanged_options_schema.html)

# 发布

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **transport** \* | integer | 被监视走带对象的 ID。  32 位无符号整数。范围：[0,4294967295] |
| **state** \* | string | 走带对象的 State。可能的值：  - playing - stopped - paused |
| **object** \* | string | 由走带对象控制的对象的 ID (GUID)。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| **gameObject** | integer | 走带对象所用的游戏对象。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |

(必填 \*)
