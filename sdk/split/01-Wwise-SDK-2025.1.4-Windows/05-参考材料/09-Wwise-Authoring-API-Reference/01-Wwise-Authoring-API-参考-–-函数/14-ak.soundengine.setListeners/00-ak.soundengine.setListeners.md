# ak.soundengine.setListeners

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setListeners

设置单个游戏对象的活跃听者。在默认情况下，所有新的游戏对象都没有活跃听者，不过可使用 `SetDefaultListeners()` 来改写此行为。非活跃听者不实施计算。参阅 `AK::SoundEngine::SetListeners` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **emitter** \* | integer | “发声体”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **listeners** \* | array | 此数组中包含一系列“听者”游戏对象 ID。游戏对象必须之前已经注册过。 |
| listeners **[...]** | integer | 游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |

(必填 \*)
