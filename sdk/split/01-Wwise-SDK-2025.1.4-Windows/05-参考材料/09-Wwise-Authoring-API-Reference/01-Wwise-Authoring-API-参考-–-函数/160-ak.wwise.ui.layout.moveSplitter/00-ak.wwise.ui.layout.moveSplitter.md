# ak.wwise.ui.layout.moveSplitter

|  |

| Wwise SDK 2025.1.4 - Windows |

ak.wwise.ui.layout.moveSplitter

Moves a splitter by a delta given in pixels.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **id** \* | string | The ID of the splitter to move. |
| **delta** \* | integer | The delta, in pixels, to add to the splitter position. |

(必填 \*)
