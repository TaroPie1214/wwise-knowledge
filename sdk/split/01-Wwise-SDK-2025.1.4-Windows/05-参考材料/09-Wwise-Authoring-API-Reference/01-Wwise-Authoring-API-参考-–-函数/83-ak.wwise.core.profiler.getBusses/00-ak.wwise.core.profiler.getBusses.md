# ak.wwise.core.profiler.getBusses

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getBusses

在特定性能分析器捕获时间检索总线。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **time** \* |  | 指定要在什么时间（毫秒）查询总线或通过哪种 Time Cursor 获取时间。User Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  One of: |
| integer | 范围：[0,\*] |
| string | 识别其中一种全局性能分析器光标。可能的值：  - user - capture |
| **busPipelineID** | integer | 所要获取的单个 Bus 实例的 Pipeline ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getbusses_args_schema.html)

# 选项

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** | array | 要针对每条总线返回的成员。默认设为 Pipeline ID、Game Object ID 和 Object GUID。  此数组中包含所要接收的选项。 |
| return **[...]** | string | 针对总线管线结构指定其中一个定义的成员。可能的值：  - pipelineID - mixBusID - objectGUID - objectName - gameObjectID - gameObjectName - deviceID - volume - downstreamGain - voiceCount - effectCount - depth |

[选项架构](ak_wwise_core_profiler_getbusses_options_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | array | 此数组中包含一系列总线。 |
| return **[...]** | object | 总线条目。  混音总线对象。使用 return 选项来指定所需的属性。 |
| return[...].**pipelineID** | integer | 总线的 Pipeline ID。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**mixBusID** |  | Unique ID assigned to a mixing bus.  Any of: |
| integer | Unsigned Integer 64-bit. 范围：[0,18446744073709551615] |
| integer | Integer 64-bit. 范围: [-9223372036854775808,9223372036854775807] |
| return[...].**objectGUID** | string | 与总线对应的 Object GUID。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| return[...].**objectName** | string | 与总线对应的 Object Name。 |
| return[...].**gameObjectID** |  | Game Object ID corresponding to the voice.  Any of: |
| integer | Unsigned Integer 64-bit. 范围：[0,18446744073709551615] |
| integer | Integer 64-bit. 范围: [-9223372036854775808,9223372036854775807] |
| return[...].**gameObjectName** | string | 与声部对应的 Game Object Name。 |
| return[...].**deviceID** | integer | 音频输出设备 ID。  64 位无符号整数。范围：[0,18446744073709551615] |
| return[...].**volume** | number | 总线的增益 (dB)。 |
| return[...].**downstreamGain** | number | 从当前总线到输出的增益 (dB)。 |
| return[...].**voiceCount** | integer | 输出到总线的声部数。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**depth** | integer | 管线中的总线的 Depth 电平。 |
| return[...].**effectCount** | integer | Number of effects on the bus.  Unsigned Integer 32-bit. 范围：[0,4294967295] |

(必填 \*)
