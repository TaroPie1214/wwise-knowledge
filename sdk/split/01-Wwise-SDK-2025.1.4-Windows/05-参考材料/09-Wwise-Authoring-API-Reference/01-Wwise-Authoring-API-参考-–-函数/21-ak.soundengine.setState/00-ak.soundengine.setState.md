# ak.soundengine.setState

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setState

设置 State Group 的 State。参阅 `AK::SoundEngine::SetState` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **stateGroup** \* |  | State Group 的 ID (GUID)、名称或 Short ID。  Any of: |
| string | 对象的名称。 |
| string | 以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| integer | Wwise 对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |
| **state** \* |  | State 的 ID (GUID)、名称或 Short ID。  Any of: |
| string | 对象的名称。 |
| string | 以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| integer | Wwise 对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)
