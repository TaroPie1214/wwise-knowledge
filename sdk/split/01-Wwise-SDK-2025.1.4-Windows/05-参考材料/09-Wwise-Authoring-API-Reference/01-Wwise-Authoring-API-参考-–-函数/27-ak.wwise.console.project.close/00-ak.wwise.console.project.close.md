# ak.wwise.console.project.close

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.console.project.close

关闭当前工程。This operation is synchronous.

# Result

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **hadProjectOpen** | boolean | 若有工程打开，则返回 true，否则返回 false。注意，若无工程打开，则不发送 "ak.wwise.core.project.preClosed" 或 "ak.wwise.core.project.postClosed" Event。 |
