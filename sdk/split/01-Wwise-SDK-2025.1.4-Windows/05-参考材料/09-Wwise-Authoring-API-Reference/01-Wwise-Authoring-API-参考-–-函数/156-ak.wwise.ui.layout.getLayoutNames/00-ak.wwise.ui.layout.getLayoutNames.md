# ak.wwise.ui.layout.getLayoutNames

|  |

| Wwise SDK 2025.1.4 - Windows |

ak.wwise.ui.layout.getLayoutNames

Retrieves a list of the factory layout names.

# Result

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **layouts** | array | An array of the factory layout names. |
| layouts **[...]** | string | Layout names, which can be used with [ak.wwise.ui.layout.getLayout](ak_wwise_ui_layout_getlayout.html). |
