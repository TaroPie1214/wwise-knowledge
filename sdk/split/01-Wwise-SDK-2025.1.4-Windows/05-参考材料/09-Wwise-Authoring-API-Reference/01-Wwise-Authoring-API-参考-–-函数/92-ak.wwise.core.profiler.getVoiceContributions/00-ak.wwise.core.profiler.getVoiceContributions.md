# ak.wwise.core.profiler.getVoiceContributions

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getVoiceContributions

检索影响声部路径的声部音量、高通滤波和低通滤波的所有参数（通过 Pipeline ID 进行解析）。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **voicePipelineID** \* | number | 要从中获取影响的声部的 Pipeline ID。  识别在播声部实例 ID。范围：[0,\*] |
| **bussesPipelineID** | array | 此数组中包含一系列总线 Pipeline ID。 |
| bussesPipelineID **[...]** | number | 属于通用声部路径的总线的 Pipeline ID。默认设为干声路径的空数组。  识别在播声部实例 ID。范围：[0,\*] |
| **time** \* |  | 指定要在什么时间（毫秒）查询声部或通过哪种 Time Cursor 获取时间。User Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  One of: |
| integer | 范围：[0,\*] |
| string | 识别其中一种全局性能分析器光标。可能的值：  - user - capture |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getvoicecontributions_args_schema.html)
