# ak.soundengine.registerGameObj

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.registerGameObj

注册游戏对象。将游戏对象注册两次并没有什么作用。注销一次便可将其彻底注销，无论其之前被注册过多少次。参阅 `AK::SoundEngine::RegisterGameObj` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **gameObject** \* | integer | ID of the game object to be registered. Valid range is [0, 0xFFFFFFFFFFFFFFDF].  A game object ID, unsigned integer 64-bit. 范围：[0,18446744073709551615] |
| **name** \* | string | 游戏对象的名称（用于监视）。 |

(必填 \*)
