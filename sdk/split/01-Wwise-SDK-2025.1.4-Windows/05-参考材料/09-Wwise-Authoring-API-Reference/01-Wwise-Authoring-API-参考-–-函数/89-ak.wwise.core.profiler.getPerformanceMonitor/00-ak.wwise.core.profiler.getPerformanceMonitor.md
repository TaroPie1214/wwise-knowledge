# ak.wwise.core.profiler.getPerformanceMonitor

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getPerformanceMonitor

Retrieves the Performance Monitor statistics at a specific profiler capture time. See [Wwise Authoring Performance Monitor Counter Identifiers](globalcountersids.html) for the available counters.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **time** \* |  | Time in milliseconds to query for Performance Monitor data, or a Time Cursor from which to acquire the time. User Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  One of: |
| integer | 范围：[0,\*] |
| string | 识别其中一种全局性能分析器光标。可能的值：  - user - capture |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getperformancemonitor_args_schema.html)

# Result

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | array | Array of Performance Monitor counters. |
| return **[...]** | object | Performance Monitor counter.  Performance Monitor counter values. |
| return[...].**name** \* | string | name of the counter as shown in Wwise Authoring. |
| return[...].**id** \* | string | unique Id of the counter. |
| return[...].**value** \* | number | value of counter at given time. |

(必填 \*)

[结果架构](ak_wwise_core_profiler_getperformancemonitor_result_schema.html)
