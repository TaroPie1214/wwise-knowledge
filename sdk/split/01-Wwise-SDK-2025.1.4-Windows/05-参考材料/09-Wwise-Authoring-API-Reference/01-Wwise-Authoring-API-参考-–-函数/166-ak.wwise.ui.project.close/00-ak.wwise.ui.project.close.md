# ak.wwise.ui.project.close

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.ui.project.close

关闭当前工程。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **bypassSave** | boolean | 指示是否不要提示用户保存当前工程。默认设为 true。 |

[参数架构](ak_wwise_ui_project_close_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **hadProjectOpen** | boolean | 若有工程打开，则返回 true，否则返回 false。注意，若无工程打开，则不发送 "ak.wwise.core.project.preClosed" 或 "ak.wwise.core.project.postClosed" Event。 |

[结果架构](ak_wwise_ui_project_close_result_schema.html)
