ak.wwise.core.log.clear

Clears the logs on the specified channel.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **channel** | string | The channel to clear. Default is 'general'.  The log channel. 可能的值：  - soundbankGenerate - conversion - copyPlatformSettings - waapi - projectLoad - general - sourceControl - lua |
