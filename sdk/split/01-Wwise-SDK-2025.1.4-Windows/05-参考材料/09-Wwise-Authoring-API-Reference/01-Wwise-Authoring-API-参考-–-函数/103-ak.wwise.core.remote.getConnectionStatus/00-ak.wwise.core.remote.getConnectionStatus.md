# ak.wwise.core.remote.getConnectionStatus

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.remote.getConnectionStatus

检索连接状态。

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **isConnected** \* | boolean | 指示 Wwise 设计工具是否被连接到了 Wwise 声音引擎进程。 |
| **status** \* | string | 当前连接状态（以文本形式显示）。 |
| **console** | object | 描述远程进程。 |
| console.**name** \* | string | 可执行文件返回的远程主机的名称。 |
| console.**platform** \* | string | 可执行文件返回的远程主机的平台。 |
| console.**customPlatform** \* | string | 此平台与可执行文件返回的远程主机的工程平台设定相同。 |
| console.**host** \* | string | 所连主机的 IPv4 地址。若 Wwise 被连接到本地文件（性能分析器会话），则此项还可为文件路径。 |
| console.**appName** \* | string | 可执行文件返回的所连应用程序的名称。 |

(必填 \*)
