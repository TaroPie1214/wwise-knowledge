ak.wwise.ui.layout.resetLayouts

Reset layouts to their default state.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **layouts** | array | An array of names of the layout to reset. |
| layouts **[...]** | string | Layout names, which can be used with [ak.wwise.ui.layout.getLayout](ak_wwise_ui_layout_getlayout.html). |
