# ak.wwise.core.transport.executeAction

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.transport.executeAction

针对给定走带对象或所有走带对象（如未作任何指定）执行某项动作。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **transport** | integer | 要与所有其他 ak.wwise.core.transport 函数结合使用的走带对象 ID。  32 位无符号整数。范围：[0,4294967295] |
| **action** \* | string | 所要执行的动作。可能的值：  - play - stop - pause - playStop - playDirectly |

(必填 \*)
