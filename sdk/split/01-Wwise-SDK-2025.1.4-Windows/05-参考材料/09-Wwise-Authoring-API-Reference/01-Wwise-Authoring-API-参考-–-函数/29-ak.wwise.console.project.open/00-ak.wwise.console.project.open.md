# ak.wwise.console.project.open

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.console.project.open

打开由路径指定的工程。This operation is synchronous.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **path** \* | string | The path to the project WPROJ file. For using WAAPI on Mac, please see [在 Mac 上使用 WAAPI](waapi_prepare.html#waapi_path_on_mac) . |
| **onMigrationRequired** | string | The action to be executed when opening a project that needs to be migrated. 可能的值：  - migrate - fail |
| **autoCheckOutToSourceControl** | boolean | Determines if Wwise automatically performs a Checkout source control operation for affected work units and for the project. 默认设为 true。 |

(必填 \*)
