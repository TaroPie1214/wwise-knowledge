# ak.soundengine.setGameObjectOutputBusVolume

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setGameObjectOutputBusVolume

设置给定游戏对象的输出总线音量（直达声）。参阅 `AK::SoundEngine::SetGameObjectOutputBusVolume` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **emitter** \* | integer | 关联“发声体”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **listener** \* | integer | 关联“听者”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **controlValue** \* | number | 倍数：0 表示静音，1 表示不变。也就是说，在值介于 0 ~ 1 之间时会将声音减小，在大于 1 时会将声音放大。 |

(必填 \*)
