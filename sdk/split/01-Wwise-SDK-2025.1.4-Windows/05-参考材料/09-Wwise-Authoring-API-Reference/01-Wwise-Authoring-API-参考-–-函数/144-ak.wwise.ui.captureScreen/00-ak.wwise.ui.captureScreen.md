# ak.wwise.ui.captureScreen

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.ui.captureScreen

捕获相对于视图的部分 Wwise UI。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **viewName** | string | The name of the view. 在默认情况下，捕获整个 UI。 |
| **viewSelectionChannel** | integer | 视图的选定声道。值可为 1、2、3 或 4。在默认情况下，自动检测视图的当前选定声道。 |
| **rect** | object | 捕获区域。在默认情况下，捕获整个视图。 |
| rect.**x** \* | integer | 捕获区域的左侧位置（像素）。 |
| rect.**y** \* | integer | 捕获区域的顶部位置（像素）。 |
| rect.**width** \* | integer | 捕获区域的宽度（像素）。 |
| rect.**height** \* | integer | 捕获区域的高度（像素）。 |

[参数架构](ak_wwise_ui_capturescreen_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **contentType** | string | 底层图像数据格式（如 image/png）。 |
| **contentBase64** | string | 编码的图像数据 (Base64)。 |

[结果架构](ak_wwise_ui_capturescreen_result_schema.html)
