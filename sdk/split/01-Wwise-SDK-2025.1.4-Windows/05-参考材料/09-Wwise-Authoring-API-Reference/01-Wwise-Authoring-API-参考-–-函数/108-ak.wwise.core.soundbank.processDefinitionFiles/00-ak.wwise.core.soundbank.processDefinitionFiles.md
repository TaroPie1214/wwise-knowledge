|  |

| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.soundbank.processDefinitionFiles

从给定文件导入 SoundBank 定义。可指定多个文件。打开 WAAPI 日志查看状态消息。

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **files** \* | array | 此数组中包含 SoundBank 定义文件。 |
| files **[...]** | string | SoundBank Definition 文件。 |

(必填 \*)
