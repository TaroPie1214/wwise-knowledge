# ak.wwise.core.profiler.enableProfilerData

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.enableProfilerData

指定所要捕获的数据的类型。不沿用用户的 Profiler Settings。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **dataTypes** \* | array | 此数组中包含一系列数据类型。 |
| dataTypes **[...]** | object | 数据类型和启用状态。 |
| dataTypes[...].**dataType** \* | string | 指定所要捕获的数据的类型。  指定可捕获的性能分析器数据的类型。可能的值：  - cpu - memory - stream - voices - listener - obstructionOcclusion - markersNotification - soundbanks - loadedMedia - preparedObjects - preparedGameSyncs - interactiveMusic - streamingDevice - meter - auxiliarySends - apiCalls - spatialAudio - spatialAudioRaycasting - voiceInspector - audioObjects - gameSyncs - customerSupportData |
| dataTypes[...].**enable** | boolean | 允许 (true) 或禁止 (false) 捕获此类型的性能分析器数据。若未指定，则默认设为 true。 |

(必填 \*)
