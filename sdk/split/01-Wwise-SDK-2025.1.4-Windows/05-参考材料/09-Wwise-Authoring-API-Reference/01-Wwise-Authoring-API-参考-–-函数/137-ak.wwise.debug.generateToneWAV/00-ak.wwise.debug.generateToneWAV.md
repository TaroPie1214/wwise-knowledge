# ak.wwise.debug.generateToneWAV

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.debug.generateToneWAV

Generate a WAV file playing a tone with a simple envelope and save it to the specified location. This is provided as a utility to generate test WAV files.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **path** \* | string | File path to write the generated WAV file. This path normally uses the file extension WAV. |
| **bitDepth** | string | Bit depth of the audio file. Default is int16. 可能的值：  - int16 - float32 |
| **sampleRate** | number | Sample rate of the audio file. Default is 48000 Range: [300,192000] |
| **channelConfig** | string | Channel configuration of the audio file. Default is 1.0. Please use the silence waveform for the ambisonics configurations, as the other tone signals will be incompatible with the format. 可能的值：  - 0.1 - 1.0 - 2.0 - 2.1 - 3.0 - 4.0 - 5.1 - 7.1 - 5.1.2 - 7.1.2 - 7.1.4 - Ambisonics 1st order - Ambisonics 2nd order - Ambisonics 3rd order - Ambisonics 4th order - Ambisonics 5th order |
| **setAnonymous** | boolean | Sets the channel configuration type to anonymous, this overrides the channel config but keeps the number of channels specified. |
| **sustainTime** | number | Number of seconds of the signal holds at the specified level. Default is 1 second. 范围：[0,\*] |
| **sustainLevel** | number | Decibel attenuation for the sustained signal. Default is 0 dB. 范围: [-100,0] |
| **attackTime** | number | Number of seconds for the signal to reach sustain level. Default is 0 second. 范围：[0,\*] |
| **releaseTime** | number | Number of seconds for the signal to release from sustain level. Default is 0 second. 范围：[0,\*] |
| **waveform** | string | Waveform type. Default is silence. 可能的值：  - silence - sine - triangle - square - whiteNoise |
| **waveformChannelMask** | integer | Specifies which channels receive waveform data. The channel mask is a bit field where each bit represents a channel. The least significant bit represents the channel at index zero. The bits with 0 are silent. When not specified, the default outputs waveform data on all channels. |
| **frequency** | number | Waveform frequency. Default is 440. 范围: [1,22000] |
| **markers** | array | A list of markers to add to the generated WAV file. Each marker is an object with 'label' and 'position' properties, where 'position' is the position in seconds. |
| markers **[...]** | object | A marker to add to the generated WAV file. |
| markers[...].**label** | string | The label of the marker. Default is an empty string. |
| markers[...].**position** \* | number | The position of the marker in seconds. |

(必填 \*)
