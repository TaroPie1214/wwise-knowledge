# ak.wwise.debug.enableAsserts

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.debug.enableAsserts

启用 Debug 断言。每次使用 'false' 调用 enableAsserts 都会导致 ref 计数递增。若使用 true 调用，则 ref 计数递减。此函数仅在 Debug 版本中可用。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **enable** \* | boolean | 指示是否应启用断言。 |

(必填 \*)
