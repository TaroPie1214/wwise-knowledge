# ak.soundengine.setListenerSpatialization

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setListenerSpatialization

设置听者的空间化参数。这样方便定义各个音频声道的听者特定音量偏置。参阅 `AK::SoundEngine::SetListenerSpatialization` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **listener** \* | integer | “听者”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **spatialized** \* | boolean | 切换空间化设定（true：启用空间化；false：禁用空间化）。 |
| **channelConfig** \* | integer | 与 "volumeOffsets" 关联的声道配置。使用 `AK::AkChannelConfig::Serialize` 将数值序列化。  32 位无符号整数。范围：[0,4294967295] |
| **volumeOffsets** \* | array | 此数组中包含各个扬声器的音量偏置 (dB)。如需了解如何操控此数组，请参阅 AkSpeakerVolumes.h。 |
| volumeOffsets **[...]** | number | 各个扬声器的音量偏置。 |

(必填 \*)
