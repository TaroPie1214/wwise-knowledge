# ak.soundengine.postEvent

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.postEvent

按照 Event ID 将 Event 异步发送到声音引擎。参阅 `AK::SoundEngine::PostEvent` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **event** \* |  | Event 的 ID (GUID)、名称或 Short ID。  Any of: |
| string | 对象的名称。 |
| string | 以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| integer | Wwise 对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |
| **gameObject** | integer | 游戏对象的唯一 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |

(必填 \*)

[参数架构](ak_soundengine_postevent_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | integer | 所发送 Event 的 Playing ID 或 `AK_INVALID_PLAYING_ID（如` Event 发送失败）。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)

[结果架构](ak_soundengine_postevent_result_schema.html)
