# ak.wwise.core.profiler.getCursorTime

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getCursorTime

返回给定性能分析器光标的当前时间（毫秒）。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **cursor** \* | string | 指定要通过哪种 Time Cursor 获取时间。The User TiUser Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  识别其中一种全局性能分析器光标。可能的值：  - user - capture |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getcursortime_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | integer | 给定 Time Cursor 的当前位置 (ms)。 |

(必填 \*)
