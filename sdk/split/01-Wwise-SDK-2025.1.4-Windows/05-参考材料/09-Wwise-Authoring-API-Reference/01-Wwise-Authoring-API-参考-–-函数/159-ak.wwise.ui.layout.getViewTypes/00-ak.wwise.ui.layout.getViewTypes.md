# ak.wwise.ui.layout.getViewTypes

|  |

| Wwise SDK 2025.1.4 - Windows |

ak.wwise.ui.layout.getViewTypes

Retrieves a list of all view types registered in Wwise.

# Result

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **viewTypes** | array | Array of all view types registered in Wwise. |
| viewTypes **[...]** | object | Association between a view name and its display name. |
| viewTypes[...].**viewName** | string | The type name of the view. |
| viewTypes[...].**viewDisplayName** | string | The display name of the view. |
