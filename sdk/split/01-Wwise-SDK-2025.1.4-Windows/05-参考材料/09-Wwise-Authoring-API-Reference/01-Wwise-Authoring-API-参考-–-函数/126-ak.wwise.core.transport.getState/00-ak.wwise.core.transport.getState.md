# ak.wwise.core.transport.getState

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.transport.getState

获取给定走带对象的 State。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **transport** \* | integer | 要与所有其他 ak.wwise.core.transport 函数结合使用的走带对象 ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)

[参数架构](ak_wwise_core_transport_getstate_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **state** \* | string | 走带对象的 State。可能的值：  - playing - stopped - paused |

(必填 \*)

[结果架构](ak_wwise_core_transport_getstate_result_schema.html)
