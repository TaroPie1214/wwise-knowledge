# ak.wwise.core.remote.getAvailableConsoles

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.remote.getAvailableConsoles

检索所有可用主机以将 Wwise 设计工具连接到声音引擎实例。

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **consoles** \* | array | 此数组中包含一系列可用主机。 |
| consoles **[...]** | object | 描述远程进程。 |
| consoles[...].**name** \* | string | 可执行文件返回的远程主机的名称。 |
| consoles[...].**platform** \* | string | 可执行文件返回的远程主机的平台。 |
| consoles[...].**customPlatform** \* | string | 此平台与可执行文件返回的远程主机的工程平台设定相同。 |
| consoles[...].**host** \* | string | 所连主机的 IPv4 地址。若 Wwise 被连接到本地文件（性能分析器会话），则此项还可为文件路径。For using WAAPI on Mac, please see [在 Mac 上使用 WAAPI](waapi_prepare.html#waapi_path_on_mac) . |
| consoles[...].**appName** \* | string | 可执行文件返回的所连应用程序的名称。须在连接到特定声音引擎实例时使用。 |
| consoles[...].**commandPort** \* | integer | 命令端口。可在连接到特定声音引擎实例时使用。  16 位无符号整数。范围：[0,65535] |

(必填 \*)
