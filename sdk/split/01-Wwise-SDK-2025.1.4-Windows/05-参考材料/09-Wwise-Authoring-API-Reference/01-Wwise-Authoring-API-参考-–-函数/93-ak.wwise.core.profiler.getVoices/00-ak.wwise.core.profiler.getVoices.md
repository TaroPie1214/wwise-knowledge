# ak.wwise.core.profiler.getVoices

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getVoices

在特定性能分析器捕获时间检索声部。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **time** \* |  | 指定要在什么时间（毫秒）查询声部或通过哪种 Time Cursor 获取时间。User Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  One of: |
| integer | 范围：[0,\*] |
| string | 识别其中一种全局性能分析器光标。可能的值：  - user - capture |
| **voicePipelineID** | integer | 所要获取的单个声部的 Pipeline ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getvoices_args_schema.html)

# 选项

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** | array | 要针对各个声部返回的成员。默认设为 Pipeline ID、Game Object ID 和 Object GUID。  此数组中包含在播声部相关选项。 |
| return **[...]** | string | 针对声部管线结构指定其中一个定义的成员。可能的值：  - pipelineID - playingID - soundID - gameObjectID - gameObjectName - objectGUID - objectName - playTargetID - playTargetGUID - playTargetName - baseVolume - gameAuxSendVolume - envelope - normalizationGain - lowPassFilter - highPassFilter - priority - isStarted - isVirtual - isForcedVirtual |

[选项架构](ak_wwise_core_profiler_getvoices_options_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | array | 此数组中包含一系列声部。 |
| return **[...]** | object | 声部条目。  在播声部。使用 return 选项来指定所需的属性。 |
| return[...].**pipelineID** | integer | 声部的 Pipeline ID。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**playingID** | integer | 声部的 Playing ID。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**soundID** | integer | 与声部对应的声音对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**gameObjectID** |  | Game Object ID corresponding to the voice.  Any of: |
| integer | Unsigned Integer 64-bit. 范围：[0,18446744073709551615] |
| integer | Integer 64-bit. 范围: [-9223372036854775808,9223372036854775807] |
| return[...].**gameObjectName** | string | 与声部对应的 Game Object Name。 |
| return[...].**objectGUID** | string | 与声部对应的 Object GUID。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| return[...].**objectName** | string | 与声部对应的 Object Name。 |
| return[...].**playTargetID** | integer | 与声部对应的 Play Target 对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |
| return[...].**playTargetGUID** | string | 与声部对应的 Play Target 对象的 GUID。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| return[...].**playTargetName** | string | 与声部对应的 Play Target 对象的名称。 |
| return[...].**baseVolume** | number | 声部音量 (dB)，包含 HDR 衰减。 |
| return[...].**gameAuxSendVolume** | number | 发送到 Auxiliary Bus Send 的音量 (dB)。 |
| return[...].**envelope** | number | 当前分析的包络值 (dB)。若不可用，则返回 0。 |
| return[...].**normalizationGain** | number | Loudness Normalization 和 Make-Up Gain (dB)。 |
| return[...].**lowPassFilter** | number | 作用于声部的 Low-Pass Filter。 |
| return[...].**highPassFilter** | number | 作用于声部的 High-Pass Filter。 |
| return[...].**priority** | integer | 赋予声部的优先级。  8 位整数。范围：[-128,127] |
| return[...].**isStarted** | boolean | 若声部已开始播放，则返回 true，否则返回 false。 |
| return[...].**isVirtual** | boolean | 若声部为虚声部，则返回 true，否则返回 false。 |
| return[...].**isForcedVirtual** | boolean | 若声部被强制设为虚声部，则返回 true，否则返回 false。 |

(必填 \*)
