# ak.soundengine.stopPlayingID

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.stopPlayingID

停止播放与给定 Playing ID 关联的当前内容。参阅 `AK::SoundEngine::StopPlayingID` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **playingId** \* | integer | 与所要停止播放的内容关联的 Playing ID。  32 位无符号整数。范围：[0,4294967295] |
| **transitionDuration** \* | integer | 淡变时长 (ms)。  32 位整数。范围：[-2147483648,2147483647] |
| **fadeCurve** \* | integer | 过渡所要使用的曲线类型。使用来自 [AkCurveInterpolation](_ak_enums_8h_a77872044bca52a4d20324739154f2399.html#a77872044bca52a4d20324739154f2399) 的值。范围：[0,9] |

(必填 \*)
