# ak.soundengine.setDefaultListeners

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setDefaultListeners

为所有后续注册的游戏对象设置默认的活跃听者。参阅 `AK::SoundEngine::SetDefaultListeners` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **listeners** \* | array | 此数组中包含一系列“听者”游戏对象 ID。游戏对象必须之前已经注册过。 |
| listeners **[...]** | integer | 游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |

(必填 \*)
