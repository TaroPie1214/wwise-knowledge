# ak.wwise.core.getInfo

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.getInfo

检索全局 Wwise 信息。

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **sessionId** \* | string | Wwise 会话 ID。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| **apiVersion** \* | number | Wwise Authoring API 的版本。范围：[1,\*] |
| **displayName** \* | string | Wwise 显示名称。 |
| **branch** \* | string | 构建的分支。 |
| **copyright** \* | string | 版权信息。 |
| **version** \* | object | Wwise 版本对象。 |
| version.**displayName** \* | string | Wwise 版本名称。 |
| version.**year** \* | integer | 版本年份。范围：[2000,2100] |
| version.**major** \* | integer | 大版本号。范围：[0,100] |
| version.**minor** \* | integer | 小版本号。范围：[0,100] |
| version.**build** \* | integer | 内部版本号。范围：[1,\*] |
| version.**nickname** \* | string | 为某一版本取的特殊名称。 |
| version.**schema** \* | integer | Wwise 工程和 Work Unit 的架构版本。范围：[1,\*] |
| **configuration** \* | string | 指示 Release 或 Debug 版本。可能的值：  - release - debug |
| **platform** \* | string | 指示 Wwise 的构建平台。可能的值：  - x64 - win32 - macosx - linux |
| **isCommandLine** \* | boolean | 指示 Wwise 是否在命令行中运行。 |
| **processId** \* | integer | Wwise 的进程标识符。 |
| **processPath** \* | string | Wwise 的进程路径。 |
| **directories** \* | object | Wwise 所用的各项目录。 |
| directories.**install** \* | string | Wwise 的根目录。此项为安装目录。 |
| directories.**authoring** \* | string | Wwise 设计工具根目录。 |
| directories.**bin** \* | string | Wwise.exe 所在的 bin 目录。 |
| directories.**help** \* | string | 帮助目录。 |
| directories.**user** \* | string | Wwise 用户数据根目录。 |

(必填 \*)
