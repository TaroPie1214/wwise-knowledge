# ak.wwise.core.getProjectInfo

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.getProjectInfo

检索有关当前所打开工程的信息，包括平台、语言和工程目录。

# Result

| **path** \* | string | WPROJ 文件的绝对路径。 |
