# ak.wwise.core.profiler.getRTPCs

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.profiler.getRTPCs

在特定性能分析器捕获时间检索活跃的 RTPC。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **time** \* |  | 指定要在什么时间（毫秒）查询 RTPC 或通过哪种 Time Cursor 获取时间。User Time Cursor 可由用户操控，Capture Time Cursor 代表当前捕获的最新时间。  One of: |
| integer | 范围：[0,\*] |
| string | 识别其中一种全局性能分析器光标。可能的值：  - user - capture |

(必填 \*)

[参数架构](ak_wwise_core_profiler_getrtpcs_args_schema.html)

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **return** \* | array | 此数组中包含一系列 RTPC。 |
| return **[...]** | object | 与在播声部关联的 RTPC。使用 return 选项来指定所需的属性。 |
| return[...].**id** \* | string | Game Parameter、LFO、Time、Envelope 或 MIDI Parameter 对象的 ID (GUID)。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| return[...].**name** \* | string | The name of the Game Parameter, LFO, Time, Envelope or MIDI Parameter object. |
| return[...].**gameObjectId** \* | integer | 与 RTPC 作用域关联的 Game Object 或与全局作用域 RTPC 对应的 AK\_INVALID\_GAME\_OBJECT。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| return[...].**value** \* | number | Game Parameter、LFO、Time、Envelope 或 MIDI Parameter 在光标时间位置的值。 |

(必填 \*)

[结果架构](ak_wwise_core_profiler_getrtpcs_result_schema.html)
