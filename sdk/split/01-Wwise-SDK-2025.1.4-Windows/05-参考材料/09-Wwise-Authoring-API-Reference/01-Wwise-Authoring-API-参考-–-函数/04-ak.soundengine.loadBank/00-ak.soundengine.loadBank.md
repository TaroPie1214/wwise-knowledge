# ak.soundengine.loadBank

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.loadBank

Load a SoundBank. See `AK::SoundEngine::LoadBank`.

# Arguments

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **soundBank** \* |  | Either the ID (GUID), name, or Short ID of the SoundBank.  Any of: |
| string | 对象的名称。 |
| string | 以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| integer | Wwise 对象的 Short ID。  32 位无符号整数。范围：[0,4294967295] |

(必填 \*)
