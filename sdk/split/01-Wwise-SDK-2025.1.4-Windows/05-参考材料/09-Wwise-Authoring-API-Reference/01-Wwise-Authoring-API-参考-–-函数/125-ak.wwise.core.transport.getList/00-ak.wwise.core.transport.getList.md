# ak.wwise.core.transport.getList

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.wwise.core.transport.getList

返回包含走带对象的列表。

# 结果

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **list** | array | 此数组中包含一系列走带对象。 |
| list **[...]** | object | 走带对象。 |
| list[...].**object** | string | 由走带对象控制的对象的 ID (GUID)。  以下形式的对象 GUID：{aabbcc00-1122-3344-5566-77889900aabb}。 |
| list[...].**gameObject** | integer | 走带对象所用的游戏对象。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| list[...].**transport** | integer | 走带对象 ID。  32 位无符号整数。范围：[0,4294967295] |
