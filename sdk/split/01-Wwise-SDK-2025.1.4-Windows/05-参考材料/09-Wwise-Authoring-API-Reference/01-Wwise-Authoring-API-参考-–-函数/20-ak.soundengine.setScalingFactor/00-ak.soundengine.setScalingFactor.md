# ak.soundengine.setScalingFactor

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setScalingFactor

设置游戏对象的缩放系数。您可以修改此游戏对象的衰减计算结果，以便模拟具有更大或更小传播区域的声音。参阅 `AK::SoundEngine::SetScalingFactor` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **gameObject** \* | integer | 游戏对象标识符。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **attenuationScalingFactor** \* | number | 缩放系数：1 表示 100；0.5 表示 50；2 表示 200…以此类推。 |

(必填 \*)
