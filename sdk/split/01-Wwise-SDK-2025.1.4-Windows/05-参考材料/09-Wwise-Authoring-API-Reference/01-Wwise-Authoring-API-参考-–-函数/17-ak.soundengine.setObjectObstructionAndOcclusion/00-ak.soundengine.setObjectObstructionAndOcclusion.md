# ak.soundengine.setObjectObstructionAndOcclusion

|  |
| --- |
| Wwise SDK 2025.1.4 - Windows |

ak.soundengine.setObjectObstructionAndOcclusion

设置游戏对象的声障和声笼电平。此函数用于影响对象应当如何被特定听者听到。参阅 `AK::SoundEngine::SetObjectObstructionAndOcclusion` 章节。

# 参数

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| **emitter** \* | integer | “发声体”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **listener** \* | integer | “听者”游戏对象 ID。  游戏对象 ID，64 位无符号整数。范围：[0,18446744073709551615] |
| **obstructionLevel** \* | number | ObstructionLevel：[0.0f..1.0f]。 |
| **occlusionLevel** \* | number | OcclusionLevel：[0.0f..1.0f]。 |

(必填 \*)
